# AWS re:Invent 2025 会议总结

## 会议概述

本次会议由 AWS 全球金融客户解决方案架构师 Phillip 和 Allianz Technology 的开发者 Tivida 共同主讲，主题聚焦于在企业规模下构建 AI 和生成式 AI 平台所面临的挑战及解决方案。会议深入探讨了 Allianz 如何应对 GenAI 时代带来的复杂性爆炸式增长，以及如何在不锁定技术栈的前提下，为多样化的用户群体提供灵活且标准化的 AI 平台。

Phillip 首先分析了过去两年 GenAI 带来的能力激增和复杂性挑战，指出传统 ML 平台的稳定能力集已被大量新功能所扩展，包括 RAG、Agent 框架、提示工程等。同时，AI 的民主化使得用户群体从数据科学家扩展到业务分析师和普通开发者，不同技能水平的用户需要不同层次的支持。Tivida 随后详细介绍了 Allianz 的实践经验，展示了基于 AWS SageMaker 和 Bedrock 构建的数据科学工作台 2.0，以及如何通过"双路径"策略和"低遗憾标准"实现快速创新与企业治理的平衡。

核心理念是识别稳定的平台元素（如 Git、OpenTelemetry、容器化），将其与快速变化的应用层（框架、模型选择）解耦，从而在保持灵活性的同时实现规模化运营。Allianz 通过提供开箱即用的工具（GenAI Lab、SageMaker Canvas）、自动化基础设施即代码模板、以及基于 Git 的提示管理系统，成功支持了从技术人员到业务用户的全方位 AI 应用开发需求。
