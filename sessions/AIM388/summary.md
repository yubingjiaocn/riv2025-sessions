# AWS re:Invent 2025 - 使用 SageMaker Studio 和 Bedrock Agent Core 加速构建智能代理

## 会议概述

本次技术分享会由 Amazon SageMaker AI 产品负责人 Summit Thakur 主持，AWS 生成式 AI 高级专家 David Galitelli 和 Robinhood 高级机器学习工程师 Nikhil Singhal 共同参与。会议重点介绍了如何利用 SageMaker Studio 和 Bedrock Agent Core 加速智能代理开发，涵盖了从模型定制到生产部署的完整工作流程。

演讲者详细展示了企业级智能代理开发的关键驱动因素，包括改善客户体验、自动化运营工作流程和提升员工生产力。会议特别强调了定制模型在智能代理中的重要性，Gartner 预测未来三年内将有超过50%的企业选择使用定制模型，相比去年仅有1%的使用率有显著提升。整个演示通过实际案例展现了从数据生成、模型训练到代理部署的端到端解决方案。
