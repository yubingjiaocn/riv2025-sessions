# AWS re:Invent 2025 Qlik 开放湖仓技术会议总结

## 会议概述 (Session Overview)

本次会议由 Qlik 公司主办，主题为"基于开放湖仓技术构建可扩展且经济高效的 AI 数据基础设施"。演讲者介绍了 Qlik 作为全球软件公司的端到端数据平台解决方案，涵盖数据集成、转换和分析功能。会议重点讨论了当前 AI 发展趋势下，如何构建强大的数据基础来支持 AI 项目的成功实施。

演讲者强调了 Qlik 与 AWS 的长期战略合作伙伴关系，以及如何利用 AWS 服务来加速客户的数据项目交付。会议核心内容围绕开放数据架构的重要性，特别是基于 Apache Iceberg 的湖仓解决方案如何在降低成本的同时提供更好的数据互操作性和实时处理能力。

## 详细时间线与关键要点 (Detailed Timeline)

### 0:00-2:00 会议开场与 Qlik 公司介绍
- 感谢参会者从繁忙的会议日程中抽出时间参加
- 介绍 Qlik 作为全球软件公司的背景和产品组合
- 强调 Qlik 提供端到端数据平台，涵盖数据集成、转换和分析
- 突出与 AWS 的长期战略合作伙伴关系

### 2:00-5:00 AI 发展现状分析
- 讨论 AI 技术的成熟度曲线和当前发展阶段
- 指出近 70% 的组织已开始制定正式的 AI 策略
- 分析 AI 项目面临的挑战：ROI 难以预测和量化
- 强调数据基础设施是 AI 项目成功的关键障碍

### 5:00-8:00 开放数据架构介绍
- 提出开放数据架构的概念和重要性
- 展示架构图：从多样化数据源到消费端的完整数据流
- 支持各种数据源：API 端点、SaaS 应用、数据库系统、实时流数据
- 强调需要处理半结构化数据和模式演进的复杂性

### 8:00-11:00 Iceberg 湖仓技术优势
- 解释为什么选择基于 Iceberg 的湖仓架构
- 两大核心价值：成本效益和互操作性
- 展示行业生态系统对 Iceberg 的广泛支持
- 介绍 AWS 服务对 Iceberg 的全面支持：S3 Tables、Glue、SageMaker、EMR、Athena

### 11:00-14:00 Qlik Open Lakehouse 产品介绍
- 发布 Qlik Open Lakehouse 托管 Iceberg 解决方案
- 三大核心功能：高吞吐量数据摄取、自适应优化器、数据仓库镜像
- 支持多种数据源：API、数据库 CDC、Amazon Kinesis、Kafka、微批处理
- 自动优化功能：文件压缩、快照过期、孤立文件删除

### 14:00-16:00 性能基准测试结果
- 展示实时流数据摄取基准测试
- 数据新鲜度提升 5 倍：湖仓 1-3 分钟 vs 数据仓库 5-15 分钟
- 成本降低 75-90%
- 强调 Iceberg 表的跨引擎互操作性优势

### 16:00-17:30 现场演示与总结
- 介绍 Qlik 展台的实时 IoT 自行车演示
- 展示从 Amazon Kinesis 到 Iceberg 表的实时数据管道
- 演示数据质量信任评分功能
- 邀请参会者到展台体验完整解决方案
