# AWS re:Invent 2025 S3 新功能发布会总结

## 会议概述

本次会议由AWS S3团队的Paul Meighan和Anna Bouchet主讲，全面回顾了过去12个月S3服务的重要更新。会议涵盖了35项新功能发布，包括安全性、持久性、S3 Express高性能存储、对象管理、元数据处理以及全新的向量存储服务。

演讲者采用了创新的演示方式，通过现场观众参与（Matt和Indra Neil作为志愿者）展示了多项新功能的实际应用，包括基于标签的访问控制、批量操作、S3元数据和S3向量搜索等核心特性。整个会议展现了S3服务在AI时代的重要演进，特别是在语义搜索和智能数据管理方面的突破。

## 详细时间线与关键要点

### 00:00-05:00 开场介绍与安全功能
- 介绍演讲者Paul Meighan和Anna Bouchet，宣布将覆盖35项新功能
- 确定现场志愿者Matt和Indra Neil参与演示
- 强调安全是AWS最重要的工作重点
- 发布基于标签的访问控制功能，简化多客户端多存储桶的权限管理
- 新增单标签操作API：untag resource、tag resource和list tags for resource

### 05:00-10:00 组织级控制与加密更新
- 推出组织级别的公共访问阻止功能
- 扩展403上下文功能至整个组织范围
- 发布PQ TLS后量子加密，保护传输中的数据
- 更新SSEC加密默认设置，2026年新存储桶将默认关闭SSEC

### 10:00-15:00 演示开始与持久性介绍
- Anna开始复杂演示，上传700个客户案例文件
- 展示增强错误消息功能，帮助调试访问拒绝问题
- Paul介绍S3的11个9持久性保证
- 发布新的数据完整性检查功能，通过批量操作系统高效验证数据

### 15:00-20:00 S3 Express更新
- S3 Express价格下调高达85%
- 提升TPS限制：每秒200万次读取，20万次写入
- 新增对象重命名API和访问点支持
- 强调S3 Express在高吞吐量训练工作负载中的应用

### 20:00-25:00 S3对象管理功能
- 条件写入功能获得快速采用，每日执行数百万次操作
- 新增条件复制和条件删除功能
- 最大对象大小从5TB提升至50TB
- 对象标签价格下调35%

### 25:00-30:00 批量操作增强
- 新增无清单选项，可直接处理整个存储桶或前缀
- 自动化IAM角色创建
- 单个作业可处理高达200亿个对象
- 性能提升高达10倍

### 30:00-35:00 演示：批量操作实践
- Anna展示批量操作的实际应用
- 成功为75个匹配条件的对象添加黑色标签
- 作业在5秒内完成，展示了批量操作的高效性

### 35:00-40:00 S3 Tables重大更新
- 客户已创建超过40万个表，显示强劲需求
- 新增Iceberg Rest catalog端点
- 表限制提升至每个表存储桶10,000个表
- 排序压缩功能和90%的压缩价格下调

### 40:00-45:00 表级功能与复制
- 表作为AWS资源的独特优势
- 表级CloudWatch、CloudTrail监控和KMS加密
- 发布表复制功能，提供只读副本
- 智能分层支持，自动优化存储成本高达80%

### 45:00-50:00 S3元数据与演示
- 日志表价格下调33%
- 新增实时清单表功能
- 与SageMaker Unified Studio集成，提供一键式SQL查询体验
- Anna演示元数据查询，确认上传了744个对象

### 50:00-57:00 S3向量存储介绍与演示
- Paul详细解释向量概念和语义搜索原理
- S3向量存储成本降低高达90%
- 支持每个索引20亿个向量，每个向量存储桶10,000个索引
- 与OpenSearch和Bedrock知识库集成
- Anna成功演示通过向量搜索找到包含志愿者姓名的会议记录
- 会议圆满结束，感谢观众参与
