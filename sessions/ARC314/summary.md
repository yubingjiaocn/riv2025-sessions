# AWS re:Invent 2025 技术会议总结：解锁微服务的智能代理AI

## 会议概述

本次AWS re:Invent 2025分组会议由Roland Barcia（AWS专业技术团队全球总监）和Midulla Grandi（北美专业服务市场负责人）主讲，主题为"解锁微服务的智能代理AI"。会议重点探讨了如何将AI代理集成到现有的微服务架构中，以及如何通过模型上下文协议（MCP）和代理到代理（A2A）通信来实现这一目标。

演讲者强调，我们正在进入"智能代理时代"，AI应用程序已经爆炸式增长。然而，挑战在于如何将这些智能代理与现有的微服务系统有效集成。传统的微服务架构虽然功能强大，但代理只能理解自然语言，需要通过上下文工程来桥接这一差距。会议介绍了AWS Agent Core平台的各种组件，包括运行时环境、网关、内存管理和可观测性工具，以及Strands开源SDK如何简化代理开发过程。
