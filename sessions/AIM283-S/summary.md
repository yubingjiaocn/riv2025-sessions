# AWS re:Invent 2025 - ServiceNow AI Control Tower 与 AWS 云平台协作会议总结

## 会议概述

本次会议重点介绍了ServiceNow AI控制塔与AWS云平台的深度集成，展示了两大平台如何共同推动企业AI驱动的工作流程转型。会议由ServiceNow和AWS的技术专家主讲，包括来自DXC保险公司的客户案例分享。主要议题涵盖了AI代理编排、数据治理、量子计算与AI的结合，以及企业级AI解决方案的实际应用场景。

会议强调了三个核心价值：成本节约、创新能力和业务影响。ServiceNow现已可直接在AWS云上运行，两个平台拥有超过100个开箱即用的集成方案。通过统一的AI平台架构，企业可以实现从混乱到清晰的数字化转型，将AI代理、数据和工作流程整合在单一平台上。

## 详细时间线与关键要点

### 0:00-10:00 开场介绍与平台优势
- ServiceNow与AWS合作的三大价值：节约成本、推动创新、产生业务影响
- ServiceNow可直接在AWS云上运行，满足数据驻留和性能要求
- 两平台拥有超过100个预构建集成，包括AWS SiteWise、Connect、Redshift等服务

- 金融服务行业案例：保险理赔自动化流程优化
