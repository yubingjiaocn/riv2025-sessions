# AWS re:Invent 2025 - 使用 AWS Transform 现代化 SQL Server 和 .NET 应用

## 会议概述

本次 AWS re:Invent 2025 分会场由 AWS Transform 产品团队负责人 Kuram Kaja、产品经理 Nit Janathan 以及软件开发高级经理 Vijay Mandari 共同主讲。会议聚焦于企业面临的核心挑战：如何现代化基于 Windows 的传统应用程序。AWS Transform 作为首个基于智能体 AI（Agentic AI）的平台，能够实现全栈现代化，包括 UI 框架升级、.NET 代码迁移、SQL Server 到 PostgreSQL 的数据库转换，以及在 Linux EC2 实例上的部署。

该解决方案通过多智能体协作框架，将传统的 .NET Framework 应用升级到 .NET 8/10，同时将 SQL Server 数据库迁移到开源的 Aurora PostgreSQL，并部署到 Linux 环境。这种端到端的现代化方案可将转换效率提升 5 倍，实现跨团队（DBA、开发人员、运维团队）的协同工作，并通过采用开源技术削减 70% 的运营和许可成本。整个过程高度自动化，通过智能体系统动态规划、评估依赖关系、转换架构和代码，并提供可组合的工作流程以适应不同客户需求。
