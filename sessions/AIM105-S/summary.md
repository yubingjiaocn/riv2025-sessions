# AWS re:Invent 2025 医疗保健患者参与会议总结

## 会议概述

本次分组会议由Kendra公司的数字化工作场所服务实践咨询合伙人Edward Mleharren和AWS美国医疗保健客户体验参与高级业务开发负责人Sunil Manon共同主持。会议重点探讨了如何利用人工智能技术在整个医疗护理过程中及之后提升患者参与度。演讲者强调了三个核心主题：通过消费化IT重新构想前端患者互动体验、消除长时间等待和低效活动、以及降低医疗服务交付的复杂性。

会议详细介绍了Amazon Connect作为全面的联络中心即服务平台的功能和优势。该平台最初由亚马逊为自身需求开发，2017年开始商业化运营，现已成为联络中心领域的领导者。演讲者通过实际演示展示了如何利用对话式AI、自然语言处理和全渠道通信能力来创造无缝的患者体验。会议还分享了实际案例，包括州政府机构如何通过部署Amazon Connect在首月实现30%的呼叫转移率，以及Jupiter医疗中心如何通过主动外呼活动将昂贵放射设备的利用率从75%提升至95%。演讲者强调了模块化部署、按使用量付费的消费模式，以及选择合适合作伙伴的重要性。

## 详细时间线

开场介绍 (0:00-2:30)
- Edward Mleharren介绍自己是Kendra公司数字化工作场所服务实践的咨询合伙人，专注于联络中心现代化
- Sunil Manon介绍自己是AWS美国医疗保健客户体验参与高级业务开发负责人
- 提供会后交流机会的QR码

核心主题介绍 (2:30-5:45)
- 主题一：利用AI重新构想患者前端互动体验，满足消费者对类似ChatGPT体验的期望
- 主题二：通过AI自助服务能力和更好的工具消除长时间等待，应对人力短缺和呼叫量增加的挑战
- 主题三：降低医疗服务交付的复杂性，通过AI全面协调预约、专科医生和后续护理

患者旅程分析 (5:45-8:15)
- Sunil详细说明患者旅程的各个阶段：接诊、沟通、预约、就诊（虚拟或现场）、术后随访和持续参与
- 强调前端接触点是收集信息和提供高质量体验的关键
- 介绍利用Amazon Lex和Nova Sonic提供自然语言AI支持的自助服务
- 指出许多客户实现了40%的呼叫转移率

Amazon Connect平台介绍 (8:15-11:30)
- Edward介绍Amazon Connect是由亚马逊为自身开发的完整联络中心即服务平台
- 2017年开始商业化，现已成为Gartner认可的行业领导者
- 平台功能包括：全渠道通信（语音、聊天、短信、视频、应用内网络通话）、统一的座席工作空间、主管报告和QA工具、简单直观的管理界面
- 强调两大优势：模块化设计和按使用量付费的消费模式

实际演示 (11:30-16:45)
- 播放完整的患者互动演示，展示以下功能：
  - 通过电话号码自动识别患者身份
  - 多因素身份验证（一次性验证码和出生日期）
  - 识别指定护理人员身份
  - 基于出院护理计划主动建议预约
  - 根据历史偏好推荐预约时间
  - 自动检查转诊授权状态
  - 提供详细的就诊准备指南
  - 通过电子邮件和短信发送确认信息

平台优势总结 (16:45-19:00)
- 通过全渠道通信增强患者参与度（演示中展示了语音、对话式AI和短信的无缝整合）
- 利用AI和自动化实现个性化护理（与EHR等关键系统集成）
- 简化行政和运营流程（在IVR中嵌入账单支付、预约重新安排等简单任务）
- 将联络中心转变为患者参与平台的战略思维

实施策略 (19:00-22:30)
- Sunil介绍客户可以从任何地方开始实施
- Amazon Connect为未来设计，支持各种AI类型：代理AI、对话式AI、预测AI、生成式AI
- 快速启动方式：
  - 改进IVR（利用Lex、Nova Sonic和代理工作流）
  - 推荐参加周四加州大学圣地亚哥分校的会议，了解他们如何利用代理工作流
  - 主动外呼活动（Jupiter医疗中心案例：将放射设备利用率从75%提升至95%）
- 传统联络中心现代化和诊所/部门的隐藏机会

客户案例研究 (22:30-25:45)
- Edward分享州政府安全网福利项目案例
- 挑战：联邦项目结束后重新注册导致呼叫量激增，等待时间超过2-4小时
- 解决方案：设计新系统支持自助服务（查询卡余额、下次福利时间、其他福利资格等）
- 成果：
  - 首月实现30%的呼叫转移率
  - 在非工作时间提供自助服务能力
  - NPS分数提升
  - 座席满意度提高
  - 平均处理时间缩短

成功关键因素 (25:45-27:30)
- Sunil总结三个关键步骤：
  1. 决定迁移到基于云的患者参与解决方案（利用数据、AI和持续互动）
  2. 采用循序渐进的方法（从IVR现代化和外呼活动开始，利用节省的成本推动其他部分现代化）
  3. 选择合适的合作伙伴（Kendra在医疗保健和Amazon Connect方面经验丰富）
  4. 让内部团队参与旅程（提升技能，为未来的代理工作流部署做准备）

  - 除Amazon Connect外还提供多种医疗保健领域服务
