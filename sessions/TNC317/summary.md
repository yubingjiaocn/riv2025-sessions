# AWS re:Invent 2025 企业级现代化会议总结

## 会议概述

本次会议由 Scott Picket 主讲，主题为"利用 Agentic AI 和 Janice 工具实现企业规模的现代化"。会议深入探讨了企业在遗留系统现代化过程中面临的核心挑战，包括技术人才退休、技术债务累积、许可费用激增以及业务创新需求等问题。TSRI 公司作为拥有 30 年行业经验的现代化解决方案提供商，展示了其如何通过确定性 AI 引擎 Janice 和生成式 AI 相结合的方式，帮助金融、保险、邮政、国防等多个行业的客户实现从 COBOL、Fortran、ADA 等遗留语言到现代 Java/C# 平台的迁移。

该解决方案的核心亮点在于将传统的基于规则的确定性 AI 与生成式 AI 技术深度融合，通过知识图谱和向量数据库实现代码的自动化评估、转换、测试和优化。特别值得关注的是其新推出的 Juno 应用代理，能够以自然语言方式回答关于遗留代码的问题，生成业务文档、测试用例和项目计划。整个现代化流程实现了"单键转换"，大幅缩短了传统上需要数月甚至数年的迁移周期，同时确保功能等效性和代码质量。

会议还分享了美国空军和 Nationwide 保险等成功案例，展示了 90% 的成本削减和快速部署能力。该方案不仅解决了遗留系统的技术问题，还为企业提供了持续现代化、微服务拆分和 SaaS 转型的能力，使企业能够在 AWS 云环境中实现业务创新和敏捷开发。

## 详细时间线

0:00 - 开场介绍
- Scott Picket 介绍会议主题：企业规模现代化与 Agentic AI

0:30 - 问题陈述
- 核心挑战一：技术人才退休（COBOL、Fortran、ADA 专家稀缺）
- 核心挑战二：技术债务不断增长（系统多年未更新，更新周期长达一年）
- 核心挑战三：许可费用激增（供应商涨价高达 300%）
- 核心挑战四：业务创新需求（遗留系统无法支持 Web 服务和新功能）

2:00 - TSRI 公司背景
- 30 年行业经验，源自波音 AI 实验室
- 服务领域：国防、联邦、零售、邮政、保险、金融等
- 成功案例：美国邮政服务、丹麦邮政、Nationwide 保险
- 行业认可：Gartner 魔力象限领导者地位

3:30 - 解决方案架构
- 确定性 AI 引擎 Janice：基于规则的代码转换，输出可预测
- 生成式 AI 集成：通过知识图谱和向量数据库增强能力
- 三大核心功能：评估、转换、现代化

5:00 - 评估阶段（Juno 应用代理）
- 自动生成业务文档、设计文档和测试文档
- 提供项目反馈和 POC 建议
- 帮助客户启动现代化旅程

6:00 - 代码转换阶段
- 全自动化转换流程，部署在 AWS 上
- 生成转换后的代码和测试输出
- 数据库架构转换和 ETL 脚本生成
- 屏幕界面转换（从绿屏到 React/Angular/Blazer）

7:30 - 现代化阶段
- 使用生成式 AI 进行惯用语重构
- 建立稳定的 CI/CD 流水线
- 生成单元测试和回归测试
- 注入 REST 端点和 API 接口
- 支持微服务拆分

9:00 - 技术细节：Janice 引擎
- 词法分析和语法解析
- 生成中间对象模型（抽象语法树）
- 消除语法问题和 GOTO 语句
- 提取嵌套循环，注入 REST 接口
- 实现容器化和基于规则的重构

11:00 - 知识图谱与生成式 AI
- 自定义提示引擎
- 生成业务文档和测试用例
- 提供测试元数据和种子测试
- 扩展测试覆盖率

13:00 - Juno 功能演示
- 项目评估和业务逻辑理解
- 可查询的智能代理
- 成本估算和 AWS 环境节省分析
- 战略建议：替换外部服务和专有程序
- 生成多种项目计划：测试计划、转换计划、外部替换计划、托管计划

16:00 - AI 驱动的文档生成
- 将 COBOL 代码转换为英文文档
- 深入理解程序实现
- 支持自然语言对话查询
- 回答关于批处理程序、支付网关等问题

18:00 - 数据库迁移支持

- 源数据库和目标数据库差异分析
- 提供详细的迁移映射

20:00 - 执行仪表板
- 替代 4-5 个项目经理的工作
- 代码集完整性检查
- 交付状态实时监控
- 代码质量摘要（SonarQube 评分）
- 安全漏洞检测
- 框架实现细节暴露
- COBOL 与 Java 数据类型映射
- 外部函数实现状态
- 测试覆盖率报告

23:00 - 单键转换功能
- 一键启动完整转换流程
- 自动完成词法分析和语法解析
- 包装器识别和解决方案验证
- 代码生成和编译确认
- 通过 CI/CD 流水线
- 单元测试和测试覆盖率分析
- 代码质量扫描
- 功能等效性测试

25:30 - 实时日志监控
- 实时显示解析进度
- 各阶段状态可视化
- 详细的日志输出

26:30 - SonarQube 集成与智能重构
- 与 SonarQube 集成生成报告
- 通过 LLM 推荐重构方案
- Agentic 重构：单击执行
- 状态保持和选择性应用

28:00 - 测试仪表板
- 从源代码提取元数据（文件、表、输入参数）
- COBOL 代码插桩自动收集测试数据
- 自动打包测试文件和数据表
- 在现代环境中运行功能等效测试

30:00 - 功能等效性测试
- 三个测试状态：运行无崩溃、无错误、输出一致
- 逐位比较输出与遗留系统
- 支持数千个测试用例
- 覆盖数千个批处理程序和数百个屏幕

31:30 - 端到端系统测试
- 批处理端：使用 Airflow 或 Python 编排

32:30 - 客户案例：美国空军
- 实现 90% 的成本削减
- 数天内完成部署
- 不到 2 个月完成重构

- 支持业务创新：专家可维护，满足业务需求，支持服务拆分和 API 提取

37:00 - MSP 服务模式
- 多年持续现代化计划
- 持续拆分服务和 API

- 持续现代化服务
