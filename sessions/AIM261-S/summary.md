# AWS re:Invent 2025 Poolside AI 会议总结

## 会议概述

本次会议由 Poolside 公司的解决方案工程负责人 Corey Dobson、首席解决方案架构师 Evo 以及诺斯罗普·格鲁曼空间系统公司的高级工程师 Pine 共同主讲。会议重点介绍了 Poolside AI 如何为高度安全的软件环境提供完整的 AI 基础设施解决方案。

Poolside 是一家成立于 2023 年的美国前沿模型实验室，专注于构建可在任何地方部署的安全企业级 AI 系统。与其他 AI 解决方案不同，Poolside 提供从 AI 模型到代理编排平台的完整技术栈，完全部署在客户自己的基础设施内，确保数据不会离开客户网络。这对于金融服务、公共部门等高度监管行业尤为重要。公司通过 AWS 的一级合作伙伴协议提供服务，客户可以使用现有的 AWS 协议和承诺支出来购买 Poolside 服务。

会议展示了三个实际应用场景：单代理在 IDE 中将 Ada 代码转换为 Rust、多代理并行处理 GitHub issues、以及多代理链式编排诊断 EKS 集群问题。诺斯罗普·格鲁曼的案例证明了该解决方案在航空航天等关键任务系统中的实际应用价值，特别是在完全气隙环境中维护数据主权和知识产权保护的能力。
