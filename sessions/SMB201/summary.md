# AWS re:Invent 2025 会议总结：NextWorld 使用 SageMaker Autopilot 实现 AutoML

## 会议概述

本次会议由 AWS 商业部门西区销售负责人 Valerie Canafo 介绍，主讲人为 NextWorld 工程与 AI 高级总监 Luke Hollandbeck。会议重点介绍了 NextWorld 如何利用 AWS SageMaker Autopilot 构建企业级 AutoML 解决方案，实现了传统机器学习流程的革命性改进。

NextWorld 是一家企业应用平台公司，提供基于云的无代码开发平台，专注于 ERP（企业资源规划）等复杂业务应用。通过集成 SageMaker Autopilot，NextWorld 成功将机器学习模型的构建时间从 4 周缩短至 2 天，准确率平均提升 33%，实现了 95% 的构建时间改进。更重要的是，这使得非技术人员和业务分析师也能够独立构建和部署高质量的机器学习模型，无需数据科学家的深度参与。

该解决方案在发布当月就取得了显著的商业成功：所有到期续约的客户不仅全部续约，还升级购买了 AutoML 功能，平均合同价值增长 54%。在实际应用案例中，一家器官移植评估公司使用 AutoML 将模型准确率从 60% 提升至 90%，仅用了 2 天时间，且只使用了 120 个维度中的 12 个，充分展示了自动化特征工程的强大能力。

## 详细时间线与关键要点

0:00 - 0:45 | 开场介绍
- Valerie Canafo 介绍 NextWorld 公司背景
- NextWorld 是基于 AI 驱动的无代码业务应用平台
- 运行在 AWS 上，使用 SageMaker 技术

0:45 - 2:15 | NextWorld 平台核心能力
- AI 原生平台：使用 AI 辅助构建应用程序
- 提供从传统机器学习到现代 LLM 和视觉模型的全方位 AI 功能
- 自动通过 MCP（模型上下文协议）暴露应用能力
- 企业级基础设施，全托管服务
- ERP 背景，擅长处理复杂业务应用

2:15 - 3:30 | 传统机器学习的局限性
- NextWorld 原有传统 ML 功能需要数据科学专业知识
- 开发周期长，通常需要数周时间
- 需要手动选择算法、调参、训练和部署
- 面向数据科学家，不适合普通业务用户

3:30 - 4:45 | AutoML 探索与选型
- 团队开始探索 AutoML 概念，自动化数据科学生命周期
- 同时进行自研和外部方案评估
- AWS SageMaker Autopilot 脱颖而出
- Autopilot 不仅满足 AutoML 需求，还提供多租户支持和自动化运维能力

4:45 - 6:00 | 实施效果与业务价值
- 构建时间改进 95%：从 4 周缩短至 2 天
- 平均准确率提升 33%
- 产品负责人（非数据科学家）即可独立完成模型构建
- 发布当月，10% 的续约客户 100% 续约并升级
- 平均合同价值增长 54%

6:00 - 8:15 | 器官移植案例研究
- 客户场景：评估遗体器官移植可行性，时间敏感且关乎生命
- 原始手工方案：准确率约 60%（不适合生产环境）
- 使用传统 ML：数据科学家团队 4 周达到 75% 准确率
- 使用 AutoML：产品负责人 2 天达到 90% 准确率
- 特征工程自动优化：从 120 个维度降至 12 个关键维度
- 速度提升 93%，准确率提升 25%

8:15 - 10:30 | 传统数据科学流程对比
- 传统 CRISP-DM 流程：数据收集 → 探索 → 特征工程 → 算法选择 → 调参训练 → 测试 → 迭代优化 → 部署监控
- 需要高技能数据科学家，耗时数周
- 普通业务分析师无法独立完成

10:30 - 11:45 | AutoML 简化流程
- 用户只需：指向数据源 → 选择要预测的特征 → 点击运行
- 几分钟内完成训练（而非数周）
- 返回 8 个优化模型候选
- 使用 LLM 生成自然语言描述，解释每个模型的优化方向
- 针对不同业务场景优化（如假阳性 vs 假阴性权衡）

11:45 - 13:00 | 技术架构与关键需求
- 必须支持 API 访问（多租户系统需求）
- 完全代码驱动，无需人工干预
- 使用 Serverless 架构降低成本
- 自动生成模型分析报告和可视化图表
- 架构简单：浏览器客户端 → NextWorld 服务器 → Aurora RDS + SageMaker Autopilot Serverless 端点

13:00 - 14:30 | 总结与未来展望
- AutoML 已成为 NextWorld 机器学习的主要方式
- 传统 ML 仍然重要，与现代 LLM 互补
- 确定性预测在业务应用中至关重要
- 可集成到智能体系统中
- 数据科学家可专注于平台功能开发，而非重复性模型构建工作

14:30 - 15:00 | 问答与交流
- 讲者邀请参会者现场交流技术细节
- 推荐了解 NextWorld 平台和 SageMaker Autopilot（原 SageMaker AutoML）
