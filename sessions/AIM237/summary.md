# AWS re:Invent 2025 会议总结：使用 AI 加速可持续发展合规审查

## 会议概述

本次会议由 AWS 解决方案架构师 Kent Sato 主讲，主题是如何利用 AI 技术将可持续发展监管从业务约束转变为业务能力。演讲者指出，虽然监管对于推动可持续发展转型至关重要，但繁琐的文书工作和官僚流程往往成为企业的负担。会议展示了一个名为 Rapid（Rapid Review and Assessment Powered by Intelligent Documentation）的开源解决方案，该方案可以显著简化大量文档审查流程。

Rapid 采用两阶段方法解决监管合规中的文档审查难题：第一阶段使用 AI 提取和结构化检查清单，并由人工验证；第二阶段针对每个检查项逐一处理相关文档，再由人工验证结果。这种人机协作的方式有效解决了 AI 在处理长文本时的"中间遗漏"问题和责任归属问题。该解决方案基于 AWS 无服务器架构构建，在早期试点项目中实现了高达 75% 的人工审查时间减少。

会议最后介绍了 Rapid 的实施路线图，从业务验证、概念验证到生产部署的完整流程，并提供了 GitHub 开源代码库和一键部署方案，鼓励参会者立即开始使用这一解决方案来加速可持续发展项目的交付。

演示：第一阶段 - 检查清单提取 (10:00 - 13:00)

- 14:30 - 展示审查结果：包括成本、通过/失败状态和置信度分数
- 15:00 - 查看详细推理：AI 的决策依据和源文本引用
- 15:30 - 示例：额定功率与铭牌匹配检查通过
- 16:00 - 低置信度示例：检查清单缺乏具体标准导致失败
