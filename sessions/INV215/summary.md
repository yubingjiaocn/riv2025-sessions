# AWS re:Invent 2025 存储与数据服务技术会议总结

## 会议概述

本次技术会议由AWS副总裁兼杰出工程师Andy Warfield主持，重点介绍了AWS存储和数据服务在过去一年中的重要进展和创新。会议深入探讨了AWS如何通过构建块（building blocks）理念来设计和优化存储服务，强调了在安全性、持久性、可用性、性能和弹性等基本原则指导下的持续创新。

Andy分享了S3服务19年发展历程中的技术演进，包括底层硬件架构的重大变革、新数据类型支持的扩展，以及面向企业级应用的解决方案。会议特别突出了S3在处理超过500万亿个对象、每秒2亿次请求的超大规模下，如何通过持续的技术创新来满足不断变化的客户需求。

## 详细时间线与关键要点

### 0:00-10:00 开场与构建块理念
- Andy Warfield介绍AWS存储和数据团队的工作理念
- 阐述"构建块"概念：为开发者提供无需过多思考的基础服务
- 强调成功的衡量标准是减少摩擦，让开发者专注于更高层次的工作
- S3服务规模数据：存储超过500万亿个对象，每秒处理2亿次请求

### 10:00-20:00 S3条件操作与小功能改进
- 详细介绍2024年8月推出的S3条件PUT操作
- 该功能占所有PUT请求的2%，每分钟处理数千万次请求
- 客户反馈：该功能帮助删除大量复杂代码，简化分布式应用架构
- 其他改进：最大对象大小增至50TB，新增条件复制和删除操作，S3 Express增加原子重命名功能

### 20:00-30:00 底层硬件创新：Metal Volumes
- 介绍从传统JBOD架构向Metal Volumes的重大转变
- Metal Volumes通过Nitro计算节点安全虚拟化物理磁盘
- 实现计算与存储解耦，提高可用性和灵活性
- Nitro Offloads技术减少60%的网络流量
- 整体功耗节省超过10%

### 30:00-40:00 S3 Express性能优化与客户案例
- S3 Express价格降低85%，性能提升至每秒200万次请求
- Meta客户案例：60PB存储，每秒100万次事务，140Tbps流量
- 新增对象重命名和访问点支持功能

### 40:00-50:00 S3 Vectors向量数据库服务
- 正式发布S3 Vectors，预览期间创建25万个向量索引
- 摄入400亿个向量，处理超过10亿次查询
- GoPro客户案例：450PB媒体资产，每日增长100万个资产
- 通过向量搜索解决长视频内容中关键时刻发现问题

### 50:00-54:00 S3 Tables与企业迁移解决方案
- S3 Tables已托管40万个表，Indeed客户每日摄入0.5PB数据
- 新增跨区域复制、智能分层优化、Iceberg V3支持
- S3元数据服务提供SQL查询能力，支持自然语言转SQL
- FSX S3访问点扩展支持NetApp ONTAP，简化企业数据迁移
