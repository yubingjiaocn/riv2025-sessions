# AWS re:Invent 2025 - 日产汽车软件定义车辆开发分会场总结

## 会议概述

本次分会场由日产汽车的Kazuma Soto和AWS首席解决方案架构师Kazuo Kajimo共同主讲，主题为"日产如何利用AWS加速软件定义车辆(SDV)开发"。会议首次全球发布了日产的SDV平台完整架构名称——"Nissan Scalable Open Software Platform"（日产可扩展开放软件平台），展示了日产在汽车行业数字化转型中的战略布局。

日产将人工智能定位为SDV转型的战略推动力和汽车行业竞争力的关键领域。通过与AWS的深度合作，日产构建了涵盖工具云、数据平台和虚拟ECU的完整开发生态系统。该平台不仅支持车辆的持续升级和个性化定制，还为日产建立了新的经常性收入来源。日产计划在2027年向客户交付搭载AI驱动功能的下一代车辆，实现从门到门的自动驾驶体验。

会议重点介绍了工程云(Engineering Cloud)在AWS上的实现，通过CI/CD自动化将测试时间缩短了75%，支持全球超过5000名开发人员的协作。日产还展示了基于生成式AI的未来愿景：车辆能够根据驾驶员的语音指令，通过云端AI实时分析并自动生成定制化软件，实现真正的个性化驾驶体验。

12:30 - CI流程自动化

- 不仅在虚拟环境，也在物理ECU上运行
