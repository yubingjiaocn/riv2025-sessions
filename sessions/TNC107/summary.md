# AWS re:Invent 2025 AI技能培训会议总结

## 会议概述

本次AWS re:Invent 2025技术会议由AWS培训产品总监Umesh Sampat和桑坦德银行代理CTO Julio Blanco Perez共同主讲，主题聚焦于AI技能提升和企业培训转型。会议深入探讨了当前AI技能培训面临的挑战与机遇，详细介绍了AWS Skill Builder平台的AI驱动学习体验，并通过桑坦德银行的实际案例展示了大规模AI技能培训的成功实践。

演讲者强调了持续学习在AI时代的重要性，指出技能半衰期正在急剧缩短，从1980年代的10年缩短至目前的4年，预计未来2-4年内可能进一步缩短至2年。同时，拥有AI技能的专业人士薪酬可提升47%，这为个人职业发展提供了巨大机遇。桑坦德银行通过与AWS合作，在巴西成功实施了Level UP培训项目，将AWS认证数量从120个增长到700个，并将培训理念从传统强制性学习转变为协作式团队学习。

## 详细时间线与关键要点

### 0:00-10:00 开场介绍与挑战分析
- **演讲者介绍**：Umesh Sampat（AWS培训产品总监）和Julio Blanco Perez（桑坦德银行代理CTO）
- **核心统计数据**：
  - 世界经济论坛报告：到2030年，40%的核心工作技能将发生变化
  - AWS研究显示：92%的组织计划在2028年前实施AI项目，但75%表示找不到合适人才
  - 技能半衰期从1980年代的10年缩短至2024年的4年，预计将进一步缩短至2年
- **机遇分析**：拥有AI技能的专业人士薪酬可提升47%

### 10:00-20:00 现代学习者旅程演进
- **学习模式转变**：
  - 传统技能与人文技能相结合
  - 微学习成为主流（15分钟-2小时的学习片段）
  - 学习-实践循环变得更加紧密和持续
- **学习周期重构**：
  - 传统瀑布式：学习→实践→准备→认证
  - 新模式：持续的学习-实践微循环，积累知识后进入准备和认证阶段

### 20:00-30:00 AWS Skill Builder产品详解
- **学习阶段产品**：
  - 330+AI技能资源，涵盖技术技能和人文技能
  - AI驱动的沉浸式学习体验（SimuLearn、Meeting Simulators）
- **实践阶段产品**：
  - **Cloud Quest**：3D游戏化学习环境，在虚拟城市中学习AWS服务
  - **SimuLearn**：模拟真实场景，与AI虚拟代理互动，获得技术和沟通技能反馈
  - **Meeting Simulator**：11月12日发布，模拟高风险对话场景，提供实时沟通反馈
  - **Builder Labs和Jam sessions**：在真实AWS环境中实践
  - **Card Clash**：游戏化团队学习产品

### 30:00-35:00 认证体系与微认证
- **四大AI认证**：
  - AI Practitioner Foundational（基础级）
  - Machine Learning Engineer Associate（机器学习工程师）
  - Data Engineer Associate（数据工程师）
  - Generative AI Professional（11月18日发布Beta版）
- **认证价值**：91%的组织认为AWS认证员工提升了创新能力，90%+认为降低了成本
- **微认证**：11月12日发布，在真实AWS环境中进行实时评估，目前包括AWS Agentic AI Demonstrated和AWS Serverless Demonstrated

### 35:00-40:00 团队协作学习与Cohort Studio
- **团队学习产品**：
  - Cloud Quest tournaments（团队竞赛）
  - AWS Jams（开放式实验室挑战）
  - AWS Card Clash（头对头知识竞赛）
  - Escape Room（团队认证考试准备）
  - Skill Builder Trivia（大规模团队竞赛）
- **Cohort Studio**：11月12日发布，15分钟内启动团队学习项目，AI驱动的综合学习管理平台

### 40:00-48:00 桑坦德银行案例分享
- **公司背景**：180年历史的全球银行，2亿客户，2万亿欧元管理资产，2018年起与AWS合作
- **Level UP项目**：
  - 混合传统学习与自主学习的框架
  - 3-9个月培训周期，分配学习小组
  - 结合Skill Builder自主学习与团队推动机制
- **项目成果**：
  - AWS认证数量从120个增长到700个
  - 数千人参与培训，数百人完成认证
  - 培训文化从强制性转向协作性
- **AWS Experience活动**：类似re:Invent的内部技术大会，包含21场技术讲座、DeepRacer竞赛、逃脱室等
- **业务应用案例**：通过AI技术改造投资顾问工作流程，客户接触量提升1.2倍，准备时间从长时间缩短至30秒-1分钟
