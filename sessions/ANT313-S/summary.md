# AWS re:Invent 2025 分会场总结

## 会议概述

本次会议由 Impetus Technologies 的解决方案架构团队负责人 Sachnid 主讲,主题聚焦于如何通过自动化现代化改造帮助企业释放生成式AI的潜力。Impetus Technologies 是AWS的顶级服务和解决方案合作伙伴,在ETL迁移领域排名第一,服务超过30家财富500强客户,在AWS市场上架了10多个服务和解决方案。

演讲核心围绕企业如何将积累数十年的遗留系统数据迁移到AWS平台,为生成式AI时代做好准备。根据调研数据,90%的企业领导者认为在进入AI领域之前,首先需要优先考虑遗留系统的现代化改造,因为这些系统(如EDW、ETL)已经运行了数十年,积累了大量业务逻辑。Impetus通过其核心工具解决方案加速器 Leap Logic,实现了90-95%的自动化代码转换,帮助企业将遗留系统快速迁移到AWS现代化平台(如Glue、Redshift等),同时确保零业务中断。

此外,演讲还介绍了Prism这一行业首个AI驱动的统一可观测性解决方案,帮助企业在迁移到云端后持续优化成本,提供多云和数据平台的统一视图,并通过AI提供预测性分析和智能化建议。
