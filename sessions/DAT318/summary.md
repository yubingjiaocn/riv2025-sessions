# AWS re:Invent 2025 Web3支付协议技术分享会总结

## 会议概述

本次技术分享会是AWS re:Invent 2025第五天的300级别会议，主要探讨了Web3、数字资产支付和代理支付(agentic payments)技术。会议由AWS的Arvvent主持，并邀请了Coinbase开发平台工程负责人Eric Repel和AWS高级解决方案架构师Joshua Smith共同分享。

会议重点介绍了AWS在Web3领域的战略布局，强调Web3不仅仅是加密货币，而是一个更广泛的去中心化技术生态系统。AWS通过托管服务、自助解决方案和合作伙伴生态系统三大支柱，为客户提供构建Web3产品所需的基础设施和工具。特别值得关注的是，会议深入讨论了Coinbase开源的X42支付协议，这是一个专为互联网原生支付设计的开放标准，能够实现低成本、快速结算的数字资产支付。

## 详细时间线及要点

- **00:30** - 主持人Arvvent感谢观众参加周五上午的技术分享

- **02:00** - 调查观众Web3背景，发现大部分人不熟悉Web3技术
- **03:30** - 介绍演讲嘉宾：Joshua Smith(AWS)和Eric Repel(Coinbase)

### AWS Web3战略概述 (05:00-15:00)
- **05:30** - 澄清Web3不仅仅是加密货币，涵盖更广泛的去中心化应用

- **12:30** - 介绍AWS三大支柱：托管服务、自助解决方案、合作伙伴生态

- **33:30** - 强调微支付的可能性和为无银行账户人群提供服务的潜力
