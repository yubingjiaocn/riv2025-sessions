# AWS re:Invent 2025 - 使用 Kiro 和 MCP 服务器加速构建无服务器应用程序

## 会议概述

本次技术分享会由 AWS 的两位解决方案架构师 Sean Kendall 和 Brian Zambrano 主讲，重点介绍了如何使用 Amazon Kiro（原 Amazon Q Developer）和模型上下文协议（MCP）服务器来加速无服务器应用程序的开发。Sean 专注于无服务器和生成式 AI，拥有超过 6 年的 AWS 经验；Brian 来自生成式 AI 创新中心，负责为客户构建生成式 AI 概念验证和 MVP。

会议通过构建一个井字棋游戏的后端来演示技术实现，展示了 Kiro CLI 和 Kiro IDE 两种开发方式。讲师们强调了无服务器架构结合 AI 工具的优势，能够在极短时间内（约 5 分钟）完成完整的后端开发和部署。整个演示过程具有很强的互动性，鼓励观众随时提问。
