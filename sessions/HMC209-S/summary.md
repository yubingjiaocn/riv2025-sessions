# AWS re:Invent 2025 混合多云高性能安全网络构建会议总结

## 会议概述

本次技术会议由 Equinix 产品经理 Brandon Wright 和 AWS 网络市场负责人 Ranga Thittai 共同主讲，重点探讨了如何为混合云和多云环境构建高性能、安全的网络架构。会议涵盖了 AWS Direct Connect 的客户使用案例、数据迁移考量因素，以及 Equinix Fabric 网络即服务平台的解决方案。

演讲者详细介绍了大规模数据迁移的重要性，特别是在生成式 AI、模型训练、RAG 向量存储构建等场景下的应用需求。会议还展示了私有混合多云连接的实际案例研究，包括金融服务和自动驾驶车辆行业的成功实施经验，为企业提供了实用的网络架构指导。

## 详细时间线与关键要点

### 0:00-2:00 会议开场与议程介绍
- Brandon Wright（Equinix 产品经理）和 Ranga Thittai（AWS 网络市场负责人）介绍
- 会议议程概览：Direct Connect 客户使用案例、数据迁移考量、私有混合多云解决方案

### 2:00-5:00 大规模数据迁移的重要性
- 生成式 AI 应用场景：模型训练、基础模型开发、RAG 向量存储构建
- 行业数据需求：自动驾驶车辆、智能制造、媒体数字化和云端制作
- 企业迁移项目：6-9个月内迁移数百 PB 数据到 AWS 的常见需求

### 5:00-8:00 AWS Direct Connect 解决方案
- 专用网络路径避免互联网传输，提供一致性能表现
- 可用性 SLA 保障，覆盖 30+ 国家 100+ 位置
- MAC 加密（MACSEC）支持合规性要求
- 性能对比：25Gbps 托管 Direct Connect 1小时完成 10PB 传输，VPN 仅完成 5%

### 8:00-12:00 Equinix 公司背景与 Fabric 平台
- 26年历史，从互联网交换起步发展为中性互联服务商
- 11,000+ 企业客户使用平台
- Equinix Fabric：覆盖 36 国家 270+ 数据中心的全球软件定义网络
- 220+ 云接入点，包括 56 个 AWS Direct Connect 连接点
- 五个九（99.999%）全球正常运行时间 SLA

### 12:00-15:00 Fabric Cloud Router 产品介绍
- 2024年1月推出的完全基于服务的解决方案
- 无需虚拟设备、许可证或客户硬件
- 分钟级按需部署，原生支持高可用连接
- 吞吐量范围：50Mbps 到 100Gbps 单连接，聚合吞吐量超过 1Tbps

### 15:00-17:00 性能测试与网络架构
- 跨地域连接测试：西雅图到阿什本，平均延迟 63 毫秒
- 四跳可预测路径 vs 互联网的不确定路由
- 线速性能：9.5Gbps TCP 和接近线速的 UDP 传输
- 私有网络的可预测性能和抖动控制

### 17:00-18:00 客户案例研究与试用信息
- 金融服务案例：4小时内完成 AWS-Azure 互联，提升汽车贷款响应速度
- 自动驾驶车辆案例：每日 500TB 数据传输，使用多个 25Gbps Direct Connect 电路
- AWS Marketplace 提供 7 天免费试用，包含两个 200Mbps 虚拟连接
- Equinix 展位 335 号提供现场咨询服务
