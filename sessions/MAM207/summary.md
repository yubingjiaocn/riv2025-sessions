# AWS re:Invent 2025 MAM 207 会议总结

## 会议概述

本次会议主题为"使用 AgentKI 重新构想您的大型机工作负载"，由 AWS 大型机现代化全球技术负责人 Paulo 主持，并邀请了 ADP 产品开发副总裁 Anu Verma 和 Itaú 银行平台执行官 Douglas Santos 分享他们的实际经验。

会议重点介绍了大型机现代化的"重新构想"(Reimagine)模式，这是一种不同于传统重新平台化(Replatform)和重构(Refactor)的方法。重新构想模式允许客户重新思考业务流程和应用程序架构，退役过时的业务规则，添加新的功能，并从批处理架构转向事件驱动的实时架构。AWS 推出了 AWS Transform 和 Kira 等代理式 AI 工具来帮助客户应对重新构想过程中的知识危机和手工流程挑战。
