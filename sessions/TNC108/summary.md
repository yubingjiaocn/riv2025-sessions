# AWS re:Invent 2025 认证价值讨论会议总结

## 会议概述

本次会议探讨了AWS认证对企业战略的价值。主讲人Andre Zoot是AWS解决方案架构师，专门负责认证团队工作。特邀嘉宾Agusto Straieri来自巴西Itaú银行，负责该行的技能提升和认证项目。会议重点讨论了认证如何帮助企业而非仅仅个人，以及如何利用认证为公司带来实际业务收益。

Andre强调，许多公司因缺乏合适的IT人才而无法完成项目，导致项目延期6个月甚至更长，这会严重影响收入、客户满意度，并给竞争对手提供颠覆行业的机会。通过系统的认证培训计划，企业可以建立技能基线，加快进入新业务领域的速度，缩短项目交付时间。Itaú银行的成功案例展示了如何通过大规模认证计划实现数字化转型，取得了事故减少99%、速度提升2000%以上、单位成本降低45%以上的显著成果。

会议还介绍了AWS认证体系的四个层级（基础级、助理级、专业级和专项级），以及新推出的微证书（Micro Credentials）和生成式AI开发者认证。Andre建议企业建立预算、制定职业发展路径、提供学习资源（如AWS Skill Builder）、建立导师计划，并通过激励措施鼓励员工获取认证。

开场介绍 (0:00-2:30)
- Andre Zoot自我介绍，说明自己是AWS解决方案架构师，隶属于认证团队

- 会议主题：认证对企业战略的价值

IT人才短缺问题 (2:30-4:00)
- 提问：有多少公司因缺乏合适的IT人才而无法完成项目
- 讨论项目延期6个月对收入、客户满意度和竞争力的影响

认证对企业的益处 (4:00-7:30)
- 安全团队案例：已实施AWS最佳实践的公司更容易进入有监管要求的新业务领域
- 机器学习和AI项目：项目交付时间以天和周计算，没有时间进行在职培训
- 认证提供跨服务的基线理解，即使对不常用的服务也有基本认知

认证对员工的价值 (7:30-9:00)
- 培训有助于员工留任
- 提升竞争优势
- 更好的培训带来更好的业务成果

AWS认证体系介绍 (9:00-13:30)
- 认证是通过考试获得，有效期3年
- 四个层级：
  - 基础级（Foundational）：适合管理者和非技术人员
  - 助理级（Associate）：建议有1年经验，5个认证
  - 专业级（Professional）：建议有2年经验，更复杂
  - 专项级（Specialty）：深入特定领域

AI和机器学习认证 (13:30-16:00)
- AI是本届re:Invent的主要主题
- AI从业者认证：帮助理解AI语言
- 机器学习工程师认证：训练和运营模型
- 生成式AI开发者认证（Beta）：开发AI应用和代理，专注于RAG架构和向量数据库
- Beta认证：85道题，210分钟，半价

- 101年历史，巴西最大银行，南美最有价值公司

数字化转型四大支柱 (18:30-20:00)

- 各级别认证：从业者、助理、专业和专项

- 关注残障人士的包容性

AWS周活动 (29:30-31:00)
- 每年停止所有IT工作一周举办AWS周
- 45场会议：高管小组、主题演讲、女性科技和包容性小组
- 邀请AWS认证团队现场举办考试
- 三天内获得超过300个认证

Capital One引用 (31:00-31:30)
- George Brady的名言强调认证对业务的价值

如何实施认证计划 (31:30-34:00)
- 留住优秀员工比招聘新员工更重要
- 使用认证筛选面试候选人
- 根据认证级别调整面试问题难度

新员工入职 (34:00-35:30)
- 新员工不可能具备所有技能
- 使用认证验证培训效果
- AWS的做法：创建半年到一年的入职计划，期望员工通过相关认证

员工激励 (35:30-37:30)
- 提问：有多少公司在员工通过认证后提供奖励
- 案例：网络工程师因参加认证考试迟到被批评，后来跳槽到提供认证奖励的公司
- 建议：提供餐厅礼品卡等小额奖励，投入产出比极高

绩效提升和职业发展 (37:30-39:00)
- 认证后的员工期望承担新责任
- 提供使用新技能的机会
- 导师计划：教授他人是巩固知识的最佳方式

微证书介绍 (39:00-41:30)
- 与传统认证不同，需要实际操作演示
- 在AWS Skill Builder上启动，在真实AWS控制台中完成任务
- 无风险环境，有防护栏
- 完成后可链接到Credly账户和LinkedIn
- 专注于特定领域（如无服务器、代理AI）

实施步骤：预算 (41:30-43:30)
- 考试费用预算
- 员工激励预算
- 学习时间预算：每周安排2小时学习时间
- 批量购买考试券可获得折扣
- 考试券有过期日期，可激励员工按时参加考试

职业发展路径 (43:30-45:30)
- 系统管理员到云解决方案架构师路径：云从业者 → 解决方案架构师助理 → 解决方案架构师专业
- 云开发者路径：云从业者 → 开发者助理 → DevOps工程师专业
- 可为不同角色定制不同路径

学习资源 (45:30-48:00)

- AWS Skill Builder平台（skillbuilder.aws）
- 免费版本：600多门自学课程和部分实验
- 订阅版本：更多实验和其他功能
- 实验特点：真实AWS控制台环境，有防护栏，完成后环境清除

实验类型 (48:00-49:00)
