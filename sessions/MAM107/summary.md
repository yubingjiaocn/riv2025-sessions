# AWS re:Invent 2025 - AWS Marketplace 会议总结

## 会议概述

本次会议由 AWS Marketplace 客户专家 Randolette Jr. 和产品经理 Ryan Worley 主讲，全面介绍了 AWS Marketplace 的核心价值和最新功能。AWS Marketplace 是一个一站式平台，帮助客户发现、购买和部署第三方软件、专业服务、数据和 AI 代理等解决方案。目前平台拥有超过 6,000 家卖家和约 30,000 种产品，涵盖 70 多个类别。

根据 2025 年 5 月 AWS Marketplace 与 Forrester 的联合研究显示，客户使用该平台可获得三大核心收益：产品发现节省 70% 时间、通过统一账单简化采购流程节省 60% 时间、以及通过高效部署使产品上市时间缩短 30%。平台提供多种部署方式，包括 EC2 的 Amazon Machine Images、容器镜像以及 SaaS 产品等，满足不同技术栈需求。

会议重点介绍了 2025 年 re:Invent 大会上发布的三项重要新功能：Agent Mode（AI 代理模式）、多产品解决方案打包，以及持续扩展的 AI 代理和工具市场。这些创新功能旨在进一步简化采购流程，提升客户体验，并帮助企业更快速地实现业务目标。

## 详细时间线

00:00 - 开场介绍
- Randolette Jr. 和 Ryan Worley 介绍自己的角色
- 说明 AWS Marketplace 是帮助客户发现、购买和部署第三方软件、专业服务、数据和 AI 代理的一站式平台
- 平台拥有 6,000 家卖家和约 30,000 种产品

01:30 - 核心价值主张
- 强调灵活的定价和付款条款
- 提供快速部署解决方案的能力
- 为采购团队提供控制和治理功能，降低风险敞口

02:15 - Forrester 研究成果
- 产品发现节省 70% 时间：通过集中平台评估供应商和解决方案
- 简化采购节省 60% 时间：通过统一账单和标准合同
- 上市时间缩短 30%：通过高效的部署选项和 SaaS 快速启动

03:45 - 卖家生态系统
- 包括独立软件供应商（ISV）和渠道合作伙伴
- 超过 30,000 种产品，涵盖 70 多个类别
- 帮助客户找到推动创新所需的解决方案

04:30 - 产品发现和评估功能
- 详细的供应商页面，包含产品功能、评论等信息
- 提供超过 2,000 个解决方案的演示和免费试用
- 可直接在控制台中请求私有报价，与供应商或经销商对话

05:45 - 基础设施解决方案
- 涵盖技术栈的每一层：DevOps、AI、安全、数据分析和专业服务
- 提供多种部署方式：EC2 的 AMI、容器镜像、SaaS 产品等

06:30 - AI 代理和工具市场
- 2025 年 7 月宣布推出 AI 代理和工具市场
- 目前已有超过 2,000 个 AI 代理
- 可用于内容创作、安全分析等多个领域
- 支持 AI 和语义搜索选项

07:15 - 产品发现演示
- 演示搜索基础设施即代码（Infrastructure as Code）解决方案
- 展示 Terraform MCP 服务器产品详情页
- 演示快速订阅流程（该软件为免费）
- 展示不同的启动选项：通过 ECS 或 Bedrock Agent Core 部署

09:00 - re:Invent 2025 新功能发布
- **Agent Mode（代理模式）**：使用 AI 在 AWS Marketplace 中搜索业务问题的解决方案
- **多产品解决方案**：允许在单个私有报价中组合专业服务和供应商产品
- **AI 代理市场扩展**：持续增长，帮助客户评估和比较解决方案

10:30 - Agent Mode 详细介绍
- 通过提供业务用例节省产品查找时间
- 智能搜索 30,000 个列表，提供最相关的推荐
- 支持深入洞察和解决方案比较功能
- 可基于业务契合度、AWS 标准合同、功能等进行比较
- 提供创建提案功能，利用 AI 帮助获得利益相关者批准

12:00 - 多产品解决方案打包
- 渠道合作伙伴和 ISV 现在可以将软件和服务打包在一起销售
- 为客户提供完整的业务解决方案，而非单个产品

12:45 - 服务控制台集成
- AWS Marketplace 在超过 10 个服务控制台中可用
- 可在网站或各种服务控制台中发现和部署软件

13:15 - EC2 控制台演示
- 展示在 EC2 控制台中浏览 Amazon Machine Images
- AWS Marketplace 在 EC2 控制台中提供超过 6,000 个机器镜像
- 演示查找和部署 Linux 服务器的过程
- 展示在几分钟内快速启动实例并连接

15:30 - 定价和支付模式
- 提供免费试用
- 支持基于使用量和消费的合同
- 支持自带许可证（BYOL）功能，用于迁移现有产品许可证
- 提供月度、年度或半年度合同
- 长期合同可节省高达 20% 或更多
- 支持灵活支付和私有报价

17:00 - 三种主要购买方式
- **自助服务**：直接在控制台中审查和接受标准合同，无需销售接触
- **ISV 定制定价（私有报价）**：与供应商直接协商 SKU 和产品，在控制台中接受
- **首选渠道合作伙伴私有报价**：通过 2,500 家经销商（如 CDW、SHI）继续采购

18:45 - 支出管理功能
- 实例标签（成本中心、业务单位）会显示在 AWS 成本和使用报告中
- 采购洞察仪表板：在 AWS Marketplace 控制台中预构建，免费使用
- 单一视图仪表板：基于 AWS 成本和使用报告构建
- 采购订单管理：可为 AWS Marketplace 支出使用独特的采购订单
- 2025 年 11 月 19 日推出电子发票功能，支持 SAP Ariba 或 Coupa

20:15 - 治理功能
- 自定义目录：可定制最终买家可购买的产品目录
- 安全审查：可要求在产品进入市场前进行审查，从零开始构建目录
- 集中账单：所有 Marketplace 和云发票集中在付款账户中
- 许可证分发：通过 License Manager 集中购买和分发许可证

21:30 - 客户案例
- EasyJet 高级技术经理 Simon Chalice 的评价
- 强调 AWS Marketplace 提供的敏捷性，能够快速部署服务以应对突发事件

22:15 - 学习资源和结束
- 提供三个 QR 码：
  - 新 AI Agent Mode 功能
  - 超过 2,000 个免费产品
  - Marketplace 一般知识和业务目标
