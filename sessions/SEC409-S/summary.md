# AWS re:Invent 2025 分会场总结：利用Teramind防范内部威胁

## 会议概述

本次分会场由Aria公司收入诚信与运营合规副总裁Justin Skagan主讲，他同时也是一名注册舞弊审查师(CFE)。会议重点介绍了Aria公司如何利用Teramind员工监控软件成功侦破内部欺诈案件，并将该工具从试点项目扩展到1500名员工的实施经验。

Aria是一家白标旅游代理公司，拥有超过2100名全球员工，主要为合作伙伴提供旅游预订服务。由于员工可以接触到客户的信用卡信息和积分账户，公司面临着严重的内部欺诈风险。Justin分享了一个关键案例：一名墨西哥办公室员工涉嫌构建客户信息列表并计划出售给诈骗团伙（可能与墨西哥卡特尔组织有关）。传统调查方法需要3-4周时间，但通过Teramind，团队在2天内就完成了调查并阻止了数据泄露。

该工具不仅用于欺诈检测，还被应用于提升生产力、优化服务水平、员工培训等积极场景。公司建立了治理委员会来管理监控权限，并采取了HIPAA和PCI合规措施来保护敏感信息，成功克服了员工对"老大哥监控"的抵触情绪。

## 详细时间线与关键要点

00:00:00 - 演讲者介绍
- Justin Skagan自我介绍：Aria公司收入诚信与运营合规副总裁
- 注册舞弊审查师(CFE)认证
- 分享个人背景：童年时因电影《Hudson Hawk》想成为艺术品窃贼，后转向法医学，最终进入反欺诈领域

00:03:30 - Aria公司背景
- 白标旅游代理公司，运营20余年
- 从分时度假交换业务起步，现提供全方位旅游服务
- 与Spirit等合作伙伴的积分兑换项目
- 全球拥有2100多名员工
- 作为交易记录商户，面临多层次风险

00:05:45 - 欺诈三角理论
- **机会(Opportunity)**：欺诈行为是否存在可乘之机
- **动机(Motive)**：通常由经济利益驱动
- **合理化(Rationalization)**：欺诈者的自我辩护逻辑，如"无受害者犯罪"或"公司欠我的"

00:08:20 - 触发调查的案件
- 发现多个会员账户出现欺诈性酒店预订
- CRM系统显示特定员工修改了账户信息（姓名、电话、邮箱）
- 员工承认共享登录凭证（严重违反安全规定）
- 员工开始互相指认同谋

00:10:15 - 发现可疑员工
- 一名员工的账户查看次数异常高（是其他员工的10-20倍）
- 但这些账户上没有任何旅游预订记录
- 怀疑该员工在构建潜在客户名单用于出售

00:11:30 - 墨西哥卡特尔诈骗背景
- 墨西哥办公室案件
- 卡特尔组织向呼叫中心员工支付现金（每条线索50美分至1美元）购买客户信息
- FBI确认这是卡特尔组织当前主要收入来源之一
- **分时度假诈骗流程**：
  - 承诺以10-15万美元购买分时度假产权
  - 要求支付5000美元"过户费"
  - 持续要求追加费用直到受害者破产
  - 二次诈骗：假装帮助追回损失，再次收费

00:15:45 - 调查困境
- 墨西哥劳动法要求必须有确凿证据才能解雇员工
- 无证据解雇需支付高额遣散费或面临诉讼
- 传统调查方法需要：
  - IT部门提取Teams消息、Outlook邮箱、OneDrive文件
  - 人工筛查数百个Excel文档寻找隐藏数据
  - 预计耗时2周获取数据 + 1周分析 = 3-4周总时长
- 担心客户信息列表在此期间被转移

00:18:30 - 引入Teramind
- 劳动力管理部门同事介绍正在试点的Teramind计算机监控软件
- **关键挑战**：2100名员工中仅20人在试点范围内
- **幸运突破**：可疑员工恰好在试点名单中

00:19:45 - Teramind功能展示
- 形容为"游戏中的上帝模式"
- 可以看到员工电脑上的一切活动（包括孩子玩Roblox、看迪士尼电影）
- 记录所有应用程序、网站访问及停留时长
- 发现大量非工作时间访问CRM系统的行为

00:21:00 - 关键证据发现
- 实时观察员工操作流程：
  1. 进入客户账户
  2. 停留几秒钟
  3. 复制PII（个人身份信息）
  4. 粘贴到Excel文件
- **高级隐藏技巧**：
  - 将字体颜色改为白色（视觉上看不到内容）
  - 隐藏工作表标签
  - 如果没有Teramind，通过OneDrive查找这些数据将极其困难

00:23:15 - 调查成果
- 从3-4周缩短到**2天**完成调查
- 精确定位需要提取的Excel文件，无需拉取整个硬盘
- **成功阻止数据泄露**：确认文件从未被发送出去
- 及时终止员工并保护客户数据

00:24:30 - 内部威胁统计
- 83%的公司会经历某种形式的内部威胁
- 大多数欺诈案件要么未被发现，要么数月后才被偶然发现
- 通常由客户投诉触发调查

00:25:45 - Teramind高级功能
- **OCR（光学字符识别）**：搜索屏幕上出现的任何文本
- **关联分析**：输入会员ID，查看所有访问过该账户的员工
- 可追踪信息出现在Excel文件、Teams消息等任何位置
- 帮助识别欺诈团伙中的所有参与者

00:27:00 - 快速扩展部署
- 从20人试点扩展到**1500名员工**（6个月内）
- 未覆盖全部2100人的原因：
  - 不同地区法律限制（如葡萄牙禁止24/7监控）
  - 需要轮换监控，无需为非监控时段购买许可证

00:28:30 - 实施挑战与解决方案

挑战1：员工抵触情绪
- "老大哥监控"担忧
- 微观管理恐惧
- 跨部门访问权限争议（营销主管不希望开发部门查看其员工）

解决方案：
- 设置权限：管理者只能查看直接下属
- 屏蔽HIPAA和PCI敏感信息（医疗记录、支付页面）
- 建立**治理委员会**审批调查请求
- 创建"特殊项目"类别用于深度调查（临时解除屏蔽）
- 全公司透明沟通监控政策（但不透露具体监控范围）

00:32:00 - 积极应用场景

场景1：服务水平优化
- 发现员工使用按键模拟器保持在线状态
- 部分员工恶意逃避工作
- 更多员工是因为呼叫量不足（20-40分钟无来电）
- 同时其他部门等待时间达10-20分钟
- **解决方案**：重新培训低呼叫量员工转岗到高需求部门
  - 员工不再无聊
  - 增加销售机会和佣金
  - 降低欺诈动机（收入增加）
  - 改善服务水平
  - 提升品牌合作伙伴满意度

00:35:15 - 场景2：顶尖销售员分析
- 研究多年销冠的工作方法
- 将Teramind屏幕录像与通话录音同步分析
- **发现关键模式**：
  - 销冠在需要前就提前打开应用程序
  - 使用完全相同的话术和笑话（高度标准化）
  - 低绩效员工临时打开应用，加载时出现尴尬沉默
  - 销售中的沉默会破坏成交
- **成果**：基于观察制定标准操作流程(SOP)，提升整体销售效率

00:38:00 - 场景3：证明员工清白
- 会员投诉：购买会员后信用卡在Target被盗刷
- 会员声称仅向Aria提供过该卡信息
- **调查过程**：
  1. 同步通话录音与Teramind屏幕录像
  2. 确认员工输入支付信息的准确时长
  3. 系统自动编辑信用卡信息，员工无法再次查看
  4. 屏幕显示员工输入后立即退出，无抄写时间
  5. Teramind提供的IP地址显示员工位置距离Target店铺4小时车程
  6. 联系Target确认刷卡时间戳
  7. 盗刷发生在员工轮班前2小时，物理上不可能到达
- **结果**：2小时内完成调查并向会员证明员工清白
- 避免了不必要的员工停职和潜在法律纠纷

00:42:00 - 其他有趣发现
- **欺诈性病假**：员工请病假但在电脑上预订度假酒店
- 可以看到员工为请假日期预订旅行
- 员工试图绕过正规请假审批流程

00:43:00 - 总结与建议
- 鼓励网络安全/IT团队与反欺诈/内审团队密切合作
- 两个团队的协同可以有效预防内部威胁
- Teramind等工具将反欺诈工作从被动响应转变为主动预防
- 在保护隐私与安全需求之间取得平衡至关重要
