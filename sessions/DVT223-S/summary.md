# AWS re:Invent 2025 技术会议总结：使用 Amazon Q Developer 最大化生产力

## 会议概述

本次技术分享会由 Pluralsight 的云架构师主讲，主题为"使用 Amazon Q Developer 最大化生产力"。演讲者介绍了 Amazon Q Developer CLI Agent 现已更名为 Kiro CLI，这是一个基于 Claude 模型的终端聊天代理工具。会议基于 Pluralsight 最新发布的技术预测报告，该报告调研了 1500 名技术高管、IT 人员和业务领导者，发现 AI 治理、伦理和投资回报率将成为下一阶段重点，而云安全、基础云技能、云架构技能以及 AI/机器学习仍是最大的技术技能缺口。

演讲者从实际工作场景出发，展示了如何在日常技术工作中有效使用 Kiro CLI 来提升生产力。作为一名同时承担技术和非技术工作的专业人士，演讲者分享了在文档处理、研究开发、技术实施和故障排除等各个环节中使用 AI 助手的实践经验，强调了在 AI 时代保持技能更新和适应能力的重要性。
