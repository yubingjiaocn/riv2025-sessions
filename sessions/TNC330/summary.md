# AWS Agent Core 技术会议总结

## 会议概述

本次会议由来自巴西圣保罗的高级技术讲师 Marilia Brito 主讲，重点介绍了 AWS Agent Core 服务。Agent Core 是一个专门构建的托管服务，采用微服务架构方法来构建智能代理系统。与传统的简单问答系统不同，Agent Core 提供了一个模块化架构，包含运行时、网关、身份验证、可观测性、内存管理等核心组件。会议详细展示了如何配置和使用这些组件，以及在 AWS re:Invent 2025 期间发布的最新功能特性。

## 详细时间线与关键要点

### 0:00-5:00 会议开场与架构概述
- 介绍智能代理系统的复杂性，强调微服务架构方法
- 展示 Agent Core 的核心组件：目标指令、工具、可观测性
- 说明传统代理系统管理的挑战性

### 5:00-10:00 Agent Core 模块介绍
- **运行时模块**：负责执行环境和隔离会话
- **网关模块**：暴露工具访问接口
- **身份模块**：处理认证和授权
- **可观测性模块**：监控系统性能
- **内存模块**：管理上下文信息

### 10:00-15:00 运行时组件详解
- 支持多种开源框架：LangChain、CrewAI、LlamaIndex 等
- 兼容 Amazon Bedrock、SageMaker AI 和外部模型
- 运行时配置包括：模型框架选择、身份配置、可观测性配置
- 使用 Docker 打包并注册到 ECR

### 15:00-20:00 会话生命周期与身份管理
- 会话状态：活跃 → 空闲（默认15分钟）→ 终止
- 最大会话持续时间可扩展至8小时
- **身份模块**支持三种认证方式：IAM、Amazon Cognito、外部提供商
- 入站和出站认证分离，支持令牌交换机制

### 20:00-25:00 网关组件与工具管理
- 支持暴露 REST API、Lambda 函数和服务器
- 使用语义搜索匹配正确工具
- 安全令牌保险库存储认证信息
- 新功能：策略代理核心（预览版），支持自然语言策略编写

### 25:00-30:00 内存管理系统
- **短期内存**：会话期间使用，包含事件、参与者、会话信息
- **长期内存**：用户偏好、语义摘要、学习用户沟通模式
- 默认存储7天，最长可存储1年
- 避免重复解释，提升用户体验

### 30:00-35:00 浏览器与代码解释器
- **浏览器模块**：使用微虚拟机访问网站界面
- 支持 VPC 安全控制，使用 Playwright 无头浏览器
- **代码解释器**：支持 JavaScript、TypeScript、Python
- 提供系统和自定义两种模式

### 35:00-40:00 可观测性与总结
- 兼容 OpenTelemetry，支持 CloudWatch 集成
- 三层可视化：总体仪表板、代理特定指标、会话级跟踪
- 新功能：代理核心评估（预览版），包含准确性、有用性、忠实性等指标
- 支持自定义评估指标和 LLM 评判机制
