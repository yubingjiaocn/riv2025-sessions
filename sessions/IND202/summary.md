# AWS re:Invent 2025 医疗保健与生命科学实时数据分析会议总结

## 会议概述

本次会议由AWS医疗保健与生命科学行业战略解决方案开发项目负责人Anne Evans主持，重点探讨了实时数据（Real-World Data, RWD）和实时证据（Real-World Evidence, RWE）在医疗保健和生命科学领域的革命性应用。会议邀请了礼来公司（Eli Lilly）实时数据高级总监Greg Cunningham和AWS首席解决方案架构师Eric Brooks，分享了如何通过AWS云平台和AI代理工作流程来加速和简化实时数据访问。

会议强调了当前医疗数据爆炸式增长的现状：全球数据生成量从2015年的16泽字节增长到2025年底的181泽字节，其中医疗数据预计达到10泽字节。然而，80%的医疗数据仍然是非结构化的且难以访问。通过AWS与合作伙伴Datavant的协作，以及多代理AI系统的开发，正在解决数据发现、获取和分析中的关键挑战。

## 详细时间线与关键要点

### 0:00-10:00 开场介绍与行业背景
- **0:00-2:00**: Anne Evans开场，介绍关键统计数据
  - 每个突破性药物上市需要分析超过1000万个患者数据点
  - 80%的医疗数据仍然非结构化且难以访问
  - 利用实时证据可将临床试验时间缩短40%，节省数百万成本
- **2:00-5:00**: 全球数据增长趋势分析
  - 2015-2025年全球数据增长11倍（16泽字节到181泽字节）
  - 2025年底预计医疗数据达到10泽字节
  - 实时数据和实时证据从新兴领域发展为主流组成部分
- **5:00-10:00**: 实时数据定义与应用场景
  - 实时数据：常规患者护理过程中收集的医疗数据
  - 实时证据：通过分析这些数据获得的洞察
  - 应用于预防性护理、临床试验设计、公共政策制定等

### 10:00-20:00 数据基础层与挑战分析
- **10:00-15:00**: 数据基础层的重要性
  - 包括历史数据、实验数据、计算数据和第三方数据
  - 强调最先进的AI只能与其可访问的数据一样智能和有效
- **15:00-20:00**: 当前数据获取面临的挑战
  - 40%的数据购买不完整或无效
  - 70%的研究需要跨多个数据集的特定用途数据
  - 数据获取通常需要4-12周时间
  - 数据协调需要额外4周时间
  - 整体药物开发成本达到20-30亿美元

### 20:00-30:00 AWS解决方案架构
- **20:00-25:00**: AWS Clean Rooms与Datavant合作方案
  - 利用Datavant的隐私保护技术简化数据发现和访问
  - 数据生产者在AWS账户中托管令牌化的个人身份信息
  - 通过AWS Data Exchange进行数据订阅和计费
- **25:00-30:00**: 令牌化技术详解
  - Datavant专有软件将PII转换为唯一字符串令牌
  - 确保患者级别数据连接而不泄露个人身份信息
  - 令牌化在数据生产者防火墙内完成，确保数据安全

### 30:00-40:00 Greg Cunningham礼来公司经验分享
- **30:00-35:00**: 礼来公司背景与实时证据定义
  - Greg在礼来工作39年，专注实时数据12年
  - FDA对实时证据的定义：从电子医疗来源获得的临床证据
  - 礼来的RWE公式：科学问题+数据+统计分析计划
- **35:00-40:00**: 实时证据在产品生命周期中的应用
  - 早期发现阶段的分子研究
  - 临床试验协议改进
  - 上市后的观察性研究和经济价值评估
  - 患者特征分析和药物性能监测

### 40:00-50:00 数据类型与获取流程
- **40:00-45:00**: 实时数据类型分类
  - 基础数据：电子健康记录、保险理赔数据
  - 新兴数据：可穿戴设备、传感器、健康应用
  - 生物标记物和基因组数据在肿瘤学中的应用
- **45:00-50:00**: 数据获取挑战与解决方案
  - 传统流程从需求评估到数据获取可能需要一年时间
  - AWS-Datavant解决方案有望将时间缩短至一个月以内
  - 数据传输和摄取过程的优化

### 50:00-60:00 Eric Brooks多代理AI系统架构
- **50:00-55:00**: 代理系统设计理念
  - 单代理vs多代理系统的权衡：复杂性换取能力
  - 七个专业化代理：助手、SQL创建器、医疗编码器、可视化创建器、研究规划器、出版物顾问、数据导航器

  - 个别代理验证的重要性
  - 深入研究防护栏和系统级验证
