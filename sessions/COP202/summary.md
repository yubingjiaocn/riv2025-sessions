# AWS re:Invent 2025 - COP 202: AWS 成本优化新功能

## 会议概述

本次会议由 AWS 成本优化产品管理团队负责人 Rick Oaks 主持，重点介绍了 AWS 成本优化工具的最新功能和发展。会议邀请了达美航空中央 FinOps 团队负责人 Bonnie Fernstall 和储蓄计划购买体验负责人 Mary Naimov 作为嘉宾演讲者。

会议主要围绕三个核心主题展开：首先是 Compute Optimizer 中新推出的 EBS 优化自动化功能，包括闲置卷清理和卷类型升级；其次是新的成本效率指标，为 FinOps 团队提供了统一的优化成功衡量标准；最后是全新的数据库储蓄计划，为多种数据库服务提供高达 35% 的折扣。达美航空的实际案例展示了如何通过系统化的方法实现 27% 的月度成本节省。

## 详细时间线与关键要点

### 0:00-5:00 - 开场介绍与背景
- Rick Oaks 介绍团队和产品范围，包括 Compute Optimizer、成本优化中心、储蓄计划分析器等
- 现场调研显示参会者中 FinOps 组织和工程团队人员各占约一半
- 大部分参会者已使用过 AWS 优化工具，为新功能演示奠定基础

### 5:00-15:00 - EBS 优化自动化功能详解
- 新推出的 EBS 优化功能解决客户反馈的"推荐太多难以执行"问题
- 重点关注低门槛优化：闲置 EBS 卷清理（自动快照后删除未挂载卷）
- 自动卷升级：GP2 升级到 GP3 节省 20% 成本，IO1 升级到 IO2 提升弹性
- 支持即时执行和策略化定期执行两种模式

### 15:00-25:00 - 自动化规则配置与安全机制
- 自动化规则可在组织、账户子集或单个账户级别创建
- 支持基于标签、区域、容量等条件的精细化规则配置
- 可设置维护窗口调度，尊重组织变更控制流程
- 内置多重安全网：一年内可回滚、执行前重新验证、失败时自动停止

### 25:00-30:00 - 控制台演示
- 展示新的自动化界面，推荐操作可多选并一键执行
- 演示自动化规则创建流程，包括组织级别和账户级别选择
- 展示标签条件配置和多值支持，解决标签大小写不一致问题
- 介绍调度设置和资源标记功能

### 30:00-35:00 - NAT 网关闲置检测
- 新增 NAT 网关闲置检测推荐，基于 30 天历史使用情况
- 不仅检查零使用率，还验证路由表关联状态，识别真正的备用网关
- 帮助客户识别遗留的未使用 NAT 网关，与网络团队协作清理

### 35:00-45:00 - 成本效率指标
- 推出全新的成本效率指标，平衡资源优化和承诺优化价值
- 解决现有指标（如有效储蓄率、平均 CPU 利用率）只显示部分情况的问题
- 计算方法：(1 - 总潜在节省/总可优化支出) × 100%
- 提供 90 天历史数据，支持按账户、区域查看，便于设定目标和基准测试

### 45:00-50:00 - 达美航空案例分享开始
- Bonnie Fernstall 分享达美航空在 1200 个账户生态系统中实施优化的经验
- 从产品经理角度分析优化推荐的实施挑战：在哪里、谁来做、做什么、何时做、为什么
- 创建机会仪表板，集中化推荐并丰富组织上下文信息
- 使用 10 字符代码作为主键的轻量级标签策略

### 50:00-54:20 - 达美航空实施策略与成果
- 从报告机制发展为包含团队决策的系统，要求提供实施时间线和延迟原因
- 通过自动化解决团队意愿与实际执行之间的时间差问题
- 从 EBS 卷清理开始，逐步扩展到卷转换和 EC2 右调大小
- 建立成本冠军计划，通过双周会议进行教育和知识分享
- 最终实现 27% 的月度成本节省和文化转型，团队开始在设计阶段就考虑优化
