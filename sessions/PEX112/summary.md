# AWS re:Invent 2025 代理AI解决方案货币化会议总结

## 会议概述

本次会议由AWS的Seenu Brahmarouthy和Leno Piperi主讲，专门为合作伙伴设计，旨在指导合作伙伴如何在代理AI领域实现货币化并推动收入增长。会议深入探讨了从生成式AI助手到完全自主的代理AI系统的演进历程，重点介绍了AWS对AI代理的定义：能够代表人类进行规划、推理和行动的自主系统。

演讲者强调，未来将有数十亿个代理存在，这为合作伙伴提供了巨大的商业机会。会议通过Druva公司的实际案例展示了代理AI的实际应用效果，该公司通过基于Amazon Bedrock AgentCore构建的DruAI解决方案，能够自主解决63%的客户安全问题。会议还详细介绍了AWS Marketplace中的AI代理和工具市场，为合作伙伴提供了完整的上市策略和技术实现路径。

## 详细时间线与关键要点

### 0:00-5:00 代理AI发展历程与定义
- 介绍了AI发展的不同阶段：从Gen AI助手（Claude、ChatGPT）到Gen AI代理，再到完全自主的代理AI系统
- AWS对AI代理的官方定义：能够代表人类进行规划、推理和行动的自主系统
- 预测未来将有数十亿个代理，为合作伙伴创造巨大商机
- 概述AWS提供的完整服务栈：应用层（Kiro、Quick Suite、AWS Marketplace）、开发工具（Bedrock、Bedrock AgentCore）、基础设施层（SageMaker、专用芯片、GPU）

### 5:00-8:00 合作伙伴成功案例与产品策略
- Druva案例研究：云数据安全领域的ISV合作伙伴，面临安全工作流程复杂性挑战
- DruAI解决方案：基于Amazon Bedrock AgentCore构建的多代理系统，包含数据代理、洞察代理、行动代理
- 显著成果：能够自主解决63%的客户安全问题
- 介绍AWS白皮书《在代理时代重新思考SaaS》中的四种产品和市场策略：
  - 扩展策略：现有客户基础上的新功能，可采用追加销售策略
  - 构建策略：面向新客户的全新产品，风险较高但收入机会大
  - 嵌入策略：为现有客户嵌入AI功能，作为客户保留策略
  - 拓展策略：为现有客户提供补充解决方案，创造额外收入流

### 8:00-12:00 AWS Marketplace AI代理和工具市场
- 2025年7月推出的AI代理和工具市场，作为客户寻找代理解决方案的一站式商店
- 三大产品类别：
  - AI代理：预构建代理、嵌入式代理、代理开发平台
  - 工具：MCP服务器等代理工作流程构建工具
  - 咨询服务：AI策略咨询和实施服务
- Leno Piperi接手介绍AWS Marketplace的优势：
  - 精选数字目录，简化客户发现、订阅和部署流程
  - 交易加速和采购效率提升
  - 与AWS战略目标对齐，共同成长
  - 利用AWS全球联合销售网络

### 12:00-15:00 定价模式与部署方法
- 灵活的定价模式：
  - 基于使用量的定价：适合小客户和开发者试用，按消费付费
  - 合同定价：企业客户偏好，支持预付款和长期合同
  - 私有报价支持：允许买卖双方协商有利条款
- 多样化部署方法：
  - 基于API的代理：在合作伙伴AWS账户中托管，通过端点提供访问
  - 基于容器的代理：定义为Docker镜像，客户可在选择的容器编排服务中部署
  - 专业服务和软件产品：提供设计实施服务和嵌入代理解决方案的SaaS产品

### 15:00-17:36 与Bedrock AgentCore集成与实际演示
- AWS Marketplace与Amazon Bedrock AgentCore的紧密集成
- AgentCore平台特性：支持多种框架、所有Bedrock基础模型、A2A和MCP等主流协议
- 客户体验演示：
  - 一键订阅流程
  - 查看使用成本、条款和配置选项
  - 选择Bedrock AgentCore作为部署方法
  - 一键部署到AgentCore运行时环境
  - 获得可直接集成的端点
