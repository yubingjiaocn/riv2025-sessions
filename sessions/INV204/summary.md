# AWS re:Invent 2025 - 构建可信赖的AI智能体系统

## 会议概览

本次AWS re:Invent 2025分会场由AWS高级首席技术产品经理Erin Kramer主持,聚焦于如何构建可信赖的AI智能体(Agentic AI)系统。会议强调了信任是AI智能体从原型走向生产环境的核心要素。Kramer回顾了AWS 20年的发展历程,从EC2、Lambda、SageMaker到Amazon Bedrock和Nova模型,每一次技术革新都以客户信任为基础。她指出,尽管Gartner预测到2027年底将有超过40%的AI智能体项目被取消,但通过建立可靠性、透明度、安全性和易用性四大信任支柱,AWS正在帮助客户将AI智能体成功部署到生产环境。

会议邀请了多位行业领袖分享实践经验,包括Sendbird CEO John Kim展示其Delight AI客户服务平台、AWS杰出工程师Mark Brooker演示Amazon Bedrock Agent Core的开发体验,以及Lyft执行副总裁Jason Vogernick分享如何利用AI彻底改造客户支持系统。这些案例充分证明,当AI智能体建立在可靠的基础设施、具备完整的可观测性、遵循严格的治理规范并易于使用时,它们能够为企业创造巨大的商业价值,同时赢得用户的信任。
