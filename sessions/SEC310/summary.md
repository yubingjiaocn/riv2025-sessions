# AWS re:Invent 2025 网络安全创新会议总结

## 会议概述

本次AWS re:Invent 2025分组会议聚焦于网络和应用程序保护服务的最新创新。会议由AWS安全组织网络和应用程序保护服务市场推广负责人Sofía Aluma-Santos和产品管理负责人Saleem Muhammad共同主持。

会议围绕网络安全运营团队面临的核心挑战展开，包括多架构复杂性管理、扩展性能需求、合规性要求、认知负荷减轻以及威胁可见性提升等。演讲者详细介绍了AWS在网络和应用程序保护领域的全新功能和服务增强，特别强调了基于全球基础设施规模和威胁情报的防护能力。

整个会议内容涵盖了从威胁情报和可见性增强，到应用程序保护创新，再到网络保护的全面升级，为与会者提供了完整的网络安全解决方案路线图。

## 详细时间线与关键要点

### 0:00-5:00 开场介绍与挑战概述
- 会议开场，介绍演讲者背景和角色
- 阐述网络安全团队面临的五大核心挑战：
  - 多架构复杂性管理（混合云、多云环境）
  - 扩展性能和可见性需求
  - 合规性和数据主权要求
  - 认知负荷和易用性问题
  - 威胁可见性和情报需求

### 5:00-10:00 网络和应用程序保护服务组合概览
- 介绍AWS网络和应用程序保护服务架构
- 详细说明典型Web应用程序架构中的安全层：
  - WAF和Shield Advanced用于入口保护
  - Network Firewall用于VPC内流量检查和出口控制
  - Route 53 Resolver DNS Firewall用于DNS路径保护
  - Firewall Manager用于集中管理和可见性
- 强调AWS全球基础设施规模：每分钟分析EB级数据，每日缓解数千次DDoS攻击

### 10:00-15:00 威胁情报和MadPot项目
- 介绍AWS威胁研究方法和欺骗技术
- 详细说明MadPot项目：
  - 在全球基础设施部署数万个诱饵系统
  - 模拟具有不同漏洞变体的易受攻击服务
  - 收集威胁行为者的侦察和攻击模式
  - 将威胁情报集成到Shield和GuardDuty等服务中

### 15:00-20:00 网络可见性新功能
- 发布Shield Network Security Director服务（预览版）：
  - 扫描网络资源并提供拓扑视图
  - 基于AWS最佳实践提供发现和建议
  - 集成Amazon Q Developer进行自然语言查询
  - 显示所有网络资源，无论是否存在安全发现
- AWS WAF新增可见性仪表板：
  - 监控WAF策略和规则的工作状态
  - 识别流量来源地理位置异常
  - 降低CloudWatch日志价格，提供更可预测的成本结构

### 20:00-25:00 Network Firewall可见性增强和应用程序保护创新
- Network Firewall新增原生控制台仪表板体验
- 提供流量洞察和优化建议，包括Private Link端点候选识别
- AWS WAF创建体验全面改版：
  - 推出保护包和启动包
  - 基于应用程序类型（电商、银行等）推荐规则配置
  - 发布CloudFront Plus Security捆绑定价
- 新增Layer 7反DDoS保护托管规则
- 增加AI代理身份验证功能，使用HTTP加密签名进行验证

### 25:00-30:00 网络保护服务深入解析
- Saleem Muhammad接手介绍网络保护面临的三大挑战：
  - 网络威胁不断演变和复杂化
  - 团队资源紧张和技能短缺
  - 需要应用程序通信的深度洞察和可见性
- 详细介绍Route 53 Resolver DNS Firewall和AWS Network Firewall功能特性
- 重点说明出口保护部署选项和新的显式代理功能

### 30:00-35:00 Network Firewall新功能和部署模式
- 发布全托管显式出口代理作为Network Firewall新资源类型
- 支持NAT网关部署，提供DNS前后过滤和TLS解密
- 介绍集中化部署模式的简化：
  - 原生TGW附件功能
  - 灵活成本分配策略
  - 自动创建检查VPC和路由表条目

### 35:00-40:00 威胁情报和合作伙伴托管规则组
- 发布Active Threat Defense托管规则组：
  - 基于MadPot威胁情报直接生成
  - 10分钟间隔自动更新威胁规则
  - 支持警报和阻止模式部署
- 介绍合作伙伴托管规则组功能，通过AWS Marketplace提供
- 详细介绍七个合作伙伴的专业威胁情报服务：
  - Check Point：基于威胁云AI的CVE和OWASP防护
  - Fortinet：FortiGuard实验室的IPS规则组
  - Infoblox：预测性DNS威胁情报
  - Lumen：Black Lotus Labs的Defender规则组
  - Rapid7：APT和勒索软件防护
  - ThreatSTOP：ITAR和OFAC合规支持
  - TrendMicro：零日攻击防护专业化

### 40:00-41:30 总结和后续学习资源
- 总结会议涵盖的威胁情报、可见性增强和保护服务创新
- 推荐相关后续学习会议和资源
- 邀请与会者进行问答和反馈交流
