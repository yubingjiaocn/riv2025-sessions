# AWS re:Invent 2025 技术会议总结

## 会议概述

本次会议由 Cockroach Labs 的高级经理 David Joy 主讲，主题为"使用智能内存构建弹性生成式AI和智能体应用"。会议重点探讨了企业在构建可扩展、弹性的生成式AI系统时面临的挑战和解决方案。

David Joy 分享了 Cockroach Labs 与 AWS 合作三年半以来，在与企业客户对话中观察到的趋势和挑战。他强调，虽然生成式AI改变了用户行为，从传统的搜索引擎转向答案引擎，但真正的挑战在于如何将这些技术从概念验证阶段推向生产环境，为数百万用户提供服务。

会议通过实际演示展示了如何使用 CockroachDB 构建包含 RAG（检索增强生成）、智能体和内存功能的银行应用助手，并在演示过程中实时模拟区域故障来验证系统的弹性。
