# AWS re:Invent 2025 合作伙伴增长与市场策略会议总结

## 会议概述

本次会议由AWS专家与合作伙伴团队负责人Julia Chen和AWS Marketplace与合作伙伴服务副总裁Matt Yanian主讲，主题为"快速增长之路：利用合作伙伴计划和市场实现成功"。会议重点探讨了AI代理时代下合作伙伴如何利用AWS的各项计划和市场创新来加速业务增长和差异化服务。

演讲者强调了AI正在重新定义可能性的边界，从MIT研究人员利用机器学习筛选1亿多种化合物发现新药，到实时多语言翻译和个性化教育路径。会议展示了合作伙伴社区如何利用AI的力量转变客户成果并改造自身业务，例如Trellix利用Amazon Bedrock和Amazon Nova将1000个日常安全警报的分析时间从80小时缩短至30秒。

## 详细时间线与关键要点

### 0:00-10:00 开场与AI变革概述
- Julia Chen介绍会议主题和背景
- 强调AI代理能够推理、规划和自主执行
- 分享MIT利用机器学习发现抗生素耐药性药物halacin的案例
- 展示AI在实时翻译、个性化教育等领域的应用

### 10:00-20:00 合作伙伴盈利框架与业务成果
- 介绍专家与合作伙伴团队的职能和愿景
- 分享Trellix网络安全案例：利用Amazon Bedrock分析安全警报效率提升
- 展示合作伙伴盈利框架：从转售到业务成果的价值递增
- 公布关键数据：表现最佳的合作伙伴每1美元AWS客户支出可获得7.13美元收入

### 20:00-30:00 AI解决方案与野生动物保护案例
- 播放Tehanu野生动物保护项目视频
- 展示Adastra如何利用Amazon Bedrock、S3和Lambda为山地大猩猩保护创建AI解决方案
- 数据处理速度提升110倍：从24小时缩短至13分钟，准确率达93%
- 强调迁移和现代化对释放AI价值的重要性

### 30:00-40:00 迁移加速计划(MAP)与现代化
- 介绍MAP计划的双重目的：结构化迁移方法和资金支持
- 分享客户达到首个迁移里程碑速度提升41%的成果

- 介绍AWS Transform：结合LLM、图神经网络和自动推理的代理迁移服务

### 40:00-50:00 托管服务与合作伙伴激励计划
- 预测到2026年MSP将管理65%的云服务提供商支出(4430亿美元)
- 展示与验证AWS MSP合作的伙伴客户增长速度快3倍
- 介绍新的MSP激励计划：客户管理激励和战略服务激励
- 播放Cap Gemini托管服务经验分享，实现15-20%成本节约

### 50:00-62:20 AWS Marketplace创新与未来展望
- Matt Yanian介绍Marketplace的愿景：在客户所在的任何地方提供服务
- 展示AI驱动的搜索功能和代理模式体验
