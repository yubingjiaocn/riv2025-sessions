# AWS re:Invent 2025 合作伙伴项目与市场分会场总结

## 会议概述

本次分会场主题为"快速增长之路:利用合作伙伴项目和市场实现成功"，由AWS合作伙伴核心项目负责人Julia Chen和AWS Marketplace及合作伙伴服务副总裁Matt Yanchish共同主讲。

会议重点探讨了AI代理时代下AWS合作伙伴生态系统的转型机遇。演讲者强调，AI不仅在改变客户业务成果的交付方式，更在重塑合作伙伴的盈利模式。数据显示，82%的AWS合作伙伴已在提供AI解决方案，73%正在使用AI转型自身业务。表现最佳的合作伙伴现在每投入1美元AWS客户支出可获得7.13美元收入，相比2022年的6.40美元有显著提升，这主要归功于AI技术的应用。

会议展示了多个成功案例，包括网络安全公司Trellix利用Amazon Bedrock和Amazon Nova将1000条日常安全警报的分析时间从80小时缩短至30秒；野生动物保护组织Tahanu与Adastra合作，使用AI将行为数据处理速度提升110倍，从24小时缩短至13分钟。这些案例充分证明了迁移、现代化与AI结合能够释放巨大商业价值。Julia Chen和Matt Yanchish详细介绍了AWS在合作伙伴项目、激励机制、市场功能等方面的最新创新，旨在帮助合作伙伴加速增长并在AI时代脱颖而出。
