# AWS re:Invent 2025 Gundam元宇宙项目技术总结

## 会议概述

本次技术分享会由AWS与万代南梦宫娱乐公司联合呈现，详细介绍了如何构建Gundam（高达）元宇宙空间的完整技术方案。项目团队在高峰期达到80名工程师的规模，由万代南梦宫负责战略规划、创意指导和质量控制，AWS专业服务团队负责基础设施、游戏服务器和3D空间开发。

该项目的核心目标是为全球高达粉丝创建一个可访问的虚拟聚集空间，实现了三大关键理念：让粉丝共同体验高达标志性世界、确保任何人可以随时随地在任何设备上访问、以及提供观看-参与-购物的一站式无缝体验。项目成功覆盖了20多个国家和地区，支持PC、iOS和Android三大平台，通过Amazon GameLift Streams实现云游戏流式传输，并集成Amazon API提供沉浸式购物体验。整个解决方案展示了如何利用AWS专业服务、云游戏技术和电商API集成来构建全球规模的元宇宙应用。
