# AWS re:Invent 2025 - Amazon Q Developer 在 SAP 工作负载中的应用

## 会议概述

本次技术分享会由 AWS 的两位 SAP 解决方案架构师 Shirley Coin 和 Harshadan Padwal 主讲,重点介绍了 Amazon Q Developer 如何帮助 SAP 客户在整个开发生命周期中提升效率。会议展示了 Q Developer 不仅仅是一个编程助手,而是一个能够覆盖从需求收集、文档编写、代码开发到应用迁移的全方位 AI 工具。

演讲者特别强调了针对 SAP ABAP 开发的创新解决方案——ABAP Accelerator 扩展。这个基于 MCP 服务器的扩展解决了 ABAP 开发的独特挑战,包括代码编译、语法检查和 ATC 检查等。通过利用 ADT API,ABAP Accelerator 能够直接与 SAP 系统交互,实现代码的自动推送和验证。会议还展示了如何使用 Q Developer 帮助企业从 ECC 迁移到 S/4HANA,包括将传统报表转换为基于 RAP(Restful ABAP Programming)的现代化应用。

整个演示过程中,讲师们通过多个实际案例展示了 Q Developer 在文档生成、Fiori 应用开发、代码分析和现代化改造等方面的强大能力,为 SAP 客户提供了切实可行的 AI 辅助开发方案。
