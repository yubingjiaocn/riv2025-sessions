# AWS re:Invent 2025 大中华区AI创新实践分享会总结

## 会议概述

本次AWS re:Invent 2025分会由AWS大中华区专业解决方案架构师Xavier Wong主持,邀请了三位来自领先科技公司的嘉宾分享他们在AI领域的创新实践。会议聚焦于大中华区市场观察到的AI应用趋势,特别强调了从生产力提升转向收入增长的战略转变。

Xavier首先指出了当前AI应用的一个关键挑战:根据Gartner数据,仅有41%的公司将AI项目投入生产。这一现象的根本原因在于CEO关注收入增长,而CIO和技术领导者却主要将AI投资集中在生产力改进上。成功的AI应用案例显示,企业需要同时关注这两个方面,但应更多地聚焦于收入增长导向的用例。会议分享了三大核心洞察:AI驱动的业务转型、软件开发的技术趋势,以及企业级AI Agent的应用。

演讲者们展示了具体案例,包括智能硬件制造商如何通过AI功能实现从一次性销售到订阅制的商业模式转变,以及如何利用AWS服务构建高性能、安全可靠的AI平台。会议强调了"Agentic"方法的重要性——不仅是完成任务,更是通过自我反思和迭代循环来优化执行过程。

### AI Agent与Agentic方法 (15:30 - 20:45)

15:30 - 18:00 - Agent vs Agentic的区别
- Agent指AI助手完成任务的能力
- Agentic强调实现方式:自我反思、迭代循环、自动纠错
- 以Amazon Q为例:不仅完成代码,还能运行、发现问题并自动修正

18:00 - 20:45 - 企业级AI Agent应用
- 模型推理能力提升使Agent能够进行任务规划和自我反思
- AWS推出Agent Core服务,提供安全、可观测的企业级Agent环境
- Manus和GenSpark等初创公司实现100%自主的报告生成和PPT创建
