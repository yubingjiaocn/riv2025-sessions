# AWS re:Invent 2025 NBA 云迁移项目总结

## 会议概述

本次会议由 AWS 高级解决方案架构师 Mike Lai 与 NBA 的 Jesse Robbins 和 Mark Bennett 共同主讲，详细介绍了 NBA 如何在短短几个月内完成全面的云迁移项目。2025年10月1日，NBA 与 AWS 宣布了一项为期5年的战略合作伙伴关系，旨在通过云技术推动体育行业的创新。作为一个全球性的体育联盟，NBA 拥有遍布全球的粉丝群体，需要全年无休地提供内容和服务。

这次迁移项目的核心挑战在于如何在保持业务连续性的前提下，快速完成从传统基础设施到 AWS 云平台的转型。团队采用了"慢即是快"的策略，在前期投入大量时间进行架构设计和决策对齐，通过约15次架构审查会议确保所有关键决策达成一致。项目涵盖了超过15个不同的技术团队，包括中间件组件、媒体服务和后端系统三大支柱。最终，团队成功在7月的夏季联赛前完成了生产环境的准备，并在新赛季开始后实现了零重大生产事故的优异成绩。

项目的成功关键在于充分利用 AWS 原生服务（如 Control Tower、CloudWAN、Memory DB、EKS 等），建立可重复的运维框架，以及 NBA 与 AWS 团队之间的紧密协作。通过采用 AWS Well-Architected Framework 进行全面的架构审查，团队不仅确保了迁移的成功，还为未来的创新奠定了坚实的基础。
