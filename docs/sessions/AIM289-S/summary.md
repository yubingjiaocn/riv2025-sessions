# AWS re:Invent 2025 会议总结：从 SIM 到 SOCK - 使用 AWS 构建 AI 原生云安全

## 会议概述

本次会议由 CrowdStrike 和 AWS 联合举办，重点介绍了 Falcon NextGen SIM（下一代安全信息管理）平台如何专门为 AWS 客户提供 AI 驱动的云安全解决方案。会议由三位演讲者主持：AWS 全球首席安全架构师 Jen Reed、CrowdStrike NextGen SIM 团队产品经理 Manoj 以及 CrowdStrike AWS 合作伙伴关系解决方案架构师 Rob Solomon。

会议首先强调了当前威胁环境的严峻性：威胁行为者的突破时间已从 56 分钟缩短至 48 分钟以内，最快记录仅为 51 秒；81% 的攻击不使用恶意软件，而是通过钓鱼和合法工具进行；云攻击增长了 136%。面对这些挑战，传统的安全工具已经无法满足需求，因为它们通常是碎片化的、缺乏上下文信息、响应速度慢且产生大量误报。

CrowdStrike Falcon NextGen SIM 通过三个核心原则解决这些问题：统一平台（将检测、调查和响应整合在单一工作空间）、关注正确的数据而非数据量（提供端点、云、身份等关键数据源的开箱即用覆盖）、以及 AI 驱动的威胁调查工作流（利用 Charlotte AI 进行自动分类、风险评分和响应建议）。该解决方案已帮助客户整合超过 15 个工具，减少 95% 的误报，并每天处理超过 20 PB 的数据。会议还重点展示了专为 AWS 客户推出的集成方案，该方案于 2025 年 12 月 1 日在 re:Invent 大会上正式发布，能够在几分钟内自动配置 AWS CloudTrail、GuardDuty 和 Security Hub 的集成，实现近实时的威胁检测和自动化响应。
