# AWS re:Invent 2025 会议总结：使用生成式AI构建软件

## 会议概述

本次会议主题为"使用生成式AI以前所未有的方式构建软件"，由AWS生成式AI开发体验服务团队的Sutlanak Miskaya和Aldi Sfano主讲，并邀请了Rackspace技术公司的首席技术官Brian Lickley分享客户实践经验。

会议重点介绍了AWS在2025年11月17日正式发布的Agentic开发环境Kira，这是一个专为生产级应用设计的结构化AI编码工具。与传统的AI辅助编码工具不同，Kira通过引入成熟的工程实践，采用规范驱动开发（Spec-driven Development）方法，帮助开发者和团队更高效地完成工作。会议还深入探讨了如何将自定义AI代理集成到业务软件应用中，以及这些技术如何在整个软件开发生命周期中加速开发进程。

Rackspace的实践案例尤为引人注目：在短短四个月内，他们在800名开发人员中推广Kira，实现了85%的效率提升，完成了51个以上的项目，节省了超过8个全职员工年的工作时间。这一成功案例充分展示了Agentic AI在企业级应用中的巨大潜力和实际价值。

### 具体用例 (31:30 - 35:30)
- **31:30** - 关键应用现代化：内部关键应用的升级转型

- **36:00** - 快速修复流程：编写缺失的单元测试、验证修复、部署

- **42:30** - 第二阶段 - 添加助燃剂：从火花到添加树枝
