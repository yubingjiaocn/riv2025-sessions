# AWS re:Invent 2025 - AWS Marketplace 采购优化会议总结

## 会议概述

本次会议由 AWS Marketplace 首席专家 Kaushik Raha 和 AWS Marketplace 及合作伙伴服务工具产品开发负责人 Luca Chertini 共同主讲，主题聚焦如何通过 AWS Marketplace 优化软件和 AI 采购流程。会议详细介绍了 AWS Marketplace 作为数字化采购平台的核心能力，包括产品发现、采购、部署的完整生命周期管理。目前平台拥有来自全球 6,000 多家供应商的近 30,000 个产品列表，涵盖 SaaS、AI 代理、MCP 服务器、AMI 等多种部署模式。

会议重点展示了 AWS Marketplace 在采购治理和报告洞察方面的创新功能，特别是单一管理面板（Single Pane of Glass Dashboard）的推出，为采购团队提供了统一的可视化界面来管理所有 Marketplace 活动。此外，会议深入探讨了 AI 革命对企业的影响，强调了 AI 代理和工具在 AWS Marketplace 中的快速增长。数据显示，60% 的组织已任命首席 AI 官，近 50% 的 2025 年预算投入到生成式 AI 项目，预计未来 5 年内 50% 的应用将具备某种形式的代理体验，这是一个 10 倍的增长。
