# AWS re:Invent 2025 会议总结：最大化您的学习体验

## 会议概述

本次会议是一场专门为参加AWS re:Invent 2025的日本参会者准备的指导性讲座，由AWS日本的两位解决方案架构师五十嵐史都（Igarashi Shito）和小林（Kobayashi）主讲。会议的核心目标是帮助参会者充分利用re:Invent这一全球最大的AWS年度盛会，最大化学习效果。

演讲者强调，AWS re:Invent不仅仅是一个营销活动，而是一个专注于提供大量学习机会的技术盛会。AWS全球员工倾注全力为这次活动准备了丰富的内容，包括主题演讲、技术会议、实践工作坊、认证培训等多种形式。会议特别提到，约80%的现场观众是首次参加re:Invent，因此本次指导会议的时机非常合适——在活动第一天早上为大家提供全面的参会建议。

演讲者详细介绍了如何根据不同的兴趣方向（技术导向或商业导向）选择合适的会议内容，如何利用会场交通工具，以及如何通过社交活动建立人脉网络。他们还特别强调了安全注意事项和实用技巧，比如在哪里购买价格合理的水和日用品，如何避免排队等候，以及如何充分利用翻译应用程序克服语言障碍。

## 详细时间轴与要点

### **开场介绍 (0:00-3:30)**
- 五十嵐史都和小林两位解决方案架构师介绍自己
- 说明会议目的：分享如何在re:Invent最大化学习的技巧
- 现场调查：约80%参会者是首次参加re:Invent，少数人参加过三次以上

### **会议议程说明 (3:30-5:00)**
- 会议流程：总体概述 → 100%享受活动的方法 → 针对技术和商业兴趣者的内容推荐 → 不容错过的推荐内容 → 实用技巧
- 会议设有问答环节，鼓励参会者分享计划和提问
- 提醒：会议正在录制，将上传至YouTube

### **re:Invent活动定位 (5:00-8:00)**
- re:Invent是AWS年度最大、最全面的活动
- 核心价值：提供大量学习机会（技术和商业层面）
- 强调学习云技术和AI技术在工作和生活中的应用

### **主题演讲（Keynote）介绍 (8:00-13:00)**
- **12月3日 8:00-10:30**：CEO Matt Garman主题演讲，讨论云技术未来，包含嘉宾演讲
- **12月4日 8:30-10:30**：VP Swami关于Agentic AI的演讲
- **12月4日下午**：合作伙伴负责人Rubabono关于合作伙伴关系的演讲
- **12月5日 9:30-10:30**：Peter DeSantis和Dave Brown关于基础设施创新的90分钟演讲
- **12月5日 15:30-16:30**：Warner Bogus面向开发者的闭幕主题演讲

### **创新演讲（Innovation Talk）(13:00-15:00)**
- 各领域负责人讨论AWS未来发展方向
- 涵盖公共部门、数据库服务等多个领域
- 建议先观看Innovation Talk了解整体思路，再深入参加技术会议

### **会议类型介绍 (15:00-20:00)**
- **Breakout Sessions**：标准演讲会议
- **Builders Sessions / Chalk Talks / Code Talks**：小组深入讨论，可观察他人行为
- **Interactive Training**：动手实践，有讲师指导
- **Lightning Talks**：快速演讲
- **Self-paced Training**：自主学习，可能包含日本未发布的内容
- **Workshops**：日本团队可能花费6个月准备的深度工作坊
- **AWS Certification**：认证考试准备和实际考试
- **Boot Camp**：长时间集中学习

### **技术难度等级说明 (20:00-24:00)**
- **Level 100**：入门级，无需先验知识
- **Level 200**：功能深入，更具体的讨论
- **Level 300**：需要一定先验知识
- **Level 400**：面向成熟用户
- **Level 500**（新增）：需要深厚技术知识，讨论前沿技术和理论
- Level 500示例：Amazon Nova风险评估、自动推理减少AI幻觉、Rust编程语言安全性

### **会场地理和交通 (24:00-32:00)**
- 会场分布：Mandalay Bay（最南端）到Venetian（东北部），直线距离约5公里
- 类比：相当于东京站周围5公里范围
- **交通建议**：
  - 使用班车（Shuttle Bus），通过re:Invent应用查看等待时间
  - 单轨电车（Monorail）：从MGM Grand出发向北，凭re:Invent徽章免费乘坐（通常$5/次）
  - 出租车和共享出行服务：注意上车点不同，共享出行通常在车库
  - 检查地图应用避开交通拥堵

### **购物和日常用品 (32:00-34:00)**
- 酒店商店价格极高
- **推荐商店**：
  - Walgreens（药店）
  - CVS Pharmacy（药店）
  - ABC Stores（便利店，夏威夷常见）
- 价格不到酒店的一半，可购买水、零食、日用品、小药品

### **实用技巧 (34:00-38:00)**
- 即使会议显示已满，到现场仍有机会walk-in
- 热门会议可能会安排重复场次，通过应用注册
- 主题演讲中宣布的新服务相关会议通常会后续添加
- 许多会议录像会上传YouTube，优先考虑现场独有体验
- **安全提醒**：小心街头搭讪，不要接受陌生人物品，看管好随身物品
- 注意休息，不要过度劳累

### **单轨电车特别用途 (38:00-40:00)**
