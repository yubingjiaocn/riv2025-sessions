# AWS re:Invent 2025 会议总结：从地面到云端 - CAE 的数字化转型之旅

## 会议概述

本次会议由 Red Hat 的 OpenShift 销售专家 Dan Dobrescu 和 AWS 高级容器专家 Math Bruneau 主持，特邀嘉宾为 CAE 公司首席信息官 Jean-Francois Gamache。会议主题"从地面到云端"巧妙地呼应了航空业的特点，展示了 CAE 如何通过 AWS 和 Red Hat OpenShift 实现数字化转型。

CAE 作为全球最大的民用航空培训服务提供商，每年培训超过 15 万名飞行员，在全球 85 个站点运营。该公司凭借其在 OpenShift 和 Red Hat 产品方面的创新应用获得了 Red Hat 创新奖。会议重点讨论了 CAE 如何利用现代化平台运行关键任务应用程序，包括在飞行模拟器中部署 OpenShift，以及如何通过 AI 和数据驱动的方法提升运营效率。

## 详细时间线与关键要点

### 0:00-5:00 开场介绍与合作伙伴关系
- Dan Dobrescu 介绍 Red Hat OpenShift 平台的核心价值
- OpenShift 可在物理、虚拟、云端和边缘环境中运行
- 基于 Red Hat Enterprise Linux 和增强版 Kubernetes 构建
- Math Bruneau 介绍 AWS 与 Red Hat 的合作历史，可追溯到 2008 年

### 5:00-10:00 ROSA 服务详解
- Red Hat OpenShift Service on AWS (ROSA) 的核心优势
- 完全托管的集群服务，几分钟内即可部署
- 提供 99.95% 的 SLA 保证，24/7 监控支持
- 支持与 AWS 服务深度集成，如 S3、RDS 等
- 基于消费的定价模式，灵活扩缩容

### 10:00-15:00 Red Hat 产品生态与 CAE 获奖背景
- AWS Marketplace 中的 Red Hat 产品组合
- CAE 获得 Red Hat 创新奖的原因：运行关键任务应用，支持每 5 秒起飞一架飞机
- OpenShift 在飞行模拟器中的创新应用
- 统一平台支持虚拟机、容器和 AI 工作负载

### 15:00-20:00 CAE 公司背景与挑战
- CAE 作为全球最大民用航空培训服务提供商的地位
- 每年培训超过 15 万名飞行员，覆盖全球 85 个站点
- 拥有 3000 名开发人员的技术团队
- 控制从研发到供应链、制造、培训中心的完整价值链

### 20:00-25:00 行业增长与技术挑战
- 到 2024 年飞行员培训需求将增长 2.4 倍
- 客户期望不断提高，需要预测、定制和个性化服务
- 合规要求日益严格，同时面临降本增效压力
- 必须持续自我颠覆以应对快速变化的技术环境

### 25:00-30:00 技术现代化战略
- 四大核心能力建设：技术能力、运营模式、文化人才、数据 AI 能力
- 从项目运营模式转向产品运营模式
- 通过 AWS 云服务消除大量技术债务
- OpenShift 作为统一平台，简化开发者与平台工程的交互

### 30:00-36:00 AI 驱动的创新成果
- 平台工程产品标准化全球开发团队工作方式
- AI 辅助生成近 40% 的代码，提升开发效率
- 基础设施成本降低高达 40%
- 部署时间和频率显著改善，拉取请求效率大幅提升
- 通过持续学习循环加速创新，快速获取客户反馈并迭代产品
