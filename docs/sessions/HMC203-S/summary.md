# AWS re:Invent 2025 会议总结：CAE 的云端创新之旅

## 会议概览

本次 AWS re:Invent 2025 分组会议由 Red Hat 的 Open Shift 销售专家 Dan Brescu、AWS 容器高级专家 Matt Bruno，以及 CAE 公司的首席信息官 Jean-François Gamash 共同主讲。会议主题为"从地面到云端"（From Ground to Cloud），象征着航空业通过云技术实现起飞、加速和创新的过程。

CAE 是全球最大的民用航空培训服务提供商，每年培训超过 15 万名飞行员，在全球 85 个培训中心运营。该公司凭借在 Red Hat OpenShift 平台上的创新应用获得了 Red Hat 创新奖。会议重点介绍了 CAE 如何利用 Red Hat OpenShift 和 AWS 云服务构建统一的技术平台，实现从本地数据中心到云端、再到边缘设备（如飞行模拟器）的一致化部署。通过采用平台工程、AI 驱动的开发工具和产品运营模式，CAE 成功提升了开发效率、降低了基础设施成本，并加速了创新周期。

该公司面临着航空业快速增长的挑战——到 2024 年需要培训的飞行员数量将增长 2.4 倍。为应对这一挑战，CAE 通过云原生技术、AI/ML 能力和 OpenShift AI，在关键任务应用（如每 5 秒起飞一架飞机的系统）和飞行模拟器中实现了技术突破。会议展示了具体的投资回报数据：基础设施成本降低 40%，部署效率提升显著，拉取请求处理速度大幅提高。
