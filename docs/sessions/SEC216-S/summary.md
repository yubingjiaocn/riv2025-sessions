# AWS re:Invent 2025 会议总结：金融科技和数字业务中基于网络的无缝欺诈防护

## 会议概述

本次会议由Vonnage全球合作伙伴解决方案高级总监Oscar Rodriguez主讲，重点介绍了如何利用移动网络智能在欺诈到达企业系统之前就将其拦截。随着欺诈攻击变得越来越自动化、成本越来越低廉，金融科技和数字平台面临着在安全性和用户体验之间取得平衡的巨大压力。演讲者指出，欺诈已经从手工操作演变为产品化服务，攻击者只需花费5美元就能订阅语音克隆SaaS服务或购买SIM卡交换服务，技术门槛大幅降低。

Vonnage作为Ericsson旗下的端到端通信提供商，服务超过10万家企业客户，拥有170万注册开发者。其解决方案结合了统一通信、联络中心功能以及全方位的通信API，更重要的是推出了下一代网络驱动的解决方案，包括移动身份验证工具、欺诈防御服务以及位置和网络洞察能力。通过直接接入移动网络，企业可以在不影响用户体验的情况下，实现静默身份验证和品牌化通信，从而创建一个无缝的保护层，提升信任度、增强转化率并降低成本。

## 详细时间线与关键要点

00:00:00 - 会议开场与演讲者介绍
- Oscar Rodriguez介绍自己的职位和演讲主题
- 强调移动网络如何在欺诈到达业务系统之前就将其阻止

00:01:00 - Vonnage公司背景介绍
- 作为Ericsson一部分的端到端通信提供商
- 服务超过10万家企业客户，覆盖100多个国家
- 平台支持超过170万注册开发者，拥有2500多名员工
- 提供统一通信、联络中心、语音/视频/消息API以及移动身份工具

00:03:00 - 欺诈产品化问题分析
- 欺诈已不再是手工操作，而是被打包成易用的服务
- 犯罪分子可以订阅5美元的语音克隆SaaS服务
- 拖放式钓鱼工具包和SIM卡交换即服务降低了攻击门槛
- AI和自动化使攻击规模化、成本极低

00:05:00 - 欺诈现状数据
- 平均数据泄露成本达440万美元
- 近四分之一的美国成年人在去年经历过账户接管或欺诈
- 2024年1月SEC官方Twitter账户被黑客攻击的案例
- 70%的数据泄露始于社会工程攻击

00:07:00 - 电话号码作为薄弱环节
- 电话号码是欺诈攻击的关键弱点
- 联络中心场景：黑客通过SIM卡交换冒充客户
- 如果没有信任信号，客服可能批准账户重置或大额转账
- 强调实时预防的重要性

00:09:00 - 下一代欺诈防护方案
- 直接接入移动网络进行欺诈检测
- 在不影响合法用户体验的情况下阻止欺诈
- 移动网络静默验证设备真实性和用户身份
- 在账户创建、登录和高价值操作时进行后台检查

00:11:00 - 可检测的威胁类型
- 虚假账户、账户接管、退款滥用
- 社会工程攻击、钓鱼和短信钓鱼
- 流量泵送等威胁
- 网络可以看到应用和浏览器无法看到的信号

00:13:00 - 客户旅程中的全程保护
- 账户创建：通过静默身份验证确认用户拥有该电话号码
- 登录和敏感操作：静默验证设备和用户匹配
- 账户恢复：检测SIM卡交换活动
- 日常交互：品牌化通话和消息确保通信合法性

00:16:00 - Vonnage保护套件介绍
- Verify API with Silent Authentication：网络驱动的双因素认证
- Identity Insights API with SIM Swap：提供设备和用户状态的实时信号
- 品牌化通信：发送难以伪造的可信品牌消息和通话
- 三者共同形成综合保护层

00:18:00 - Identity Insights API演示
- 演示API调用和响应payload
- 包含SIM卡交换、用户匹配、当前和原始运营商信息
- 所有美国主要运营商已全面可用
- 未来计划推出位置验证、用户匹配、漫游和可达性洞察

00:20:00 - API功能详解
- Current Carrier：识别线路类型（移动线路）
- Format：提供国家代码、时区和格式化信息
- Original Carrier：与当前运营商对比判断号码是否被携号转网
- SIM Swap：提供交换时间戳，可检查1小时到100天内的交换记录
- Subscriber Match：验证姓名、出生日期、地址等信息

00:23:00 - 其他API功能
- Location Verification：检查设备是否在特定位置半径内
- Roaming：返回用户是否漫游及漫游国家
- Reachability：确定设备是否通过SMS或数据通道连接到网络
- API调用仅需毫秒，但可避免440万美元的数据泄露成本

00:25:00 - Lydia银行案例研究
- Lydia是欧洲增长最快的新银行之一，拥有超过800万用户
- 采用Vonnage Verify和静默身份验证
- 身份验证时间缩短50%
- 完全消除基于SMS的社会工程攻击
- 降低总拥有成本，提高转化率

00:27:00 - 会议总结与联系方式
