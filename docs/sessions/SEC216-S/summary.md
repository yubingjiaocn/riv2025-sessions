# AWS re:Invent 2025 技术会议总结

## 会议概述

本次技术会议由Vonage全球合作伙伴解决方案高级总监Oscar Rodriguez主讲，主题为"金融科技和数字业务中的无缝网络防欺诈保护"。会议重点探讨了如何利用移动网络智能在欺诈行为到达业务系统之前就将其阻止。

Rodriguez详细介绍了Vonage作为爱立信子公司的端到端通信解决方案，服务超过10万企业客户，覆盖100多个国家。会议核心内容围绕网络驱动的身份验证、静默认证和品牌通信如何创建无缝保护层，提升信任度、增强转化率并降低成本。演讲强调了当前欺诈攻击已经产业化和自动化的现状，以及传统防护策略需要相应升级的迫切性。

## 详细时间线与关键要点

### 0:00-2:00 开场介绍与公司背景
- Oscar Rodriguez自我介绍，担任Vonage全球合作伙伴解决方案高级总监
- 介绍会议主题：金融科技和数字业务中的无缝网络防欺诈保护
- Vonage公司概况：服务10万+企业客户，覆盖100+国家，170万注册开发者，2500+员工

### 2:00-5:00 欺诈攻击的产业化趋势
- 欺诈不再是手工操作，已经产品化为易用服务
- 攻击成本大幅下降：5美元语音克隆SaaS、拖拽式钓鱼工具包、SIM卡交换即服务
- AI和自动化降低了技术门槛，任何人都可以发起大规模攻击
- 防护策略必须匹配这种工业化规模和自动化水平

### 5:00-8:00 欺诈现状与影响
- 平均数据泄露成本达440万美元
- 近四分之一美国成年人去年经历过账户接管或欺诈
- 2024年1月SEC官方Twitter账户被黑客攻击案例
- 70%的数据泄露始于社会工程学攻击
- 手机号码成为最薄弱环节

### 8:00-11:00 网络驱动的防欺诈解决方案
- 下一代欺诈防护直接接入移动网络
- 在欺诈到达系统前就进行阻止，不影响合法用户体验
- 移动网络静默验证设备真实性和用户身份
- 实时检测和阻止多种威胁：虚假账户、账户接管、退款滥用、社会工程学攻击等

### 11:00-13:00 全客户旅程保护
- 账户创建阶段：静默认证确认手机号码所有权
- 登录和敏感操作：网络验证设备和用户匹配
- 账户恢复：检测SIM卡交换活动
- 日常交互：品牌通信确保通信合法性
- 在整个流程中提供连续且隐形的保护层

### 13:00-15:00 Vonage保护套件演示
- Verify API配合静默认证：网络驱动的双因素认证
- Identity Insights API：提供SIM卡交换等实时网络信号
- 品牌通信：发送可信的品牌消息和通话
- 现场API演示展示各项功能：载体信息、格式验证、SIM卡交换检测、用户匹配等

### 15:00-17:00 实际案例与总结
- Lydia银行案例：欧洲增长最快的新银行之一，服务800万用户
- 实施效果：认证时间减少50%，完全消除基于SMS的社会工程学攻击
- 降低总拥有成本，同时提升转化率
- 演讲总结和联系方式分享
