# AWS re:Invent 2025 - Amazon Bedrock Agent Core Memory 会议总结

## 会议概述

本次会议由AWS首席生成式AI解决方案架构师Manik Kanuja主持，产品经理Jay和Experian工程高级副总裁Imran共同参与，深入探讨了Amazon Bedrock Agent Core Memory（代理核心记忆）功能。

会议的核心主题是如何让AI代理具备记忆能力，从而提供更智能、更个性化的用户体验。演讲者通过一个实际的演示案例——构建能够记住用户偏好的幻灯片生成代理，展示了记忆功能如何显著改善代理应用的表现。记忆被定义为代理上下文中经常被忽视但至关重要的组成部分，它能够让代理在正确的时间获得正确的上下文信息。

Experian作为客户案例分享了他们如何从短期记忆架构迁移到统一记忆架构的实践经验。通过采用Agent Core Memory，Experian简化了技术实现，缩短了产品上市时间，并实现了企业级规模的AI应用部署。该服务提供了完全托管的解决方案，包括自动化的记忆提取、整合和检索功能，让开发者可以专注于构建应用而非管理基础设施。
