# AWS re:Invent 2025 技术分享会总结：解锁智能体未来

## 会议概述

本次分享会由JetBrains产品负责人Chris Kang主讲，主题为"解锁智能体未来"。会议重点探讨了在软件开发中AI使用率快速增长的背景下，如何在提高生产力的同时维持对AI生成代码的信任度。Chris分享了JetBrains对15万开发者的调研结果，显示90%的开发者正在使用AI，其中65%使用频率很高，但只有30%的人真正信任AI输出的代码质量。

演讲深入分析了AI能力的快速发展趋势，引用Berkeley研究显示AI能力每7个月翻倍的惊人增长速度。Chris通过对比自己在2025年初和年末使用AI的方式变化，展示了从简单的代码补全和聊天助手，发展到协调多个AI智能体完成复杂项目的演进过程。会议还分享了Meta和Amazon等大型企业如何通过建立完善的治理框架、明确意图规范、选择合适工具和严格实验测量来成功应用AI技术的实践案例。
