# AWS re:Invent 2025 供应链管理AI解决方案技术分享

## 会议概述

本次技术分享由富士通瑞士国家CTO兼AWS大使Chris Bingham主讲，重点介绍了富士通如何运用AI技术解决供应链管理中的实际业务问题。演讲者强调，当前供应链管理面临着波动性、中断和转型三大核心挑战，而AI技术特别适合处理这种高复杂性、高维度的场景。

富士通的uvance决策智能解决方案旨在通过数据和AI赋能团队，提供作为工作伙伴的解决方案，结合富士通的技术专长和客户的领域专业知识。该方案的核心理念是在快速节奏下提供高质量决策，创造积极的商业和社会影响。演讲详细介绍了从第一代解决方案到基于AWS Bedrock AgentCore的第二代解决方案的技术演进过程。
