# AWS re:Invent 2025 Web3支付协议与代理支付会议总结

## 会议概述

本次会议是AWS re:Invent 2025第五天的技术分享，主题聚焦于Web3、数字资产支付协议以及代理支付系统。会议由AWS的Arvind和Joshua Smith，以及Coinbase开发平台工程负责人Eric Rappel共同主讲。会议深入探讨了Web3技术在AWS上的应用，重点介绍了X402开放支付协议以及如何在AWS基础设施上构建代理支付系统。演讲者强调了稳定币在AI代理支付中的重要作用，并展示了如何利用AWS的托管服务和合作伙伴生态系统来简化Web3应用的开发和部署。

会议采用300级别的技术深度，不仅提供了概念性介绍，还包含了具体的代码实现和架构设计。演讲者特别关注了传统支付系统的局限性，如高处理费用、长结算延迟和复杂的跨境交易，并展示了数字资产支付如何解决这些问题。整个会议体现了AWS对Web3技术的全面支持态度，从基础设施服务到合作伙伴生态系统的完整解决方案。
