# AWS re:Invent 2025 分组会议总结

## 会议概述

本次会议探讨了技术、金融和未来出行的交叉领域，重点介绍了Lucid Motors（一家软件定义的电动汽车制造商）如何与PWC合作，利用AWS服务在财务组织中建立AI代理（Agentic AI）概念验证。Lucid Motors不仅在制造电动汽车，更在重新定义数字优先的汽车制造商应该是什么样子。

会议的核心主题是Lucid如何在财务组织中快速采用AI技术，以"Sapphire速度"（0-60英里加速不到2秒）推动组织变革。Lucid与PWC合作，针对从董事会、高管到分析师等各个层级建立解决方案，目标是将财务部门转变为具有竞争优势的增长催化剂。

演讲嘉宾包括Lucid Motors的业务财务负责人Aditya Baheti、财务系统开发负责人Will，以及PWC的科技媒体电信咨询业务负责人Lori Driscoll和财务解决方案实践负责人Brad Donaldson。他们分享了在财务转型旅程中的经验、挑战和早期成果，强调了高管支持、文化创新和人机协作的重要性。

## 详细时间线与关键要点

开场介绍
- 会议介绍了Lucid Motors作为技术驱动的电动汽车公司的背景
- 强调Lucid与PWC利用AWS服务在财务组织中建立AI概念验证
- 提到Sapphire车型的加速性能（0-60英里不到2秒）作为组织速度的隐喻

Lucid Motors公司介绍
- Lucid是一家软件定义的电动汽车公司，自2021年起销售Lucid Air轿车
- 获得了汽车行业的众多奖项和认可
- 今年推出了Gravity SUV，同样获得了极高评价
- 计划明年推出中型平台车型，价格更实惠，将释放大规模增长潜力
- 与Nvidia合作开发自动驾驶汽车，直接面向消费者
- 与Neuro和Uber合作开发机器人出租车车队，旧金山湾区将成为明年底首个推出城市
- 在美国销售的每辆车都在美国制造

AI转型的驱动因素
- **时机选择**：为即将到来的大规模增长做准备，财务组织需要以可持续的方式扩展以支持增长
- **CFO愿景**：将财务转变为竞争优势，提供更多实时分析和业务洞察，推动数据驱动的决策
- **外部环境**：在快节奏和不确定的环境中（如关税和贸易影响），需要快速适应能力
- **董事会压力**：财富500强公司中，在AI上投资超过0.5%的公司，总股东回报率提高了21%或更多
- **高管支持**：CFO在首次会议结束时表示"有时你必须冒险一试"，这种自上而下的支持至关重要

财务转型的愿景
- 将财务打造成增长的催化剂和竞争优势
- 利用财务可访问的大量数据（不仅限于财务领域），提供实时预测分析
- 提高快速适应和调整的能力，通过多场景规划评估风险状况
- 量化风险水平，主动管理风险以执行战略

文化与创新
- 创新是Lucid的核心主题，体现在汽车的卓越性能和功能上
- 从技术角度不断突破边界，探索未来出行和交通方式
- 财务组织秉承同样的创新理念，以更具成本效益的方式利用可用技术
- 为增长进行投资，但采用更可持续和更具成本效益的方式

用例开发
- 最初并未计划14个用例，而是从痛点出发，识别阻碍财务组织扩展和支持增长的关键领域
- **需求预测**：结合内部和外部因素（CPI、GDP增长、汽油价格、利率、激励措施、折扣、订单转化率等）
- 能够在几分之一秒内运行多个统计模型和场景
- **投资者/分析师仪表板**：不仅显示投资者持股情况，还显示某些投资者对公司股票采取的卖出或看跌期权头寸
- 识别出一些集团对其姐妹公司进行对冲投资，这降低了股价波动性
- 其他用例涵盖财务、会计、内部审计等领域，旨在提高效率、准确性和速度

技术架构选择
- 选择Agentic方法的原因：认识到拥有大量可访问的数据
- PWC帮助发现财务组织拥有比自己熟悉的更多数据
- 除了SAP系统外，Redshift上还有大量有价值的数据
- 利用来自Salesforce和其他平台的数据，为财务构建高价值解决方案
- 部署了Agent OS编排层（PWC预构建，AWS Bedrock原生），提供保障、策略遵从、代理间协作
- 目标不是拥有10个代理，而是随时间推移拥有数百个代理

人机协作的重要性
- 使用旅行规划的类比：ChatGPT可以提供行程，但当地人的建议（人类触感）提供了关键的业务背景
- AI增强而非完全替代财务团队的工作
- 财务团队获得更多速度和带宽，专注于业务洞察，成为真正的思想领袖和战略业务合作伙伴
- 模型仍存在幻觉问题（例如，模型假设9月30日的EV税收抵免取消事件每年都会发生）
- 需要人类提供业务背景来纠正这些假设

变革管理与人才发展
- 技术自动化程度越高，领导者越需要展现人性化特质
- 需要展现同理心，因为AI带来的工作不确定性导致员工焦虑
- 财务组织需要重新培训技能，不仅仅是报告数字和对账，而是学习如何利用工具并更具战略性
- 类比计算器的引入：最初人们担心工作会消失，但实际上只是提高了生产力
- AI将把生产力提高10倍或50倍，但仍需要一定程度的人类参与
- PWC计划在2026年推出一项计划，围绕以人为中心的15个要点进行培训，包括同理心、在不确定性中指导、当技术实时改变工作流程时的辅导

快速执行的关键因素
- 项目仅用了10周左右就完成了概念验证
- 成功因素包括：
  - 高管层的一致支持和自上而下的认可
  - 组织文化：快节奏、创新驱动
  - 合作伙伴支持：PWC和AWS的协作
  - 快速决策：CIO在会议期间就能通过手机批准数据访问权限
  - 技术就绪：云基础设施和数据架构已经到位

行业趋势观察
- 2025年是从试点实验转向有目的性投资代理和工作流程的转折点
- 董事会层面要求CFO制定AI战略，既关注治理（如何控制AI驱动的财务数据），也关注竞争优势
- 需要在大型语言模型和边缘解决方案之间做出架构选择，以应对GPU容量挑战
- 由于能源需求，未来几个月和几年将面临容量挑战
- 文化适应因公司而异：相对较新的科技公司（如Lucid）采用率更高，传统公司需要更多变革管理

早期成果与持续学习
- 团队强调他们不是专家，只是分享经验
- 这个领域以极快的速度发展，今天相关的内容明天可能就不相关了
- 已经看到与PWC和AWS合作的早期成果
- 持续改进模型的准确性是一个旅程，类似于Waymo花了10年时间打造自动驾驶汽车
- 目标是看到AI增强财务工作的世界，而不是完全替代
