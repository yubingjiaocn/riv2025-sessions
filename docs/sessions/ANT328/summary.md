# AWS re:Invent 2025 数据处理架构会议总结

## 会议概述

本次AWS re:Invent 2025分会场讨论了如何构建支持AI解决方案的数据处理架构。会议由AWS首席数据和AI解决方案架构师Saki Mishra和Amazon EMR首席产品经理Radhikarella主讲。

会议开始时进行了一个小调查，询问参会者是否认为其组织的数据基础设施已为AI做好准备，结果显示只有很少的人举手，这反映了当前企业在AI数据准备方面面临的普遍挑战。根据哈佛商业评论的数据，89%的首席执行官正在优先考虑生成式AI计划，但超过一半的受访者认为他们的数据基础设施尚未为AI做好准备。

演讲者强调了数据在AI中的关键作用，并通过一个HR入职代理系统的实际用例来说明AI代理如何需要访问结构化数据、非结构化数据、向量存储和实时流数据。会议重点讨论了三个主要挑战：人员和流程整合、现有数据平台的敏捷性改造，以及传统数据处理效率的提升。
