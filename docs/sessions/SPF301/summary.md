# AWS re:Invent 2025 - Catapult Sports 运动员数据监控解决方案

## 会议概述

本次技术分享会由AWS澳大利亚体育解决方案架构师Mike和高级IoT专家解决方案架构师Greg主讲，深入介绍了Catapult Sports如何利用AWS IoT服务构建下一代运动员监控解决方案Vector 8。Catapult Sports是一家在澳大利亚证券交易所上市的公司，在全球24个地点拥有超过400名员工，服务于128个国家的5000多支精英运动队伍。

会议重点展示了Vector 8系统如何通过可穿戴设备收集运动员数据，包括实时10Hz的比赛数据和100Hz的原始惯性数据，并通过AWS IoT Core、Greengrass等服务实现设备管理、数据处理和机器学习分析。演讲者详细阐述了从设备健康监控、热数据流传输到冷数据批处理的完整数据摄取架构，以及如何将这些数据转化为600多种独特的运动员表现指标。

## 详细时间线与关键要点

### 0:00-5:00 - 开场介绍与Catapult Sports概述
- 几乎所有职业和半职业足球队都使用可穿戴设备追踪数据
- Catapult Sports致力于释放每位运动员和团队的潜力
- 公司业务涵盖运动科学与分析的交叉领域
- 三大解决方案：运动员监控、球探招募、视频套件

### 5:00-10:00 - Vector 8系统组件介绍
- Vector 8标签：紧凑耐用，支持GNSS和本地定位
- 支持室内外无缝工作，电池续航6小时
- Vector 8基座：支持快速Wi-Fi下载和云端数据同步
- Vector 8接收器：通过Wi-Fi和以太网连接，实时流传输数据
- Vector 8中继器：400米覆盖范围，支持大型场馆部署

### 10:00-15:00 - AWS IoT Core与数据摄取路径
- AWS IoT Core提供可扩展的云端点和设备认证
- 三种数据类型：设备健康数据、热数据(10Hz实时)、冷数据(100Hz原始)
- 设备健康监控架构：通过UWB传输到接收器，再通过MQTT发布到IoT Core
- 使用Amazon MSK进行实时流处理和缓冲

### 15:00-20:00 - 热数据实时处理
- 10Hz实时运动员表现数据传输
- 教练可在iOS应用上看到即时可视化
- 包含心率、位置、速度和加速度数据
- 支持实时干预决策和战术调整

### 20:00-25:00 - 冷数据与机器学习
- 100Hz采样的赛后数据处理
- NHL比赛示例：2.5小时比赛，40名运动员，产生3600万个数据点
- 30秒内完成30名运动员2小时训练数据上传到S3
- Catapult可为每位球员每场比赛创建600种独特指标

### 25:00-30:00 - 运动专项指标与Player Load
- Player Load公式：三轴加速度计数据的综合计算
- 澳大利亚体育学院与橄榄球联盟共同开发
- 用于比较球员间差异和追踪伤病恢复进程
- 客户案例：通过数据分析延迟球员复出时间

### 30:00-35:00 - 设备软件架构与AWS IoT Greengrass
- Greengrass作为Vector 8基座和接收器的基础
- 边缘运行时和云服务，支持多进程IoT应用管理
- 组件版本控制和依赖解析
- 接收器架构：数据处理器组件通过IPC发布设备健康消息

### 35:00-40:00 - 设备入网与固件更新
- Vector 8实现10分钟内从开箱到运动员追踪的自助服务体验
- 相比Vector 7的2天手动配置过程大幅简化
- 空中更新(OTA)速度提升32倍
- 97%的设备始终运行最新固件，大幅减少客户支持案例

### 40:00-45:00 - 设备配置与诊断，未来展望
- 使用AWS IoT Shadows进行云端设备配置管理
- 安全隧道功能支持远程SSH访问进行诊断
- Log Manager组件实现实时日志云端访问
- 未来计划：实时数据云端流传输、边缘ML推理增强、团队战术AI分析
