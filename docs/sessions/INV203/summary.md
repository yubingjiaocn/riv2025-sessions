# AWS re:Invent 2025 Amazon Connect 与 QuickSuite 会议总结

## 会议概览

本次会议由AWS Amazon Connect副总裁兼总经理Pasquale DeMayo主持，重点介绍了AI如何变革企业工作场所和客户服务体验。会议指出，尽管当前42%的生成式AI项目面临失败，但成功实施的人机协作AI项目能够带来70%的工作效率提升。AWS提出了"以人为中心的AI"理念，强调AI应该增强人类能力而非仅作为工具存在。

会议发布了两大核心产品：Amazon QuickSuite（统一的AI工作空间）和增强版Amazon Connect（AI驱动的客户服务解决方案）。QuickSuite通过整合企业数据源、提供智能代理和自动化工作流，解决了企业中AI工具碎片化的问题。Amazon Connect则推出了29项新功能，实现了完全代理化的客户服务能力，能够在客户联系的全生命周期中提供智能支持。

会议邀请了BMW、AstraZeneca、3M和Priceline等客户分享实际应用案例，展示了这些AI解决方案在汽车制造、制药研发、销售管理和客户服务等领域的显著成效。AWS强调这些技术已在Amazon内部大规模应用，拥有数十万用户和10万客服代理的实战验证。
