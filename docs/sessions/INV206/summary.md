# AWS re:Invent 2025 云运维创新会议总结

## 会议概述

本次会议由AWS搜索、可观测性和云运维副总裁Nandini Ramani主持,重点介绍了AWS在2025年云运维领域的创新,特别是针对AI代理(Agentic AI)时代的可观测性解决方案。会议强调,2025年是AI代理从实验阶段进入生产环境的关键一年,这些代理不仅能响应请求,还能思考、行动并代表用户做出决策。随着AI应用的普及,88%的组织已在至少一个业务功能中使用AI,超过三分之一的组织正在全面扩展或部署AI应用。

会议通过PGA巡回赛和华纳兄弟探索公司的实际案例,展示了如何利用AWS云运维工具来监控和优化AI应用及大规模流媒体平台。AWS推出了多项新功能,包括CloudWatch生成式AI可观测性、应用程序性能监控(APM)、AI驱动的故障调查工具、以及跨账户跨区域的日志集中化等。这些创新旨在帮助客户在三个核心支柱上取得成功:监控和信任AI代理应用、利用AI实现更智能的运维、以及简化整体云运维操作。
