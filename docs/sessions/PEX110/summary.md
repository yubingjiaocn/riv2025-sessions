# AWS re:Invent 2025 - AWS Marketplace 解决方案销售会议总结

## 会议概述

本次会议由 AWS Marketplace 团队成员主讲，重点介绍了 AWS Marketplace 在本周推出的重要新功能——多产品解决方案（Multi-Product Solutions）。演讲者通过一个生动的厨房装修类比，说明了当今客户在云采购中面临的选择：是自己购买各个组件并组装，还是寻求完整的解决方案。这个类比恰好反映了市场的重大转变——客户不再只是寻找单个产品，而是希望获得能够满足其业务需求的完整解决方案。

AWS Marketplace 是一个拥有超过 30,000 个产品列表、涵盖 70 多个类别、拥有 6,000 多家卖家的全球性综合目录平台。随着 AI 技术的加速发展，市场需求正在发生根本性变化。业务买家越来越多地参与技术决策，他们关注的是业务结果而非单个产品功能。例如，客户会问"如何实现联络中心运营的现代化"，而不是"我需要购买哪个具体产品"。多产品解决方案功能正是为了应对这一市场转变而设计，它允许客户在一次交易中购买来自不同类别的多个产品，包括软件解决方案和专业服务，从根本上简化了购买流程。

## 详细时间线与关键要点

00:00 - 02:30 | 开场与背景介绍
- 演讲者分享个人厨房装修的故事作为类比
- 说明客户在云采购中面临类似的决策过程：自己构建、部分外包或完整解决方案

02:30 - 04:15 | AWS Marketplace 平台介绍
- AWS Marketplace 拥有超过 30,000 个产品列表
- 覆盖 70 多个产品类别
- 拥有 6,000 多家卖家
- 支持 6 种以上货币的全球市场
- 提供灵活定价、标准化采购流程和简化的供应商管理

04:15 - 06:45 | 市场转变分析
- 过去：技术团队识别产品、评估功能、自行集成解决方案
- 现在：业务买家参与技术决策，关注业务结果而非单个产品
- AI 技术加速了这一转变
- 客户带着业务结果需求来到市场，而非特定产品需求
- 示例：可观测性和监控解决方案、AI 转型等

06:45 - 09:00 | 多产品解决方案功能发布
- 本周在 AWS Marketplace 上推出的新功能
- 首次允许客户在单次交易中购买来自不同类别的多个产品
- 可以组合软件解决方案和专业服务
- 简化客户购买流程，优化合作伙伴销售流程

09:00 - 11:30 | 合作伙伴的三大转变
- 从单个产品采购转向完整解决方案
- 从通用产品转向针对特定用例和行业的解决方案
- 带来规模化能力：可在销售周期中轻松添加、删除或替换组件
- 简化采购流程，同时保持每个产品独立的条款和定价

11:30 - 14:00 | 实际案例介绍
- 展示 Accenture 和 Elastic 的合作案例
- 解决方案名称：数据准备引擎（Data Readiness Engine for GenAI）
- 包含四个产品组件：
  - Elastic Cloud（用于语义和向量搜索）
  - Elastic MCP 服务器（支持自然语言交互）
  - Accenture Aspire 连接器（大规模数据注入）
  - Accenture 数据到知识实施服务（部署和持续服务）

14:00 - 17:30 | 解决方案创建流程演示
- 合作伙伴在 Partner Central 网站创建解决方案
- 四个简单步骤：
  1. 提供基本信息（名称、描述、Logo 等）
  2. 添加产品（自有产品或其他合作伙伴产品）
  3. 添加用例（用于 AI 搜索和定制化）
  4. 预览并发布解决方案

17:30 - 19:45 | 客户发现解决方案流程
- 客户可通过新推出的代理模式（Agentic Mode）输入用例和需求
- 可使用搜索和筛选功能
- 新增"多产品"筛选器帮助发现包含多个产品的解决方案
- 解决方案详情页显示描述、用例、包含的产品、产品部署位置及集成方式

19:45 - 23:00 | 私有报价集（Offer Set）创建
- 客户点击"请求私有报价"生成 AWS 来源的商机
- 合作伙伴在 AWS Marketplace Management Portal 创建私有报价集
- 为解决方案中的每个产品创建单独的私有报价
- 灵活性：每个产品可有不同的定价条款、EULA 和期限
- 示例：SaaS 产品可采用付款计划，专业服务可采用里程碑计费，另一个 SaaS 产品可采用按需付费

23:00 - 25:30 | 客户采购流程
- 客户收到报价集可用的电子邮件通知
- 在 Marketplace Console 查看报价集
- 显示所有单个产品/组件及其价格明细
- 提供总成本便于组织审批
- 客户接受报价集后可部署软件
- 所有协议都标记回报价集 ID，便于生命周期管理

25:30 - 27:45 | 启动合作伙伴展示
- 已有 70 个解决方案上线
- 来自 65 家启动合作伙伴
- ISV 合作伙伴：Mistral AI、Trend Micro、Okta 等组合自有产品和服务
- 系统集成商和渠道合作伙伴：创建行业特定解决方案
- 示例：SEC 和 Westcon 整合 CrowdStrike 和 Zscaler 的安全解决方案
- 四家合作伙伴使用 Amazon Connect 创建联络中心解决方案

27:45 - 28:30 | 资源与结束
- 提供三个 QR 码：
  1. 查看 Marketplace 上的 70 个解决方案
  2. 合作伙伴学习如何创建解决方案的资料
  3. 企业买家了解如何购买解决方案的详细信息
