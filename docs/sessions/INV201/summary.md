# AWS re:Invent 2025 数据分析领导力会议总结

## 会议概览

本次会议由AWS数据与分析服务副总裁Mai-Lan Tomsen Bukovec主持,深入探讨了数据分析领域的三大新兴趋势。会议重点介绍了AI代理如何在数据处理的每个环节提供智能辅助,从代码编写到数据管道处理,再到数据产品使用,实现了前所未有的自动化和效率提升。AWS在过去一年中将AI深度集成到各项分析服务中,使AI成为数据工作流程中不可或缺的自然组成部分。

会议展示了AWS最新发布的高度优化的Spark 3.5.6引擎,该引擎已集成到EMR、Glue、Athena Spark和SageMaker新笔记本中。这一统一的Spark引擎显著提升了Iceberg数据格式的读写性能和交互式查询速度。AWS还推出了AI辅助的Spark升级代理,能够自动处理从旧版本到新版本的迁移工作,将原本需要数月的升级工作缩短至数周。此外,会议还邀请了Intuit和Supabase等客户分享了他们在开放架构和语义数据层方面的实践经验,展示了如何利用开放数据格式(如Apache Iceberg)构建灵活、可扩展的现代数据架构。

第三个重要趋势是向量存储技术的快速发展。AWS S3 Vectors正式发布,支持存储高达20万亿个向量,为语义理解和AI代理记忆扩展提供了强大的基础设施。结合OpenSearch的GPU加速和自动优化功能,AWS为构建大规模向量数据库提供了完整的解决方案,使客户能够在数据湖规模上实现语义理解和智能应用。
