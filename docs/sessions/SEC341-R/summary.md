# AWS re:Invent 2025 技术会议总结

## 会议概述

本次会议是AWS re:Invent 2025的一场技术分享会，会议编号为SEC 341，主题为"使用GitHub Actions自动化IAM策略验证和分析"。会议由两位AWS解决方案架构师主讲：来自悉尼的Mitch（专注金融服务客户）和来自弗吉尼亚州阿灵顿的Nitin Kulkari（身份解决方案团队成员）。

会议采用代码演示的形式，重点展示如何构建自动化流水线来验证开发者编写的IAM策略是否符合组织安全基线。演讲者介绍了从传统的集中式IAM管理模式到委托式模式的演进，并展示了如何利用AWS IAM Access Analyzer的自定义策略检查功能，结合GitHub Actions和Amazon Bedrock，为开发者提供智能化的策略验证和修复建议。
