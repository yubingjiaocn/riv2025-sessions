# AWS re:Invent 2025 分会场总结

## 会议概览

本次分会场由AWS网络和应用服务产品经理Amish Shah和Lumen公司托管安全服务高级总监Sushita Nayak共同主讲,重点介绍了AWS Network Firewall与Lumen威胁情报的集成解决方案。

AWS Network Firewall是一项完全托管的云防火墙服务,可以检查进出VPC的所有流量,并在VPC之间应用过滤策略以阻止横向移动。该服务具有自动水平扩展能力,最高可达100 Gbps,内置状态检测引擎,支持基于IP、域名、URL过滤和IDS/IPS威胁签名的规则配置。在本次re:Invent大会上,AWS宣布推出合作伙伴托管规则(Partner Managed Rules)功能,Lumen作为首批七家合作伙伴之一,将其Black Lotus Labs的威胁情报直接集成到AWS Network Firewall中。

这项集成为客户带来了革命性的简化体验。用户无需编写自定义规则、管理IP列表或维护威胁签名,只需在AWS Network Firewall控制台中点击几下即可订阅Lumen Defender托管规则组,立即获得针对恶意软件、命令与控制(C2)、僵尸网络和钓鱼攻击等威胁的防护。Lumen的威胁情报基于其全球一级互联网服务提供商网络的独特可见性,能够在攻击发生之前就识别威胁基础设施,并提供每日自动更新的规则集,这在业界是极为罕见的。Black Lotus Labs每天追踪和阻止200万个威胁,监控超过2000亿次流量会话和DNS查询,每月执行超过150次攻击基础设施的拆除行动。

## 详细时间线与关键要点

00:00 - 01:30 | 开场介绍
- Amish Shah介绍自己是AWS网络和应用服务产品经理,专注于网络防火墙和威胁情报
- Sushita Nayak介绍自己是Lumen托管安全服务高级总监
- 双方将讨论如何通过合作提升云工作负载的安全性

01:30 - 04:00 | AWS Network Firewall产品介绍
- AWS Network Firewall是完全托管的云原生防火墙服务
- 可检查进出VPC的流量,并在VPC之间应用过滤策略防止横向移动
- 完全托管意味着无需担心基础设施部署、软件维护和升级
- 自动水平扩展,最高可达100 Gbps
- 内置状态检测引擎,支持基于IP、FQDN、URL过滤的规则
- 支持IDS/IPS威胁签名规则
- 通过Firewall Manager服务实现跨AWS组织和账户的统一策略管理

04:00 - 06:30 | 合作伙伴托管规则发布
- 客户反馈:大规模管理规则是一项挑战,希望简化安全管理
- 宣布推出合作伙伴托管规则(Partner Managed Rules)
- 首批合作伙伴包括Lumen在内的七家供应商
- 提供精选的威胁情报,直接集成到Network Firewall
- 威胁情报自动更新,无需手动编写规则、管理IP列表或域名列表
- 合作伙伴的安全专家团队代表客户工作,提供最佳威胁情报
- 在攻击被发现之前就能保持领先

06:30 - 08:00 | 控制台集成演示
- 完全集成到Network Firewall控制台
- 无需访问AWS Marketplace订阅规则
- 几次点击即可将规则应用到防火墙策略
- 显著简化了大规模编写和管理策略的操作复杂性
- 控制台中可看到"添加合作伙伴托管规则"选项
- 显示所有可用规则组,包括Lumen Defender托管规则组
- 明确标注覆盖的威胁类型:恶意软件、命令与控制(C2)、僵尸网络活动等

- 完全托管的规则集,无运营成本

- 规则自动更新到防火墙,保护网络

17:30 - 21:00 | Lumen威胁情报的三大核心优势
- **早期威胁基础设施检测**:作为ISP,拥有全球最广泛的对等网络,可以看到跨网络的所有流量,能够识别好流量与坏流量,在流量进入网络时立即发现并阻止恶意流量
- **快速自动化保护**:无需用户操作,Lumen与AWS Network Firewall基础设施在后台自动更新规则,每日更新规则(业界罕见),大多数威胁情报公司每周更新几次,Lumen是唯一每日更新的供应商之一
- **具有上下文的可操作情报**:不仅提供IP、DNS、URL等冷数据,还提供可操作的上下文信息,包括威胁行为者、恶意软件类型、攻击类别(恶意软件/僵尸网络/C2/代理)、严重程度、IP信誉等,AI算法结合人工智能对IP进行排名和评级,自动分类IP并提供上下文,未来版本将提供更丰富的上下文(如威胁家族、民族国家行为者等)
