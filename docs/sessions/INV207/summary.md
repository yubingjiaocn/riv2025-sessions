# AWS re:Invent 2025 EC2 核心平台技术会议总结

## 会议概览

本次会议由AWS Amazon EC2核心平台副总裁Willem Visser主讲,重点介绍了AWS在云计算领域的最新创新成果。会议强调了在人工智能、自动化和实时分析快速发展的时代,组织需要快速适应和执行创新才能在市场中获胜。AWS通过与Intel、AMD、Apple、Nvidia等合作伙伴的深度协作,以及自研芯片Graviton和Trainium的持续迭代,为客户提供了从云端到边缘的全方位计算能力。

会议展示了多个客户成功案例,包括自动驾驶公司Zoox、数字银行Nubank、卫星互联网项目Amazon Kuiper以及AI客户服务平台Intercom等。这些案例充分证明了AWS计算服务如何帮助客户解决看似不可能的挑战。AWS每天在EC2上启动超过2.71亿个实例,自2006年以来累计启动了2870亿个实例,这种规模为AWS提供了独特的洞察力,能够持续优化基础设施的各个层面。

会议还深入介绍了AWS在运营卓越性方面的改进,包括实例启动时间优化、EBS快照复制、存储健康检查等功能。此外,Jeremy Connolly和Barry Cooks分别介绍了计算抽象层面的创新,如Lambda租户隔离、持久化函数、ECS Express模式、EKS托管能力等,旨在让开发者能够更专注于业务逻辑而非基础设施管理。
