# AWS re:Invent 2025 EC2 计算平台创新会议总结

## 会议概述

本次会议由AWS EC2核心平台副总裁Willem Visser主讲，重点介绍了AWS在计算平台方面的最新创新和突破。会议展示了AWS如何通过与Intel、AMD、Apple、Nvidia等合作伙伴的深度协作，以及自研芯片技术，为客户提供从云端到边缘的全方位计算解决方案。

会议强调了AWS在规模化运营方面的优势，每天在EC2上启动超过2.71亿个实例，自2006年以来累计启动了2870亿个实例。通过展示Zoox自动驾驶汽车、Nubank数字银行、Amazon Kuiper卫星项目等客户案例，演示了AWS计算服务如何支持各行业的数字化转型和创新应用。

## 详细时间线与关键要点

### 0:00-10:00 开场与规模展示
- Willem Visser介绍AWS EC2平台的规模优势
- 每天启动2.71亿个实例，每周超过10亿个实例
- 自2006年以来累计启动2870亿个实例
- 38个全球区域，超过1000种实例类型
