# AIM 383: 通过模型定制构建更有效的AI智能体

## 会议概述

本次AWS re:Invent 2025技术分享会由David De Galitelli（SageMaker AI团队全球高级专家解决方案架构师）和Sam Palani（AWS Nova产品上市团队负责人）共同主持。会议重点介绍了如何使用Amazon Nova基础模型构建定制化AI智能体，涵盖了从基础概念到高级定制技术的完整流程。

演讲者详细阐述了AI智能体的核心定义——在循环中调用工具来完成目标的大语言模型，并深入探讨了推理模型的突破如何推动了智能体AI的发展。会议特别强调了模型定制在智能体应用中的重要性，包括减少复合错误、适应特定业务语义、优化工具调用等关键价值。内容涵盖了从简单的提示工程到复杂的持续预训练等多种定制技术，并通过实际演示展示了在Amazon Bedrock和SageMaker平台上的具体实现方法。

## 详细时间线与关键要点

### 0:00-5:00 开场介绍与议程概览
- 介绍两位演讲者：David De Galitelli和Sam Palani
- 会议议程：构建定制AI智能体、Amazon Nova模型家族、模型定制技术、实际演示和资源分享
- 强调这是一个静默会议，内容丰富且紧凑

### 5:00-10:00 AI智能体基础概念
- AI智能体定义：在循环中调用工具来完成目标的LLM
- 智能体工作机制：模型+工具+目标→行动→分析结果→循环直至完成
- 智能体AI发展的三大推动因素：推理模型突破、安全数据基础设施建设、专业工具和技术的出现

### 10:00-15:00 智能体AI的市场趋势与定制必要性
- Gartner预测：到2028年1/3的企业应用将由生成式AI或智能体AI驱动
- 15%的日常业务决策将由AI智能体自主完成
- 定制化的四大价值：减少复合错误、适应业务语义、优化工具调用、压缩工作流链条

### 15:00-20:00 模型定制技术分类
- 定制化定义：修改模型行为的任何操作
- 两大类别：优化输出（无权重更新）vs 修改模型（权重更新）
- 优化输出技术：提示工程、RAG、上下文工程
- 模型修改技术：后训练（微调、蒸馏、强化学习）和预训练（持续预训练）

### 20:00-25:00 Amazon Nova模型家族介绍
- 理解推理模型：Nova Micro（最快最高效）、Nova Light/Pro/Premiere（多模态，智能递增）
- 创意内容生成：Nova Canvas（文本到图像）、Nova Reels（文本到视频）
- 新发布模型：Nova 2 Pro（最高性能）、Nova 2 Omni（图像生成）、Nova Sonic（实时语音）、Nova Act（浏览器智能体）
- Nova多模态嵌入模型：业界唯一支持文本、视频、图像、音频统一向量空间的嵌入模型

### 25:00-30:00 Amazon Bedrock上的Nova定制选项
- 性能高效微调：标注数据上传→微调作业→模型部署
- 应用场景：工具调用模式一致性、角色切换（销售/支持）
- 模型蒸馏：教师模型+学生模型+任务提示→自动生成响应→微调学生模型
- 部署选项：按需推理（按token付费）vs 预置吞吐量（固定价格保证吞吐量）

### 30:00-35:00 SageMaker高级定制功能
- SageMaker优势：端到端ML平台，提供更底层控制
- 两种训练方式：Training Jobs API（全托管无服务器）vs HyperPod（完全自定义编排）
- HyperPod特性：基于EKS或SLURM的托管集群，支持工作负载调度优化
- 优化配方：预构建训练脚本，专为AWS架构优化

### 35:00-40:00 实际演示与评估部署
- SageMaker Studio演示：模型选择→训练选项→集群配置→Jupyter notebook
- 模型评估方法：统计指标（困惑度、准确率）、任务特定指标、LLM评判
- 部署选项：Bedrock自定义模型导入、SageMaker实时推理端点、HyperPod推理

### 40:00-43:24 生产化与总结
- 开源框架支持：Strands Agents、LangGraph、LangChain等
- Amazon Bedrock Agent Core：端到端智能体部署管理平台
- 实施建议：先测试基础模型→评估定制需求→选择合适技术→评估性能→部署生产
- 学习资源：AWS Skill Builder平台提供Nova相关培训
