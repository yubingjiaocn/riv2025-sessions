# AWS re:Invent 2025 Snowflake Intelligence 会议总结

## 会议概述

本次会议由Snowflake人工智能产品营销高级总监Santiago Geraldo和Desile工程高级副总裁Brian Newman共同主讲，重点介绍了Snowflake Intelligence这一创新解决方案。会议深入探讨了企业在数据分析和决策制定过程中面临的核心挑战：业务团队与数据团队之间存在的巨大鸿沟，以及传统静态报告无法满足实时决策需求的问题。

Snowflake Intelligence作为企业智能代理，旨在通过生成式AI技术打破这些障碍，使企业中的每个员工都能够通过简单的自然语言提问获得深度分析结果。该解决方案不仅提供透明可验证的答案，还能追溯到原始数据源，确保决策的可信度。Brian Newman分享了Desile作为早期采用者的实践经验，展示了如何利用Snowflake Intelligence快速构建AI驱动的电商分析平台，大幅提升客户自助分析能力，减少75%的支持请求。

会议强调了Snowflake Intelligence的三大核心原则：深度分析能力（不仅回答"发生了什么"，更能回答"为什么"和"下一步该做什么"）、可信度（透明且可验证的答案）、以及企业就绪性（原生集成于Snowflake平台，确保安全和治理）。通过DBT模型、语义视图和Cortex API的无缝集成，企业可以快速部署智能代理，实现数据民主化。
