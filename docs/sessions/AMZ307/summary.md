# AWS re:Invent 2025 技术会议总结

## 会议概述

本次会议是AWS re:Invent 2025"向Amazon学习"系列的一部分，由AWS全球媒体娱乐策略团队高级合作伙伴解决方案架构师Jason Omali和Prime Video及Amazon MGM Studios全球制作与后期制作技术负责人Greg Young共同主讲。会议重点介绍了Amazon MGM Studios如何通过"云端工作室"(Studio in the Cloud)概念革新影视制作流程，实现从传统本地制作向云原生工作流的转型。

该会议展示了Amazon MGM Studios如何利用AWS云服务优化整个内容制作生命周期，从主体摄影到后期制作的九个核心工作流程。通过将创意人员带到内容所在的云端，而不是将内容运送到全球各地的创意人员那里，他们成功减少了制作时间、降低了成本，并显著减少了碳足迹。会议还特别介绍了一个创新的移动上行链路摄像头到云端工作流解决方案，以及一辆定制的Rivian电动车，展示了可持续制作的未来愿景。

## 详细时间线

### 开场介绍 (0:00-2:30)
- **0:00-0:30**: Jason Omali开场，介绍"向Amazon学习"系列的目的和价值
- **0:30-1:00**: 介绍演讲者背景 - Jason作为AWS媒体娱乐策略团队成员，Greg作为Prime Video制作技术负责人
- **1:00-2:30**: 展示Amazon运营规模数据，包括Prime Day的万亿级处理能力

### 云端工作室概念介绍 (2:30-8:00)
- **2:30-4:00**: Greg介绍LOL节目案例 - 70多季的多摄像头格式节目，40-50个摄像头同时工作
- **4:00-5:30**: 传统制作方式的挑战 - 大量广播车、本地存储租赁、设备租赁和发电机
- **5:30-8:00**: 云端解决方案 - 摄像头流直接传输到AWS，洛杉矶团队远程指导，延迟不到1秒

### 制作工作流程详解 (8:00-12:00)
- **8:00-10:00**: 九个核心工作流程介绍：主体摄影、每日素材、剪辑、套底、视觉效果、调色、声音、母版制作和质量控制
- **10:00-12:00**: 现实中的复杂性 - 涉及全球数百个供应商、卡车、发电机、存储租赁和运输承运商

### 云原生转型策略 (12:00-16:00)
- **12:00-14:00**: 传统归档方式的转变 - 从制作结束后上传AWS转为尽早上传内容
- **14:00-16:00**: 与供应商合作 - Avid、Colorfront、Adobe、Blackmagic等工具的云端优化

### 技术架构深入分析 (16:00-22:00)
- **16:00-18:00**: 传统工作流程的低效性 - 多次复制、物理运输、资源浪费
- **18:00-20:00**: 云原生架构 - 直接上传到S3、蜂窝网络连接、远程编辑桌面
- **20:00-22:00**: 详细架构组件 - 共享服务VPC、生产专用账户、Avid Nexus共享存储

### 可持续发展目标 (22:00-25:00)
- **22:00-23:30**: Amazon气候承诺 - 2040年实现净零碳排放
- **23:30-25:00**: 行业影响数据 - 51%的碳足迹来自旅行和运输，Albert组织的环境影响研究

### Rivian移动解决方案 (25:00-28:00)
- **25:00-26:30**: 定制Rivian R1S介绍 - 205千瓦电池、1000瓦太阳能、多载波移动互联网
- **26:30-28:00**: 实际应用场景 - 《指环王：力量之戒》第一季的远程乡村拍摄挑战

### 成果与未来规划 (28:00-30:00)
- **28:00-29:00**: 具体成效 - 每日素材处理速度提升50%，制作时间节省数天到数周
- **29:00-30:00**: 规模化目标 - 2025年完成70个制作项目，2026年目标超过100个云端制作项目

### 结束与互动 (30:00-31:00)
- **30:00-31:00**: 现场展示邀请 - 参会者可在AWS可持续发展展区实地查看Rivian车辆，鼓励反馈和持续学习
