# AWS re:Invent 2025 分会场总结

## 会议概述

本次分会场由 CDG 公司的 CTO Tony Stout 主讲,重点介绍了一家拥有 55 年历史的中西部电信公司如何通过 AWS 云服务实现数字化转型的故事。会议开场强调了美国数字鸿沟问题——八分之一的美国人仍然无法在家中接入互联网,这影响了远程医疗、远程办公等基本需求。CDG 作为一家为电信行业提供计费、用户管理和运营支持系统的 ISV 公司,总部位于伊利诺伊州香槟市,服务对象包括为美国小城镇和农村地区提供服务的独立服务提供商。

Tony 详细阐述了 CDG 如何从传统数据中心的遗留基础设施转型为云原生、AI 原生的现代化平台。三年半前,CDG 面临着基础设施老化、技术债务累积、需要提升市场竞争力等多重挑战。通过与 AWS 的深度合作,CDG 不仅完成了应用迁移,还构建了名为 "CDG Elements" 的下一代平台,并在生成式 AI 领域成为行业创新领导者。这一转型带来了显著的业务增长——连续三年销售额翻倍,营收增长高达 58%,新客户规模是以前的 5-10 倍。

会议还展示了 CDG 如何利用 AWS Bedrock、Amazon Connect 等服务开发 AI 聊天机器人、语音机器人、AI 运维解决方案等创新应用,帮助客户提升运营效率,缩短服务部署时间,最终助力缩小美国农村地区的数字鸿沟。Tony 分享了多个实际案例和经验教训,强调了战略规划、合作伙伴关系、创新文化和以客户为中心的重要性。
