# AWS re:Invent 2025 - AMD EPYC CPU 在通用计算和 AI 工作负载中的应用

## 会议概述

本次 AWS re:Invent 2025 分会场重点介绍了 AMD EPYC CPU 如何在通用计算和 AI 工作负载中提供卓越性能和成本效益。会议由 AMD 服务器产品团队负责人 Madhur Ranjan、AMD 公有云产品负责人 Mike Thompson 以及 CVS Health 的 FinOps 实践负责人 Kyle McLaughlin 共同主讲。

AMD 强调其作为端到端解决方案提供商的定位，不仅提供 CPU 和 GPU 硬件，还通过软件层和系统级设计（通过收购 ZT Systems）提供完整的解决方案。第五代 AMD EPYC（代号 Turin）处理器在各类工作负载中展现出显著的性能优势，相比竞争对手可提供 2-4 倍的性能提升。特别值得注意的是，随着 AI 推理成本在 18 个月内下降了 280 倍（从 2022 年的每百万 token 20 美元降至 2024 年的 7 美分），CPU 在 AI 推理工作负载中的角色变得越来越重要。

会议还展示了实际案例，Mike Thompson 详细说明了通过选择 AMD 驱动的实例可以实现 27-45% 的成本节省，而 CVS Health 的 Kyle McLaughlin 分享了他们如何通过 FinOps 实践和标准化 SKU 选择来优化云支出，实现显著的运营成本降低。对于 AI 推理工作负载，特别是小型到中型模型、非实时场景以及混合工作负载，CPU 提供了更具成本效益和可用性的解决方案。
