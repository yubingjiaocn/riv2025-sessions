# AWS re:Invent 2025 会议总结：智能垃圾分类解决方案

## 概述

本次会议由来自AWS智利圣地亚哥办公室的Gonzalo Vasquez和Rodrigo Mon主讲，介绍了一个创新的AI驱动智能垃圾分类系统。该项目源于办公室日常面临的实际问题：员工在处理废弃物时经常不确定应该将物品放入哪个回收容器（纸板、塑料、堆肥还是垃圾填埋）。演讲者展示了一个已在智利AWS办公室实际部署的解决方案，用户只需将物品放在摄像头前并按下按钮，系统就能在3-5秒内通过Amazon Nova Pro模型识别物品并显示正确的分类结果。

该解决方案不仅在AWS办公室内部使用，还在AWS Experience活动中投入生产环境，部署了三个回收站点，成功实现了零污染事故。项目背景源于拉丁美洲地区严峻的废弃物管理现状：预计到2050年废弃物将增长250%，而智利每年产生1700万吨垃圾，但回收率极低，大部分废弃物最终进入垃圾填埋场。该系统采用混合架构，结合边缘计算和云端AI处理，成本低廉（硬件成本仅350美元），且完全采用无服务器架构，实现了95%的分类准确率，使办公室回收效率在六个月内提升了52%。

该项目计划扩展到更多AWS办公室（包括布宜诺斯艾利斯、波哥大、圣保罗、阿灵顿和纽约），甚至吸引了客户的兴趣。团队承诺将在未来几周内通过GitHub AWS Samples代码库开源该解决方案，让更多组织能够部署这一环保技术。

## 详细时间线

00:00:00 - 00:01:30 - 开场与问题引入
- 演讲者以午餐后处理披萨盒的场景引入主题
- 介绍在AWS智利圣地亚哥办公室面临的垃圾分类难题
- 展示实时演示视频：系统识别塑料瓶并显示分类结果

00:01:30 - 00:03:00 - 区域背景与环境挑战
- 拉丁美洲废弃物预计增长250%
- 智利回收率极低，每年产生1700万吨垃圾
- 大部分废弃物最终进入垃圾填埋场

00:03:00 - 00:04:30 - 系统工作流程介绍
- 用户将物品（如香蕉皮、塑料瓶、披萨盒）放在摄像头前
- 按下彩色编码按钮（绿色按钮）
- Amazon Nova Pro模型在Amazon Bedrock上进行推理
- 3-5秒内在屏幕上显示分类结果和指导

00:04:30 - 00:06:00 - 硬件组件说明
- 触发拍照的按钮
- 3D打印外壳内的Raspberry Pi
- 普通网络摄像头（图像经过压缩和调整大小以提高效率）
- Fire平板电脑显示结果、分类图像、参考标签和统计数据

00:06:00 - 00:07:30 - 实际部署案例
- 在AWS Experience活动中投入生产
- 部署了三个回收站点
- 主要分类为纸张，无污染事故报告
- 目前已在圣地亚哥部署，计划扩展到布宜诺斯艾利斯、波哥大、圣保罗、阿灵顿和纽约

00:07:30 - 00:09:00 - 架构概述（Rodrigo讲解）
- 混合架构：边缘组件+云端处理
- 边缘层：Raspberry Pi、网络摄像头、按钮、显示屏
- 云端层：使用AI处理图像

00:09:00 - 00:11:00 - 边缘处理层详解
- AWS IoT Greengrass：开源运行时，在边缘设备上本地处理数据
- AWS IoT Core：通过MQTT、HTTPS和LoRaWAN协议连接设备
- 组件包括：网络摄像头、Nucleus（Greengrass运行时）、Smart Bin组件（包含Python脚本用于图像压缩）

00:11:00 - 00:13:00 - 数据传输与事件处理
- Stream Manager负责将图像上传到云端，处理本地缓冲和重试
- 图像上传到Amazon S3
- Amazon EventBridge捕获S3上传事件并路由到Step Functions管道
- AWS Step Functions编排AI工作流

00:13:00 - 00:15:30 - AI推理流程
- AWS Lambda函数预处理图像并调用Amazon Bedrock
- 使用Amazon Nova Pro多模态模型分析文本和图像
- 确定废弃物分类类型和置信度分数

00:15:30 - 00:18:00 - 提示词（Prompt）设计详解
- 角色设定：定义AI为废弃物分类专家
- 任务定义：分步指令（分析图像、根据回收规则确定分类、返回0-1之间的置信度分数、使用JSON格式输出）
- 规则定义：针对特定设施的分类规则（如智利办公室只回收塑料瓶，不是所有塑料）

00:18:00 - 00:19:30 - 数据存储与显示
- AWS AppSync提供GraphQL API层
- 结果存储到DynamoDB表中
- 支持实时数据查询
- AWS Amplify提供无服务器托管和部署
- 平板电脑上的Web界面显示分类结果和回收效率仪表板

00:19:30 - 00:21:00 - 用户体验总结
- 无缝体验：接近站点、按下按钮、获得即时反馈
- 用户可将物品放入正确的容器

00:21:00 - 00:23:00 - 影响评估与成本分析
- 准确率：达到近95%的分类准确率
- 硬件成本：350美元
- 使用成本：每1000张图像4美元
- 完全无服务器架构，无使用时几乎零成本（仅S3存储成本）

00:23:00 - 00:24:30 - 环境影响
- 六个月评估期内，办公室回收效率提升52%
- 数据来自建筑物业主提供的月度指标（不同材料容器的重量统计）

00:24:30 - 00:26:00 - 可扩展性与未来计划
- 易于扩展到其他办公室和客户场所
- 通过自然语言更新提示词规则即可适配不同场所
- 成本效益高，结果可衡量
- 计划在未来几周通过GitHub AWS Samples开源

- 提供英文版博客文章（原版为西班牙语）
