# AWS re:Invent 2025 智能代理构建会议总结

## 会议概述

本次AWS re:Invent 2025第四天的分组会议重点讨论了构建智能代理的核心挑战和解决方案。演讲者强调，尽管会议标题提到了MongoDB和Voyage的应用，但本次会议更侧重于教育性内容，特别是代理记忆架构的基础知识。

演讲者指出，根据MIT的一项研究，95%的组织在AI代理实施中未能看到实际价值。这个问题的根源不在于模型质量、监管或基础设施，而在于代理工作流和系统的设计。大多数代理失败的原因是记忆失败，而非推理失败，因为当前大多数代理本质上是无状态系统，却试图表现得像有状态系统。

演讲者从人类学角度分析了这个问题，指出人类历来通过构建外部记忆系统（如口述传统、书籍、图书馆、互联网）来扩展智能。AI代理同样需要这种外部记忆系统来持久化知识、经验和状态。
