# AWS re:Invent 2025 - 掌握碳足迹：AWS客户碳足迹工具深度解析

## 会议概述

本次技术分享会由AWS可持续发展技术团队负责人Alexis Bateman主讲，重点介绍了AWS客户碳足迹工具的构建原理和技术实现。作为AWS可持续发展目标的核心支撑团队，该团队负责支持AWS的可再生能源目标、水资源正效益以及净零碳排放等重要环保倡议。

会议深入探讨了AWS如何通过大规模数据处理和科学建模方法，将企业级碳足迹数据转化为客户可用的碳排放分析工具。Bateman详细阐述了从数据摄取、碳化处理到客户分配的完整技术架构，展示了AWS在可持续发展领域的技术创新和工程实践。

- Alexis Bateman介绍AWS可持续发展技术团队职责
- 团队作为AWS可持续发展目标的技术支撑backbone

### 2:00-5:00 AWS可持续发展目标概览
- 气候承诺签署方：与591家企业共同承诺2040年实现净零碳排放
- 2023年提前实现100%可再生能源目标
- 电力使用效率(PUE)达到1.15，优于行业平均1.25
- 循环经济项目：自2023年回收超过2300万个组件
- 水资源正效益目标：2030年实现完全水资源正效益，目前完成53%

### 5:00-8:00 AWS运营规模与效率优势
- 运营规模：117个可用区，覆盖245个国家和地区，37个地理区域
- 相比本地计算提供4.1倍的能效优势
- 通过规模化运营支持客户可持续发展目标

### 8:00-12:00 客户碳足迹工具技术架构
- 数据流程：摄取数百万个数据集，从硬件组件到数据中心建材
- "碳化"过程：科学团队建模转换各类数据为碳排放数值
- 分配模型：将AWS总体碳足迹按客户使用量进行分配
- 数据源类型：硬件组件、数据中心设备、能源消耗、水资源使用

- 支持内部采购、优化和规划决策
