# AWS re:Invent 2025 技术会议总结

## 会议概述

本次会议展示了智利国家银行（Banco Estado）与IBM咨询和AWS合作开发的创新住房生态系统平台。会议由IBM咨询智利金融服务合作伙伴Bruno Vacetello主持，智利国家银行商业贷款总经理Soledad Ovando参与分享。会议重点讨论了人工智能在银行业的应用现状，特别是代理AI（Agentic AI）在金融服务中的实际部署和规模化挑战。

会议后半部分由IBM自动化开发负责人Vikram Murali介绍了企业级代理AI的监控、优化和治理解决方案，并邀请了Global Payments和FlexiVan的高管分享他们在AI应用方面的实践经验。整个会议展现了从概念验证到生产环境部署的完整AI转型路径。

## 详细时间线与关键要点

### 0:00-10:00 开场与AI在银行业的现状
- Bruno Vacetello介绍会议背景，强调AI已成为银行业转型的核心
- 根据IBM商业价值研究院研究，四分之一的组织尚未在智能工作流中使用AI
- 预测到2026年底，银行业智能工作流将增长12倍
- 回顾IBM与智利国家银行60年合作历史，从1969年首个远程处理网络开始

### 10:00-20:00 智利国家银行介绍
- Soledad Ovando介绍银行基本情况：160年历史，政府控制的银行
- 服务2000万智利人口中的1750万客户，拥有1200万活跃用户
- 银行应用在智利下载量第一，50%用户每日使用
- 三大战略重点：数字客户体验、金融生态系统发展、国家可持续发展

### 20:00-30:00 住房生态系统平台展示
- 播放产品演示视频，展示数字化住房购买流程
- 平台整合政府、房地产公司、建筑公司、公证处等各方参与者
- 基于机器学习的风险模型，为更多智利人提供住房贷款机会
- 创建统一的从建筑许可到房屋管理的完整体验

### 30:00-40:00 技术实施与业务目标
- Bruno详细介绍新业务模式和技术平台架构
- 关键业绩指标：抵押贷款审批时间减少15天，结算时间提升30%
- 客户与企业互动减少50%，特定房产实现零天评估争议
- 处理成本逐步下降，系统完全运营后达到30%的成本节约

### 40:00-45:00 产品发布公告
- 正式宣布住房平台将于12月9日上线
- 网址：casaverso.cl
- 强调这是智利首个此类综合性房地产平台

### 45:00-55:00 企业级代理AI挑战与解决方案
- Vikram Murali介绍代理AI规模化面临的三大挑战：动态复杂性、多样化基础设施连接、快速技术变化
- 提出企业级代理AI的四个关键要素：观察、优化、自动化、治理
- 介绍IBM解决方案：Instana（应用性能监控）、Turbonomic（资源优化）、webMethods（安全治理）

### 55:00-60:00 客户案例分享
- Global Payments副总裁Ganesh Somaskandan分享支付行业应用：网络星期一处理15亿笔交易
- FlexiVan CIO Sagar Chikkala介绍物流行业AI应用：从预测物流到AI视觉、企业GPT和代理AI
- 两位客户强调数据集成的重要性，以及在保持安全性的同时实现AI自动化的平衡
