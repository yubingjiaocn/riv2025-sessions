# AWS re:Invent 2025 技术会议总结

## 会议概述

本次会议由AWS支持团队的高级首席技术专家Sab主持，与同事Tepu以及来自Whoop公司的Dwight Bidd共同分享了AWS支持服务如何通过AI技术实现转型升级。会议重点讨论了AWS支持团队从传统的被动响应模式向主动预防模式的转变，以及这种转变如何为客户带来更好的体验和业务成果。

会议展示了AWS支持服务15年来的演进历程，从最初的论坛式支持发展为今天包含多个层级的综合支持体系。随着云工作负载复杂性的不断增加，传统的"救火式"支持模式已无法满足客户需求。因此，AWS推出了全新的统一运营(Unified Operations)服务，这是专为关键业务工作负载设计的顶级支持计划，提供5分钟响应时间和专门的技术团队支持。
