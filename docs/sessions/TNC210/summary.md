# AWS re:Invent 2025 - Amazon QuickSuite 会议总结

## 会议概述

本次会议由 Prakash Best 主讲,主题为"Amazon QuickSuite：探索新功能和特性"。Amazon QuickSuite 是构建在 Amazon QuickSight 之上的全新服务,集成了生成式 AI 和代理式 AI 能力,旨在为企业提供统一的商业智能、数据分析和工作流自动化平台。

与传统 BI 系统不同,QuickSuite 允许用户使用自然语言与数据交互,无需编写复杂查询。该服务提供了多项核心功能,包括聊天代理(Chat Agents)、协作空间(Spaces)、工作流自动化(Flows)、深度研究(Research)以及企业级自动化(Quick Automate)。服务具有易于部署、企业级安全(内置 AES-256 加密)、自动扩展等特点,并支持与多种数据源的集成。演讲者通过现场演示展示了如何使用虚拟咖啡公司的场景,演示了创建聊天代理、设置协作空间、构建自动化工作流以及生成市场研究报告等实际应用。
