# AWS re:Invent 2025 网络安全分会场总结

## 会议概述

本次AWS re:Invent 2025网络安全分会场由Sophia Alumas Santos（网络和应用保护服务市场负责人）和Salem Muhammad（产品管理负责人）共同主持。会议重点介绍了AWS在网络和应用保护领域的最新创新，这些创新都是基于过去12个月与客户的深入交流和反馈而开发的。

演讲者强调了AWS网络安全服务的核心设计原则：支持复杂的多云和混合架构、提供可扩展的性能、满足合规和数据主权要求、降低团队认知负担、增强可见性，以及应对AI时代不断演变的威胁。会议详细介绍了AWS网络和应用保护产品组合，包括AWS WAF、Shield Advanced、Network Firewall、Route 53 Resolver DNS Firewall、Firewall Manager和新推出的Shield Network Security Director等服务。

整个演讲展示了AWS如何通过其全球基础设施规模（每分钟处理EB级数据，每天缓解数千次DDoS攻击）来收集威胁情报，并通过MadPot等欺骗技术项目主动识别威胁。这些威胁情报被整合到AWS的安全服务中，为客户提供更强大的保护。
