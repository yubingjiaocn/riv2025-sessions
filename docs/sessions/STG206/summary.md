# AWS re:Invent 2025 S3 新功能发布会总结

## 会议概述

本次会议由AWS S3团队的Paul Meighan和Anna Bouchet主持，全面回顾了S3团队在过去12个月中推出的35项新功能和改进。会议采用了创新的互动演示形式，邀请观众Matt和Indra Neil参与复杂的技术演示，展示了从安全控制到向量存储等多个领域的重大进展。

演讲者强调这是一场200级别的技术会议，旨在快速覆盖大量新功能，为开发者和架构师提供S3生态系统的最新发展概览。会议内容涵盖了安全性、持久性、高性能存储、对象管理、元数据处理和向量存储等六个核心主题，展现了S3服务在云存储领域的持续创新和发展。

## 详细时间线与关键要点

### 00:00-10:00 开场介绍与安全功能更新
- **会议开场**：介绍演讲者和会议结构，预告将覆盖35项新功能
- **基于标签的访问控制**：简化多存储桶、多客户端环境下的权限管理
  - 使用简单策略配合资源标签控制权限
  - 新增单标签操作API：TagResource、UntagResource、ListTagsForResource
  - 支持所有S3存储桶类型和资源
- **组织级控制**：将Block Public Access和403上下文功能扩展到AWS Organizations级别
- **加密更新**：
  - 推出PQ TLS后量子加密，保护传输中数据
  - SSEC加密选项默认关闭，2026年新存储桶将默认禁用

### 10:00-20:00 演示环节与持久性功能
- **互动演示启动**：使用Amazon Transcribe处理视频，上传700个客户案例文件
- **访问拒绝调试**：展示增强错误消息功能，快速定位权限问题
- **11个9持久性**：
  - 端到端完整性检查
  - 多设备冗余存储
  - 持续的持久性审计
- **数据完整性检查**：新功能允许高效验证S3中静态数据的校验和

### 20:00-30:00 S3 Express高性能存储更新
- **性能提升**：价格降低高达85%，TPS限制提升至200万读取/20万写入每秒
- **新功能**：
  - 对象重命名API
  - 访问点支持
  - 单位数毫秒级延迟保持

### 30:00-40:00 S3对象管理与API增强
- **条件操作扩展**：
  - 条件复制：防止意外覆盖
  - 条件删除：确保删除正确对象
  - 成为最快采用的功能之一
- **大对象支持**：最大对象大小从5TB提升至50TB
- **批量操作改进**：
  - 无清单选项：直接处理整个存储桶或前缀
  - IAM角色自动创建
  - 支持高达200亿对象的作业
  - 性能提升10倍

### 40:00-50:00 S3 Tables与元数据功能
- **S3 Tables成熟度**：
  - 超过40万张表创建
  - Iceberg REST目录端点
  - 表级别CloudWatch和CloudTrail支持
  - KMS表级加密
- **表复制功能**：提供Iceberg表的只读副本，包含目录一致性保证
- **智能分层**：为S3 Tables提供自动成本优化，最高节省80%存储成本
- **SageMaker Unified Studio集成**：一键式SQL编辑器和自然语言查询支持
- **S3元数据**：
  - 日志表记录所有S3变更
  - 实时清单表提供时点快照
  - 价格降低33%

### 50:00-57:00 S3 Vectors向量存储与总结
- **向量概念解释**：将数据转换为数值列表，实现语义相似性计算
- **S3 Vectors特性**：
  - 向量存储成本降低90%
  - 100毫秒级延迟
  - 支持每个索引20亿向量，每个向量存储桶1万个索引
- **集成能力**：
  - OpenSearch冷层集成
  - Bedrock知识库支持RAG应用
- **最终演示**：使用Bedrock知识库和S3 Vectors成功识别会议志愿者姓名
- **会议总结**：35项功能发布，感谢参与者，鼓励反馈评价
