# AWS re:Invent 2025 会议总结：联合航空如何破解中小企业市场密码

## 会议概述

本次分享会由ZS咨询公司旅游与酒店业务负责人Kunal和联合航空B2B营销及国际营销负责人Jane共同主讲，重点探讨了联合航空如何成功开拓中小企业(SMB)市场。Jane分享了她在联合航空六年来推动这一全新业务能力的经验，特别是如何通过构建完整的生态系统来服务这一此前渗透率极低的市场细分领域。

联合航空在2019年通过市场分析发现，虽然与全球大型企业建立了深厚的合作关系，但在中小企业市场的渗透率仅为个位数。主要原因是缺乏能够盈利地服务这一群体的能力。传统销售模式的投资回报率不足以支撑对中小企业的开发。因此，联合航空决定通过技术创新和营销自动化来高效触达这一市场，建立了被内部称为"小企业需求中心"(Small Business Demand Center)的全新能力体系。

该项目的成功关键在于将其定位为一个完整的生态系统，而非单纯的技术项目。这个生态系统包含四个核心组成部分：客户注册门户网站、数字营销能力、数据与算法支持、以及专门的运营团队。通过这种整体性方法，联合航空不仅成功吸引了大量高价值的增量客户，还为未来的持续优化和扩展奠定了坚实基础。

## 详细时间线与关键要点

[00:00-02:30] 开场介绍与背景
- Kunal介绍自己是ZS咨询公司旅游与酒店业务负责人
- Jane分享刚从意大利威尼斯出差返回，在联合航空工作六年，负责B2B营销团队
- Jane表示很高兴能分享推出这一全新SMB业务能力的经验

[02:30-05:00] 识别市场机会
- 2019年Jane入职联合航空时，ZS帮助分析了整个美国商务旅行市场格局
- 发现最大的机会在于SMB市场——联合航空与大型企业有深厚关系，但在小企业市场渗透率仅为个位数
- 根本原因：缺乏能够盈利地服务小企业的能力
- 这不是联合航空第一次尝试这个市场，之前的"United Business Go"项目曾经失败

[05:00-08:30] 构建生态系统的四大支柱
- Jane强调关键问题："这次我们要做什么不同的事情？"
- 核心理念：不是一项技术，而是一个生态系统，包含四个部分：
  1. 门户网站：供SMB客户注册项目的平台
  2. 数字营销能力：驱动客户访问网站并完成注册
  3. 数据支持：构建目标客户列表、细分市场、吸引客户
  4. 专业团队：协调整个生态系统运作
- 强调必须同时具备这四个要素才能建立可扩展的长期基础

[08:30-11:00] 理解双层影响者模型
- 商务旅行市场有两类买家：
  - 旅行管理者（CEO、CFO、采购负责人）：决定是否加入项目
  - 实际旅行者：决定是否预订航班
- 针对不同角色传递不同信息：向管理者强调成本节省和报告工具，向旅行者强调便利性
- 技术实现：只需将工作邮箱关联到商务账户，折扣自动显示在应用中
- 利用现有的B2C旅行者生态系统，无需重新发明轮子
- Jane推荐联合航空的应用是行业第一

[11:00-15:30] 关键成功因素与经验教训
- **讲好故事并持续传播**：构建了严密的商业案例，在组织各层级反复宣讲同样的内容
- **内部品牌化**：将项目命名为"小企业需求中心"，明确定义为一种新能力而非技术或平台
- **不要等待完美才开始**：最初计划构建四个机器学习算法，但选择先启动再逐步优化
- 门户功能仍在持续开发中（启动两年后）
- **基于真实客户反馈优化**：使用实际客户数据训练算法，而非第三方假设数据
- 发现最初提供的三个选项中，90%的客户选择了同一个，说明产品差异化不够明显

[15:30-18:00] 保持耐心与专注核心指标
- 项目不是短期工程，需要长期投入和算法微调
- 第一年未达到收入目标，但向财务部门解释时强调了真正重要的指标
- 核心成功指标：高收益增量客户（high yield incremental revenue）
- 虽然账户数量和总收入未达标，但带来的是高价值客户，投资回报率仍然为正
- 获得了跨职能团队的一致支持和领导层的持续承诺

[18:00-21:00] 团队与运营模式的重要性
- 项目资金中包含了与ZS合作构建团队、编写职位描述和运营模式的预算
- 明确新团队如何在现有组织结构中运作，建立标准操作流程(SOP)
- **招聘合适的领导者至关重要**：需要具备以下特质：
  - 协作能力强
  - 能够在灰色地带工作（不再有非黑即白的答案）
  - 极强的主人翁意识
  - 能够打破障碍、推动事情发生
- Jane强调这是她现在评估人才的首要标准

[21:00-25:00] 未来发展方向
- **产品优化**：推出基于积分和奖励的新计划（2026年中期上线）
  - 现有计划满足了只关注折扣的客户
  - 新计划针对重视体验和免费福利的客户群体
  - 通过旅行累积积分兑换免费服务
- **算法持续改进**：
  - 正在开发"下一步最佳行动"算法
  - 基于潜在客户来源、内容互动情况和历史转化数据推荐营销行动
  - 这是ZS最初提议的四个算法中的最后一个
- **生成式AI聊天机器人升级**：
  - 目前仅有两名员工服务数万客户，聊天机器人提高了效率
  - 现有聊天机器人不够智能，经常需要升级到人工处理
  - 需要提升生成式AI能力以支持规模化扩展

[25:00-26:30] 总结与建议
- Jane的核心建议：
  - 讲好你的故事，保持清晰和紧凑
  - 反复讲述你的故事
  - 专注于对整个组织真正重要的指标
- Kunal总结：成功的关键在于投资正确的基础设施，保持耐心，持续投入，最终达到理想状态
- 会议在预定时间准时结束
