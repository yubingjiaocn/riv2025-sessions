# AWS re:Invent 2025 数据库服务主题演讲总结

## 一、会议概述

本次AWS re:Invent 2025数据库服务主题演讲由AWS数据库服务副总裁G2 Krishnamoorthy和Colin Lee Kear共同主持,重点探讨了AWS数据库服务在AI代理时代的战略愿景和技术创新。演讲强调,在生成式AI和代理AI的变革性机遇下,**数据是企业的核心差异化因素**,AWS致力于成为客户构建AI代理平台的最佳长期合作伙伴。

AWS预测到2028年将有超过13亿个AI代理投入生产使用,这将带来应用程序和数据量的指数级增长。为应对这一规模,AWS正在五个核心构建模块上进行创新:轻松扩展、持续自动优化基础设施、以可信数据为核心构建AI代理基础、拥抱开放架构,以及实现前所未有的全球规模下的安全性、可靠性和卓越运营。演讲中还邀请了Vercel的首席产品官Tom Aquino和Robinhood的软件工程总监Tim Ludiker分享了他们基于AWS数据库构建创新解决方案的实践经验。

整场演讲传达的核心信息是:AWS正在让数据库服务变得"毫不费力"(effortless)——从快速启动、无缝迁移、智能管理到与AI工具的深度集成,使开发者能够以"思想的速度"进行构建,专注于创新而非基础设施管理。
