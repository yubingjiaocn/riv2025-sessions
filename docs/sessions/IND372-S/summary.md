# AWS re:Invent 2025 技术会议总结

## 会议概述

本次会议由Cloudera全球AI解决方案总监Adrien主讲，主题为"使用代理AI现代化信贷风险应用"。会议重点探讨了银行业如何利用代理AI技术解决商业银行信贷风险管理中的具体挑战，特别是针对中小企业(SME)贷款审批流程的优化。

Adrien分享了Cloudera AI平台在AWS上的实际应用案例，展示了如何通过AI代理系统自动化信贷政策合规性检查，从而提高贷款审批效率。会议包含了完整的技术演示，展现了从政策文档解析到信贷申请评估的端到端解决方案。

## 详细时间线与关键要点

### 0:00-2:00 开场介绍与行业背景
- 演讲者自我介绍：Adrien，Cloudera全球AI解决方案总监
- 银行业是生成式AI影响最大的行业
- 根据麦肯锡报告，生成式AI在银行业的潜在价值创造达3400亿美元
- 面临的主要挑战：现有系统集成、人员培训、AI安全防护

### 2:00-4:00 Cloudera平台介绍
- Cloudera作为AI和数据平台的核心能力
- 提供从数据摄取到AI应用的统一平台
- 完全混合云架构，AWS为首选合作伙伴
- 统一数据结构和单一安全层的优势
- 私有AI部署能力，满足金融服务行业合规要求

### 4:00-7:00 代理AI在商业银行的应用价值
- 商业银行流程改进潜力达50%
- 中小企业贷款是主要痛点，运营成本高
- 与个人抵押贷款相比，SME贷款自动化程度低
- 理想状态vs现实挑战：完美信息、完美信任、完美决策的缺失
- 贷款审批流程中的关键阻碍：物理文档、人工政策解读、大量人工干预

### 7:00-10:00 AI驱动的信贷解决方案
- AI驱动的贷款发起：文档审查自动化
- 信贷备忘录与政策匹配的自动化检查
- AI驱动的贷款管理：自动催收、风险审查
- 重点案例：政策合规性检查自动化
- 技术架构：每个政策规则创建对应AI代理

### 10:00-14:00 技术实现与演示
- 技术栈介绍：Cloudera + Neo4j图数据库 + Docling文档处理器
- 实际演示Cloudera AI工作台应用
- 信贷政策上传和代理提取流程
- 知识图谱构建提升规则关系理解
- Galileo合作伙伴提供可观测性支持

### 14:00-17:00 演示结果与实际应用
- 合规性评分系统展示
- 具体规则检查结果：信贷报告年龄限制、银行对账单年龄、首次购房者最低首付等
- 工作流程监控和调试功能
- 成本管理和重新运行机制
- 实际客户案例：某银行使用该系统进行人工审查的双重验证
- 代理AI仍处于初期阶段，需要重复检查机制

### 17:00-17:30 总结与展望
- 强调解决方案的实用性，不仅是理论演示
- 实际银行客户正在使用该系统检测信贷风险审批流程中的潜在审计问题
