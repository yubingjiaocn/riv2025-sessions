# AWS re:Invent 2025 - IND 395 会议总结

## 会议概述

本次会议主题为"使用生成式AI和LLM代理加速游戏设计审查"（IND 395）。两位AWS解决方案架构师Sam Pzer和Christina介绍了如何利用AI代理技术来改进游戏设计文档的审查流程。传统的游戏分析师需要从多个维度评估游戏设计文档，包括游戏背景设定、质量保证、游戏玩法设计和商业策略等方面。这个过程通常耗时较长，且容易受到主观偏见的影响。

演讲者展示了一个创新的解决方案：构建多个专业化的AI代理，每个代理专注于特定领域的评估，然后通过一个协调器代理整合所有反馈。他们使用Amazon Bedrock Agent Core、Strands开源框架以及知识库等AWS服务来实现这一系统。现场演示中，他们以《新世界》（New World）游戏为例，展示了如何评估一个"在游戏中添加精灵种族"的设计提案，系统能够快速识别出该提案与游戏背景设定不符的问题。
