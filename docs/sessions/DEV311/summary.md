# AWS re:Invent 2025 - 构建生产级 Agent Swarm 监控工业 AI

## 会议概述

本次 AWS re:Invent 技术分享由 AWS 开发者 Betty Jung 和 AWS 数据英雄、T8ET 高级 AI 工程师 Trista Pan 共同主讲，主题聚焦于如何构建生产级的 AI Agent 系统。演讲者强调，AI 已经超越了简单的聊天应用，现在的 AI 系统能够像工程师团队一样进行规划、协作和解决复杂问题。

会议深入探讨了 AI Agent 的核心架构、多 Agent 系统的协作模式，以及如何将这些系统安全地部署到生产环境中。演讲者通过实际案例展示了 AI Agent 在客户支持和故障排查等场景中的应用，并详细介绍了 AWS Bedrock Agent、Strand Agent 等工具如何简化 Agent 的开发和部署流程。特别强调了数据质量、安全控制和 MCP（Model Context Protocol）服务器在构建企业级 AI Agent 中的重要性。
