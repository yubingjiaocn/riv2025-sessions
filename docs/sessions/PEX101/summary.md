# AWS re:Invent 2025 分会场总结

## 会议概述

本次分会场由AWS全球合作伙伴渠道和市场推广项目负责人Christine Henley和美国合作伙伴销售负责人Rayhon Shaw共同主持，主题聚焦于如何通过AWS合作伙伴网络加速新客户获取和收入增长。会议面向ISV（独立软件供应商）、SI（系统集成商）、咨询公司和分销商等合作伙伴。

演讲者强调了当前云计算市场的巨大机遇：AWS合作伙伴网络已发展至140,000家合作伙伴，覆盖200多个国家和地区。特别值得关注的是，98%的组织属于中小企业（SMB），其中90%计划在未来三年内采用生成式AI解决方案。AWS预测到2026年，50%的市场交易将通过渠道合作伙伴完成。会议详细介绍了AWS的合作伙伴盈利框架（构建-营销-销售-增长），以及两个重要的新项目：合作伙伴绿地计划（PGP）和小企业加速计划（SBI），这些项目旨在帮助合作伙伴建立可规模化、可重复的业务实践。

演讲者通过多个实际案例展示了合作伙伴如何利用AWS项目创造客户价值，包括北爱尔兰的水质治理项目、全球物流平台优化，以及医疗保健行业的数字化转型。核心信息是：合作伙伴需要从机会驱动型合作转向建立系统化的业务实践，通过专注于特定行业、客户群体和解决方案，与AWS深度整合，实现规模化增长。

## 详细时间线

开场介绍 (00:00-02:30)
- Christine Henley介绍自己作为AWS全球合作伙伴项目负责人的角色
- Rayhon Shaw介绍自己负责美国地区合作伙伴销售和新客户获取
- 会议主题：加速新客户参与和收入增长
- 现场调查：ISV合作伙伴、SI/咨询公司、分销商的参会情况

市场机遇概述 (02:30-05:45)
- AWS合作伙伴网络规模：140,000家合作伙伴，覆盖200多个国家和地区
- 关键数据：98%的组织是中小企业
- 90%的中小企业将在未来三年考虑生成式AI解决方案
- 预测：2026年50%的市场交易将通过渠道合作伙伴完成

客户趋势分析 (05:45-08:30)
- 三大客户趋势：
  1. 行业专注度持续提升 - Gartner预测75%的企业买家将基于深度垂直行业知识和专业能力做出采购决策
  2. 生成式AI成为现实 - 81%的中小企业将在未来一年投资AI（Tech Aisle数据）
  3. 多云战略扩展 - 不仅限于大企业，中小企业也在考虑多云解决方案

客户成功案例分享 (08:30-12:00)
- **案例1：北爱尔兰Loch Neagh水质治理**
  - 问题：有毒藻类威胁40%的饮用水供应
  - 解决方案：Ingram Micro、LG Sonic和Whole School Software合作开发AI驱动的预测分析解决方案
 
- **案例2：全球物流平台**
  - 问题：小企业在实时追踪和准确预测方面处于劣势
  - 解决方案：JDS和Woodmark Consulting利用SMB能力认证、生成式AI快速资金和MAP项目，开发物流智能平台
 
- **案例3：医疗保健行业**
  - 问题：50%的医护人员因职业倦怠计划转行
  - 解决方案：Care.com与Cloudhesive和Guide House合作，利用Amazon Connect和Epic EHR系统集成，创建符合HIPAA的AI驱动联络中心，服务超过4500万家庭

合作伙伴盈利框架介绍 (12:00-15:30)
- 四个阶段：
  1. **构建（Build）**：创建解决实际客户问题的能力和解决方案
  2. **营销（Market）**：生成需求并识别客户契合度
  3. **销售（Sell）**：将客户转化为实际机会和业务
  4. **增长（Grow）**：帮助客户成功并长期扩展

合作伙伴成熟度旅程 (15:30-18:00)
- **第一阶段：机会驱动型**
  - 逐个机会与AWS合作
  - 资源密集型，反应式工作模式
 
- **第二阶段：意向性合作**
  - 有意识地选择重点行业、客户群体和AWS团队
  - 创建可重复的合作模式
 
- **第三阶段：业务实践**
  - 建立系统化、可重复的方法
  - 整合AI和自动化
  - 明确的指标、专职团队和长期客户参与飞轮

新项目发布 (18:00-21:00)
- **合作伙伴绿地计划（PGP）**
  - 本周刚刚发布
  - 提供多年期投资支持新客户获取
  - 面向中型市场和大型企业客户
 
- **小企业加速计划（SBI）扩展**
  - 去年在re:Invent首次亮相
  - 今年已接触超过30,000家客户
  - 明年将全球扩展

两大项目对比 (21:00-23:30)
- 共同点：多年期投资、支持、培训、打包的市场推广解决方案、营销材料、市场研究
- 差异：
  - PGP：针对中型和大型企业，与AWS销售团队共同销售
  - SBI：针对小企业，需要不同的规模化市场推广方法，合作伙伴主导，与合作伙伴区域经理合作

构建阶段深入讲解 (23:30-30:00)
- 准备清单：
  - 明确定义的范围和目标客户
  - 完善的解决方案和能力
  - 良好的团队培训模型和执行能力
 
- **如何最大化AWS项目收益**
  - 从解决方案和目标客户反向规划
  - 识别相关AWS项目进行组合
  - 示例：AWS业务成果加速器（BOA）+ 市场开发基金（MDF）+ SMB能力认证

SMB能力认证详解 (30:00-34:00)
- 三大核心收益：
  1. **市场推广支持**：市场研究、行业洞察、客户趋势、联合营销活动
  2. **SMB解决方案社区**：与AWS和其他合作伙伴交流、技术建议、预打包入门套件
  3. **专业验证**：徽章展示、AWS网站优先展示、本地活动、营销活动特色展示
 
- 成效：SMB能力认证合作伙伴的SMB业务同比增长率比类似未认证合作伙伴高35个百分点

2026年SMB能力认证新增福利 (34:00-36:00)
- 市场推广能力自评工具
- 更多构建研讨会和圆桌会议
- 额外的市场研究和市场开发资金
- 共同销售研讨会
- Partner Equip本地活动中的SMB专场

构建阶段工具和资源 (36:00-38:30)
- **Skill Builder**：定制培训、不同角色学习路径（营销、销售、技术）、新增托管服务培训
- **合作伙伴分析仪表板**：查看团队培训和认证进度
- **AWS Marketplace和AI Agent Marketplace**：查找可消费的解决方案和预构建AI代理，本周发布三个新代理（包括迁移代理）

构建阶段行动要点 (38:30-40:00)
- 申请创新沙盒积分（Innovation Sandbox Credits）
- 审查更新的能力认证要求（行业能力认证、增强的AI能力认证）
- 安排解决方案架构师审查或SMB能力认证研讨会

营销阶段介绍 (40:00-43:30)
- Rayhon强调营销阶段的重要性
- **基础要素**：
  - 利用基础项目、能力认证和证书
  - 解锁特殊资金以提供客户激励
  - 重点项目：SBI和PGP
 
- **合作伙伴工具和资源**：
  1. **合作伙伴营销中心（Partner Marketing Central）**：AWS预制的联合品牌营销材料
  2. **市场开发基金（MDF）**：根据细分市场和重点构建市场推广策略
  3. 指定专人掌握营销工具的使用时机

营销阶段合作伙伴案例 (43:30-46:30)
- **Innovative公司三步法**：
  1. 在客户所在地与客户见面（参加行业研讨会、热门活动、本地活动）
  2. 快速解决客户问题 - 开发了基于Amazon Bedrock的Derosol IQ工具，将工作说明书（SOW）生成时间从2周缩短至2小时（减少90%销售周期）
  3. 利用8项AWS能力认证加速业务

营销阶段行动要点 (46:30-48:00)
- 评估现有营销计划并充分利用AWS营销工具
- 利用Partner Central了解适用于特定机会的资金项目
- 持续评估和迭代营销计划（特别是在生成式AI时代）
- 识别、构建和执行支持联合营销机会的活动

销售阶段 - 云销售中心介绍 (48:00-52:00)
- **云销售中心（Cloud Sales Center, CSC）**是AWS的重大投资
- 两大支柱：
  1. 绿地客户（新接触云的客户）
  2. 小企业客户
 
- **销售细分**：企业级、中型、小型（企业和中型有行业专注）
 
- **三种角色类型**：
  1. **需求生成团队**：执行需求生成计划，致电客户，安排会议，接触未接触的客户
  2. **云销售代表**：内部客户经理，与合作伙伴共同销售
  3. **合作伙伴区域经理（PTM）**：SBI计划的一部分，与合作伙伴共同制定区域级计划

与云销售中心合作的三大要点 (52:00-54:30)
1. MAP（迁移援助加速计划）：了解如何在机会层面应用，与CSC紧密合作
2. 生成式AI资金：快速资金或生成式AI创新中心，如何与框架对齐以扩展POC
3. 合作伙伴绿地计划（PGP）：已上线，可立即注册

PGP与SBI对比 (54:30-57:00)
- **相似点**：核心目标都是帮助合作伙伴建立盈利和可扩展的业务
- **PGP（左侧）**：
  - 适合中型和大型企业
  - 与AWS现场销售或云销售中心团队共同销售
  - 在销售层面与AWS销售人员合作
 
- **SBI（右侧）**：
  - 合作伙伴主导
  - 与合作伙伴区域经理合作
  - 拥有销售区域，共同规划、使用工具、联合营销、共同举办活动

销售阶段工具和资源 (57:00-60:00)
- **ACE（AWS合作伙伴CRM连接器）**：
  - 被称为"圣杯"
  - 如果不使用ACE，AWS无法看到机会
  - 必须配置和激活
 
- **合作伙伴分析仪表板**：
  - 查看整体业务
  - 识别停滞的机会
  - 查看管道、AWS发送的线索、需要清理的数据
  - 建议每日查看
 
- **AWS Marketplace**：
  - 成为Marketplace交易的专家和倡导者
  - 适用于所有客户规模和行业

销售阶段合作伙伴案例 (60:00-63:00)
- **Predictive公司 - 阿拉斯加案例**：
  - 与阿拉斯加大学小企业发展中心合作，启动竞赛推介系列
  - 帮助当地企业家通过AI实现业务现代化
  - 获胜者Arsenetto：该地区首个AI初创公司，帮助部落社区通过共享社区日历查找资源和活动
  - 每位获胜者获得25,000-50,000美元的AWS专业服务积分，加上Predictive的直接指导
  - 该模式已扩展至全国，体现了共同销售和共同增长

销售阶段行动要点 (63:00-65:00)
1. 确定与CSC合作的方向（SBI、PGP、小企业、中型或大型企业）
2. 与相应团队建立联系和关系，开始共同销售
3. 确保ACE配置正确并激活
4. 将目标受众、客户细分、行业与CSC特定细分对齐
5. 审查Partner Central，确保为每个机会利用最佳资金项目

  - 更长期的合作
  - 更高级的工作负载
