# AWS re:Invent 2025 合作伙伴业务增长会议总结

## 会议概述

本次会议由AWS全球合作伙伴项目负责人Christine Henley和美国合作伙伴销售负责人Rayhan Shaw主讲，重点讨论如何通过AWS合作伙伴网络加速新客户参与和收入增长。会议面向ISV合作伙伴、系统集成商和咨询公司，介绍了AWS合作伙伴盈利框架的四个阶段：构建、营销、销售和增长。

演讲者强调了规模化市场进入的重要性，特别是针对占98%的中小企业市场。会议详细介绍了两个新的合作伙伴项目：合作伙伴绿地项目(PGP)和小企业加速倡议(SBAI)，这些项目提供多年投资支持，帮助合作伙伴建立可重复的业务实践。通过具体的合作伙伴成功案例，展示了如何利用AWS项目和工具实现业务增长和客户成功。

## 详细时间线与关键要点

### 0:00-5:00 开场介绍
- Christine Henley介绍自己作为AWS全球合作伙伴项目负责人
- Rayhan Shaw介绍自己作为美国合作伙伴销售负责人
- 会议主题：加速新客户参与和收入增长
- 现场调研：ISV合作伙伴、SI/咨询公司和分销商的参与情况

### 5:00-10:00 市场机遇分析
- AWS合作伙伴网络已发展至1400个合作伙伴，覆盖200多个国家和地区
- 98%的组织是中小企业，这是巨大的市场机遇
- 90%的中小企业将在未来3年考虑生成式AI解决方案
- 预测2026年50%的市场交易将通过渠道合作伙伴完成

### 10:00-15:00 客户趋势分析
- 行业专业化持续成为趋势重点
- Gartner预测75%的企业买家将基于深度垂直知识和专业技能进行采购
- 81%的中小企业和中端市场公司将在来年投资AI技术
- 多云解决方案需求扩展至中小企业

### 15:00-20:00 客户成功案例
- 北爱尔兰Loch Neagh有毒藻类治理项目：Ingram Micro、LG Sonic等合作伙伴联合创建AI预测分析解决方案
- JDS Woodmark咨询公司：为全球海事物流组织实施物流智能平台
- 医疗保健行业：Care.com与Cloudhesive和Guidehouse合作，创建HIPAA合规的AI驱动联系中心

### 20:00-25:00 合作伙伴盈利框架介绍
- 四个阶段：构建(Build)、营销(Market)、销售(Sell)、增长(Grow)
- 合作伙伴成熟度三个阶段：机会阶段、意向阶段、业务实践阶段
- 新推出的合作伙伴绿地项目(PGP)和扩展的小企业加速倡议(SBAI)

### 25:00-30:00 构建阶段详解
- 明确定义范围和目标客户的重要性
- 从解决方案和客户需求出发，反向选择AWS项目
- SMB能力认证项目的三大优势：市场进入支持、解决方案社区、专业技能验证
- SMB能力认证合作伙伴的中小企业业务同比增长35%

### 30:00-35:00 构建阶段工具和资源
- SkillBuilder平台提供定制培训和学习路径
- AWS Marketplace和AI代理市场提供现成解决方案
- 合作伙伴分析仪表板跟踪团队培训认证情况
- 2026年SMB能力认证新增功能：市场进入能力评估、更多构建研讨会

### 35:00-40:00 营销阶段策略
- 利用基础项目和能力认证获得特殊资金支持
- 合作伙伴营销中心提供共同品牌营销材料
- 营销开发资金(MDF)支持市场进入活动
- Innovative公司案例：通过Amazon Bedrock构建Dursol IQ工具，将销售周期从2周缩短至2小时

### 40:00-45:00 销售和增长阶段
- 云销售中心(CSC)的三种角色：需求生成团队、云销售代表、合作伙伴区域经理
- PGP适用于中大型企业，SBAI专注于小企业市场
- ACE管道管理的重要性
- 合作伙伴主导支持项目：客户保留率提高29%
- SoftChoice公司案例：年增长率超过50%

### 45:00-46:00 总结和行动项目
- 三个关键行动项目：从四个支柱中各选择一个项目、评估合作伙伴成熟度、与团队分享框架
- 强调意向性和建立业务实践的重要性
