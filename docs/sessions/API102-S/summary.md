# AWS re:Invent 2025 - Agentic AI 领导者经验分享

## 会议概述

本次分享会由IBM企业战略负责人Roger Primo主持,重点探讨了Agentic AI(智能体AI)在企业中的实际应用和最佳实践。Roger强调,尽管MIT研究显示只有5%的企业从AI中获得了真正价值,但Agentic AI正在成为比以往任何技术更具颠覆性的力量。会议通过IBM自身的转型案例以及Claims Connection Group和Apollo Tires等客户的实际经验,展示了如何通过"提升底线"(优化现有流程)和"突破天花板"(创造新业务模式)两种方式获得价值。

IBM通过Gen AI转型,预计到2025年底将实现每年45亿美元的业务改善。会议强调了几个关键成功要素:从高层到员工的愿景统一、关注端到端流程而非孤立试点、投资数据质量、建立技术基础设施,以及采用新的"智能体开发生命周期"(ADLC)方法。Mark Murphy分享了他如何在18个月内将估算流程从每天8个提升到244个,每天节省14,000美元。核心建议是:找到业务中的"痛点"(money suck),从那里开始,而不是追求过于复杂的用例。

## 核心要点总结

关键建议:
1. 找到业务中的"资金消耗点"(money suck)并从那里开始
2. 专注于端到端流程而非孤立试点
3. 投资数据质量和技术基础设施
4. 采用智能体开发生命周期(ADLC)方法
5. 如果还没开始,现在就行动 - "你已经晚了两年"

成功模式:
- 客户服务、后台办公、代码开发是最成熟的应用领域
- 从"提升底线"开始,逐步走向"突破天花板"
- 将生产力收益再投资于创新,形成复合效应
