# AWS re:Invent 2025 - Autodesk Assistant 会议总结

## 会议概览

本次会议由 Autodesk 软件架构师 Matthew Liam 和 AWS 解决方案架构师 Sahil Sahini 共同主讲,重点介绍了 Autodesk Assistant 如何在 AWS Bedrock 之上构建 Agentic AI 平台。Autodesk 作为全球领先的 3D 设计、工程和娱乐软件提供商,拥有超过 40 年的行业经验和 15,000 名全球员工,其客户涵盖建筑、制造、媒体娱乐等多个领域。

面对客户在供应链延迟、劳动力短缺、项目复杂度提升等方面的挑战,Autodesk 开发了 Autodesk Assistant 这一 Agent AI 合作伙伴。该助手能够自动化手动任务、连接断开的工作流程,并提供实时洞察,从根本上改变了设计和制造领域的工作方式。通过将"点击操作"模式转变为"描述即执行"的新时代,Autodesk Assistant 大幅缩短了从创意到实现的时间,让客户能够专注于创新而非日常繁琐工作。

在技术架构方面,Autodesk 采用了平台化战略,构建了包括 MCP 服务器、多种 AI 代理、统一后端服务的完整生态系统。通过利用 AWS Agent Core Runtime、Strands 框架以及 Agent Core Gateway 等服务,Autodesk 实现了安全、可扩展、可观测的生产级 Agentic AI 平台,为不同产品线和团队提供了统一的开发和部署基础设施。
