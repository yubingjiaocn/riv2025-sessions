# AWS re:Invent 2025 QuickSight 嵌入式分析会议总结

## 会议概述

本次会议由 AWS QuickSight 产品负责人 Shekhar Kopuri 和 Jose Banos 主讲，并邀请了 Moody's 保险解决方案的产品管理团队成员 Kirtan Dave 分享客户实践案例。会议重点介绍了 Amazon QuickSight 的嵌入式分析能力，以及新推出的 QuickSuite AI 原生代理平台。

演讲者们详细阐述了如何通过嵌入式模式从 QuickSight 中提取价值，包括可视化嵌入、完整页面分析、自助服务创作等多种应用场景。Moody's 的成功案例展示了如何将 QuickSight 集成到风险数据湖产品中，实现多租户架构下的安全数据访问和分析能力。会议还介绍了 QuickSuite 平台的最新 AI 功能，包括对话式分析、自动化工作流程等创新特性。

## 详细时间线与关键要点

### 0:00-10:00 会议开场与 QuickSight 产品介绍
- **产品定位**：QuickSight 作为 QuickSuite 内的统一 BI 体验平台
- **核心能力**：支持交互式仪表板、像素完美报告、对话式分析
- **技术优势**：云原生无服务器架构，集成 Highcharts 可视化库
- **经济效益**：独立第三方调研显示，3年内成本降低67%，用户参与度提升300%

### 10:00-20:00 嵌入式分析应用场景
- **企业应用**：HR工具、营销工具、销售工具中的原生BI能力部署
- **ISV增值**：独立软件供应商通过嵌入分析提升客户满意度和留存率
- **规模化部署**：支持从10个用户到数十万用户的弹性扩展
- **多种嵌入模式**：可视化嵌入、完整页面分析、邮件报告、自助创作

### 20:00-30:00 企业级特性与架构灵活性
- **安全治理**：行级安全(RLS)、列级安全、身份传播、细粒度访问控制
- **多租户支持**：命名空间隔离、基于标签的RLS/CLS规则、独立账户栈
- **身份集成**：AWS IAM Identity Center、第三方IDP(Okta、Ping)、Active Directory
- **API与SDK**：强大的嵌入SDK、一键嵌入、双向交互、自定义品牌

### 30:00-40:00 NFL 成功案例分析
- **NextGen Stats**：近实时球员和比赛数据推送给球迷
- **Digital Athlete**：提升球员安全的产品
- **NFL Combine**：实时球员速度和力量数据展示
- **Draft IQ Assistant**：首个对话式聊天助手
- **规模成就**：吸引超过100万用户访问

### 40:00-50:00 Moody's 风险数据湖实施案例
- **业务背景**：为财产意外险行业提供灾难风险量化分析解决方案
- **产品架构**：基于智能风险平台构建的应用数据湖
- **多租户实现**：通过命名空间实现数百个客户的独立部署
- **安全控制**：集成平台身份，实现行级安全的细粒度访问控制
- **嵌入体验**：设计器和消费者体验完全嵌入应用内部

### 50:00-55:00 QuickSuite AI 平台与未来展望
- **平台演进**：从传统BI向AI辅助BI转变
- **核心能力**：聊天代理、研究功能、QuickSight流程、自动化
- **数据整合**：结构化和非结构化数据统一处理
- **第三方集成**：Outlook、Slack、浏览器插件等多平台支持
- **行动导向**：从仪表板直接触发第三方应用操作
