# AWS re:Invent 2025 会议总结：AI代理的编排力量

## 会议概述

本次技术分享由Mortado公司的Rahul Dareja主讲，主题为"如何让AI代理从承诺转化为实际影响力"。演讲重点探讨了AI代理在企业应用中的局限性，以及编排(Orchestration)作为关键技术如何赋能AI代理真正与企业后端系统交互并执行实际业务操作。

演讲者强调，虽然AI技术已经广泛应用，但大多数AI代理仅限于前端任务如邮件撰写和信息查找，无法真正操作企业核心应用系统如Salesforce、SAP等。通过引入编排层，AI代理可以获得访问企业数据、执行业务流程、实现角色管理和建立持续生命周期的能力，从而实现从辅助工具到业务执行者的转变。
