# AWS re:Invent 2025 技术会议总结

## 会议概述

本次会议主题为"通过自主企业运营改变商业未来"，由Digitate公司代表Rajiv和AWS全球战略合作伙伴销售团队负责人Todd Carey共同主讲。会议重点介绍了Digitate的Ignio平台如何通过AI驱动的自主运营解决企业IT问题，以及与AWS的战略合作关系。

演讲者通过三个真实客户案例展示了IT故障如何直接影响业务运营，包括奢侈品零售商Tapestry的订单处理问题、医疗技术公司的假肢交付承诺，以及消费品公司的直接门店配送挑战。会议还宣布了与BMC Control-M在AWS Marketplace上的首次多产品SaaS合作发布。

## 详细时间线与关键要点

### 0:00-2:00 开场介绍
- 感谢现场Kenvue客户副总裁Pravin的参与
- 宣布与Control-M在AWS Marketplace的合作发布
- 介绍会议结构：三个客户案例 + 产品介绍 + AWS合作

### 2:00-5:00 客户案例一：Tapestry奢侈品零售
- **问题**：37个全球店面，订单未能及时处理、履行和交付
- **影响**：收入损失和客户体验下降
- **解决方案**：部署统一可观测性，覆盖从订单到现金的全流程
- **成果**：上周处理超过100万订单，避免了价值300万美元的订单损失

### 5:00-8:00 客户案例二：医疗技术公司假肢制造
- **背景**：通过收购成长的公司，制造假肢产品
- **挑战**：70个不同接口系统，需要向医院确认假肢交付承诺日期
- **解决方案**：整合SaaS、本地部署和遗留应用系统的数据流
- **成果**：需求规划流程从数周缩短至数小时

### 8:00-10:00 客户案例三：消费品公司直接门店配送
- **场景**：板球比赛期间零售商备货需求
- **问题**：订单处理故障导致库存无法及时补充
- **影响**：零售商收入损失、品牌份额下降
- **解决方案**：监控从订单接收到制造、配送、上架的全流程
- **规模**：每周处理数亿美元订单

### 10:00-13:00 Ignio平台技术架构
- **三大核心能力**：
  - 统一可观测性：水平和垂直技术栈监控
  - AI驱动洞察：预测性分析和故障预警
  - 闭环自动化：机器速度问题隔离和修复
- **五个智能代理**：感知、推理、行动、学习、控制和LLM模型代理
- **覆盖范围**：云运营、数据中心、定制应用、SAP系统、批处理作业和终端设备

### 13:00-15:00 平台数据和AWS合作
- **运营规模**：
  - 管理34,200个全球工作负载，覆盖20个地区
  - 处理12亿事件，87%噪音减少
  - 每年执行3亿次自动化操作
- **合规标准**：SOC 2 Type 2等企业级标准
- Todd Carey介绍AWS与Digitate的战略合作关系
- 强调AWS Marketplace作为下一代销售平台的重要性

### 15:00-16:30 问答环节和总结
- 开放现场问答环节
