# AWS re:Invent 2025 - 智能体工厂(Agent Factory)会议总结

## 会议概述

本次技术会议由AWS高级解决方案架构师Girish和Justin主讲,重点介绍了如何构建一个"智能体工厂"(Agent Factory)系统。这个系统能够根据用户需求自动创建和部署AI智能体或多智能体系统。会议展示了如何利用开源Strands SDK、模型上下文协议(MCP)、智能体间通信协议(A2A)以及AWS Agent Core服务,将智能体从原型开发环境快速部署到生产环境。

演讲者通过实时编码演示,展示了从创建简单聊天机器人到构建复杂多智能体系统的完整过程。核心理念是利用"构建者智能体"(Builder Agent)作为智能体工厂的核心,它可以接收自然语言请求,自动生成代码、创建Lambda函数、配置网关,并将智能体部署到Agent Core运行时环境。整个过程大大简化了智能体开发和部署的复杂度,使开发者能够专注于业务逻辑而非基础设施管理。

会议最后通过实际案例演示了如何快速创建风险分析智能体和费用管理智能体,展现了智能体工厂在实际业务场景中的应用价值。
