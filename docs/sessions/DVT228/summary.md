# AWS re:Invent 2025 - Kiro Powers 技术会议总结

## 会议概述

本次技术会议介绍了AWS Kiro平台新推出的"Powers"功能，这是一个旨在解决AI代理在专业化任务中面临挑战的创新解决方案。会议由Kiro团队主导，并邀请了Postman的CTO Harry Mauer和Superbase的开发工程师Tyler Schuckert作为合作伙伴分享实际应用案例。

Powers功能通过将MCP（模型上下文提供者）和引导指令相结合，为AI代理提供可扩展的专业化能力，同时避免了上下文过载的问题。该功能采用渐进式披露的概念，让代理只在需要时获取相关的专业知识和工具，从而提高开发效率和代码质量。会议展示了与Figma、Postman、Superbase等主流开发工具的集成案例，演示了从UI设计到API开发再到后端数据库管理的完整开发流程。
