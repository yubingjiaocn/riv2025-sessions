# AWS re:Invent 2025 - 扩展可信AI采用：Salesforce与AWS合作案例

## 会议概述

本次技术分享会聚焦于如何通过Salesforce和AWS的合作伙伴关系实现可信AI的规模化部署。主讲人包括Salesforce的Shibani Ahuja、AWS首席解决方案架构师Hrushi、德勤咨询总监Pravin，以及CrowdStrike应用市场总监Amit Bangad。会议深入探讨了企业在AI转型过程中面临的挑战，特别是从生成式AI向代理式AI（Agentic AI）的演进。

演讲者分享了当前企业CIO在AI实施中的主要趋势，包括对不同AI架构类型的理解差异、构建与购买决策的演变，以及对SaaS提供商与云服务提供商合作模式的重新认知。CrowdStrike作为成功案例，展示了如何成为一个代理式企业，通过Salesforce Agentforce和AWS的集成解决方案实现AI治理和规模化部署。
