# AWS re:Invent 2025 云安全检测与响应会议总结

## 会议概述

本次会议由CrowdStrike首席产品经理Vinayak Shastri主讲，主题为"重新构想云检测和响应与智能体验"。会议深入探讨了当前云安全威胁的最新趋势，特别是Scattered Spider和Genesis Panda等高级威胁行为者如何利用云基础设施构建完整的威胁生态系统。

演讲者重点分析了现代攻击者如何利用AI技术提升攻击效率，将传统需要数小时或数天的侦察工作缩短至分钟甚至秒级。面对中位数入侵时间已降至48分钟、最快入侵时间仅51秒的严峻现实，传统的云安全检测和响应方案已无法满足防护需求。CrowdStrike通过重新架构其Falcon云安全平台，实现了实时检测、统一关联分析和自动化响应的完整解决方案。

## 详细时间线与关键要点

### 0:00-5:00 开场与威胁趋势分析
- 介绍演讲者Vinayak Shastri及会议议程
- **Scattered Spider威胁分析**：2023年占云威胁总数的1/3，2024年降至13%，但这并非因为活动减少，而是其他成熟威胁行为者的崛起
- **Genesis Panda攻击演进**：从凭证窃取发展到建立SSH密钥持久化访问，完全绕过端点检测

### 5:00-10:00 AI增强威胁与防护挑战
- **AI对攻击者的增强作用**：
  - 侦察效率提升：从小时级缩短至分钟或秒级
  - 漏洞利用自动化：通过LLM快速生成攻击脚本
  - 降低攻击门槛：技术支持和工具生成更加便捷
- **关键统计数据**：中位数入侵时间48分钟，最快入侵时间51秒
- **云环境防护挑战**：缺乏内联传感器、调查复杂且孤立、响应机制有限

### 10:00-15:00 CrowdStrike解决方案架构
- **重新构想的三大核心**：
  - 实时可见性和检测：流式日志处理，检测时间从分钟级降至秒级
  - 关联和上下文化：统一身份、云检测和传感器检测，消除孤岛效应
  - 加速响应：集成Charlotte AI进行自动分类，结合Falcon Fusion SOAR实现实时响应
- **新检测引擎特性**：每日处理5万亿事件，经过OverWatch和MDR团队十年验证

### 15:00-20:00 实时演示
- **攻击场景模拟**：展示Scattered Spider典型攻击手法
  - 通过社会工程获取AWS凭证
  - 使用云shell创建SSH密钥对
  - 建立具有管理权限的IAM角色
  - 启动附加特权角色的EC2实例
- **防护响应展示**：
  - 实时图形化攻击视图
  - 基于异常行为的即时检测
  - 自动化Fusion工作流阻断威胁
  - 详细时间线分析和取证保护

### 20:00-24:00 智能体验与平台整合
- **智能AI代理功能**：
  - 提升SOC团队生产力，节省数周分析时间
  - 减少平均响应时间(MTTR)
  - 解决技能短缺问题，提供预训练的安全专家代理
- **统一安全平台架构**：
  - 单一传感器覆盖Linux、Windows、Mac、Kubernetes和容器
  - 企业级风险关联图谱
  - 跨端点、云、SaaS和身份的统一检测
  - 全方位智能体AI支持，实现从检测到修复的完整闭环
