# AWS re:Invent 2025 零售边缘计算创新会议总结

## 会议概述

本次会议探讨了Yum Brands、AWS和Spectral Cloud三方合作如何重新定义零售边缘计算架构。会议由Spectral Cloud高级副总裁Kyle Goodwin主持，与会嘉宾包括Yum Brands全球可靠性工程副总裁Shiv Arihapan、可靠性和平台工程总监Ryan Good，以及AWS全球实体零售负责人Justin Swaggler。

会议重点讨论了Yum Brands作为全球最大餐饮连锁企业，如何通过云边协同架构在全球40,000多个门店中实现数字化转型。三方合作通过Amazon EKS混合节点和Kubernetes统一管理平台，成功构建了从云端到边缘的无缝技术生态系统，不仅提升了运营效率和客户体验，还为AI和机器学习工作负载的部署奠定了坚实基础。

### 0:00-2:30 开场介绍
