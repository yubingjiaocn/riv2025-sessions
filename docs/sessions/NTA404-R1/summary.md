# NTA404: AI Code to Production: Context Matters

## 会议概述 (Session Overview)

本次AWS re:Invent 2025分会场由解决方案架构师Pranjit和Marcus Mattson主讲，主题为"AI Code to Production: Context Matters"。会议重点探讨了AI编程助手在生产环境中面临的挑战，以及如何通过上下文工程来构建生产就绪的代码。

演讲者通过开发者Sarah的故事，展示了从"氛围编程"(vibe coding)到"上下文架构师"(context architect)的转变过程。会议深入介绍了AWS的代理IDE工具Kiro，演示了如何利用规范驱动开发、上下文分层和最佳实践来构建可直接部署到生产环境的应用程序。整个会议以实际编码演示为主，展现了从原型到生产的完整开发流程。
