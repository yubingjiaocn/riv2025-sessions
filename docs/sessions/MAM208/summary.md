# AWS Transform：将自定义代理和工具引入云迁移服务

## 会议概述

本次闪电演讲由AWS的Tushar Sakenna主讲，重点介绍了AWS Transform这一首个用于迁移和现代化的代理式AI服务。演讲者通过GPS导航系统的类比，说明了云迁移工具的演进历程——从早期需要手动规划路线的纸质地图，发展到现在能够自动导航、实时调整路径的智能助手。AWS Transform正是这样一个"迁移GPS"，通过AI代理团队为客户提供从发现到重新托管的端到端迁移指导。

演讲的核心亮点是昨天刚刚发布的"可组合性"(Composability)功能，这一创新特性允许合作伙伴和客户将自己的代理和工具集成到AWS Transform中，创建定制化的迁移工作流。通过代理注册表和自定义编排器，用户可以将AWS的最佳实践代理与自己的专有工具相结合，获得统一的用户体验和协作平台。演讲者还展示了AWS专业服务团队如何利用这一功能构建包含交付代理、云迁移工厂工具和容器化代理的端到端定制工作流。
