# AWS re:Invent 2025 技术会议总结

## 会议概述

本次会议由SolarWinds的Pete Di Stefano和Paciolan的Ryan Lauer共同主讲，重点分享了Paciolan公司的云迁移之旅和运营韧性实践。Paciolan是一家专注于现场活动票务的公司，主要服务于大学体育赛事领域，为客户提供票务购买和入场扫描服务。

会议探讨了现代IT环境中面临的挑战，特别是在混合云架构下如何实现运营韧性。通过Paciolan的实际案例，展示了如何利用SolarWinds的全栈可观测性平台和AWS云服务来应对流量高峰、快速扩展和实时监控等关键业务需求。会议强调了在复杂环境中快速响应和持续服务交付的重要性。

## 详细时间线与关键要点

### 0:00-2:00 会议开场与议程介绍
- Pete Di Stefano介绍会议议程，包括行业挑战、人工智能、Paciolan概述等主题
- 幽默提及AI话题的普遍性，为后续技术讨论做铺垫
- 强调将重点关注Paciolan公司的云迁移实践经验

### 2:00-4:00 行业挑战与运营韧性概念
- 讨论IT行业面临的共同挑战：故障排除、根因分析、可见性缺口等
- 引入"运营韧性"概念：预先规划应对系统故障，确保业务连续性
- 强调时间因素在服务交付中的关键作用，特别是在复杂的混合环境中

### 4:00-6:00 Paciolan公司介绍与业务特点
- Ryan Lauer介绍Paciolan的核心业务：现场活动票务，主要服务大学体育赛事
- 说明业务特点：流量高峰集中在票务开售和比赛日入场扫描时段
- 强调在关键时期（如碗赛选择期）保持系统稳定运行的重要性

### 6:00-8:00 IT架构与历史挑战
- 描述Paciolan的混合IT环境：包含EKS和本地Kubernetes集群，结合AWS和本地架构
- 分享容器化和微服务架构迁移过程中遇到的挑战
- 介绍SolarWinds APM工具在服务交互分析和网络延迟监控中的应用价值

### 8:00-10:00 监控策略与实时运营
- 详细说明多层面监控需求：基础设施、应用性能和AWS资源的统一监控
- 介绍针对不同团队（SRE、开发、客户服务）的定制化仪表板策略
- Ryan分享当天早晨监控重要票务开售活动的实际经验

### 10:00-12:00 AWS云服务优势与SolarWinds平台展示
- 总结8年AWS使用经验，强调云服务在快速扩展方面的优势
- 介绍SolarWinds全栈可观测性平台的完整功能：监控、事件响应、服务管理
- 展示平台涵盖的监控范围：网络、基础设施、应用、用户体验、数据库和安全
- 提及机器学习、生成式AI和代理AI等新技术特性
