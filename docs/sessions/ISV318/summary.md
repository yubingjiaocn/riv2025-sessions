# AWS re:Invent 2025 Reltio 分会场总结

## 会议概述

本次分会场由AWS ISV解决方案架构师团队负责人Kristen Hughes主持,Reltio首席产品官Ankur进行了主题演讲。Reltio是企业主数据管理(MDM)领域的革新者,也是首个将代理式AI(Agentic AI)与可信数据基础无缝连接的云端MDM平台。

Ankur详细介绍了Reltio的核心价值主张——数据统一(Data Unification)。该平台通过整合来自不同数据源的记录,选择最高保真度的数据属性,创建出比输入数据更可信的"黄金记录"。这种能力已经演进为构建"可信档案"(Trusted Profiles),为客户、供应商、产品等业务实体建立丰富的实时数据档案。Reltio的知识图谱作为企业运营的核心上下文层,通过API和事件机制提供近实时的数据访问能力。

演讲的核心聚焦于Reltio如何利用AWS的Agent Core和Bedrock服务推出新产品Agent Flow,通过代理式AI解决数据管理中仍需大量人工干预的痛点,包括实体解析、数据质量管理和复杂数据模型处理等。Ankur通过CarMax和华纳兄弟两个客户案例以及两个现场演示,展示了Agent Flow如何让数据管理变得更快、更便宜且更优质。

- 知识图谱作为企业运营的核心上下文层
- 通过API和事件提供近实时数据访问
- 创建可信的、受治理的核心业务数据层

09:00 - 11:30 - 客户案例:CarMax

- 目标:让数据管理效率提升1-2个数量级
- 更便宜、更快速、可证明更优质
- 与AWS紧密合作开发的新产品

- AI代理自动推理、搜索和提出解决方案
- 从互联网获取数据进行数据增强
- 针对Solar Turbines公司的重复数据解析
- 提供自动化建议并展示推理过程
