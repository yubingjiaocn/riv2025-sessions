# AWS re:Invent 2025 技术会议总结

## 会议概述

本次会议由Pearson的客户合作伙伴John Rogers主持，与ServiceNow AI数据分析高级销售总监Ian Gott以及Pearson首席营收官兼高级副总裁Ray Murray共同探讨了代理AI对未来工作的重新定义。会议基于Pearson与ServiceNow合作开展的劳动力技能预测报告，该报告运用机器学习、经济分析和预测分析技术，评估了代理AI及相关技术对任务、角色、行业、国家乃至宏观经济的影响。

会议的核心观点是代理AI不仅仅是IT升级，而是对全球每个经济体前所未有的巨大变革。到2030年，仅在美国就有约800万人的工作角色将受到冲击，但这种冲击将通过技能增强和角色重新定义的方式发生。研究显示，在所调研的10个国家中，有8个国家将出现工作岗位多于工作者的情况，这对技术人才来说是积极的就业保障信号。

## 详细时间线与关键要点

### 0:00-3:00 开场介绍与研究背景
- John Rogers介绍自己作为Pearson客户合作伙伴的身份
- 介绍经过6个月研究完成的劳动力技能预测报告
- 报告采用机器学习、经济分析和预测分析方法
- 引入两位合作伙伴：ServiceNow的Ian Gott和Pearson的Ray Murray

### 3:00-8:00 宏观经济影响分析
- Ray Murray强调代理AI是巨大的地震式变革，而非简单的IT升级
- 到2030年美国将有800万人的工作角色受到冲击
- AI冲击速度很快，但对技能的需求增长更快
- 10个研究国家中有8个将出现工作多于工作者的情况
- AI实际上在推动而非减少对技术人才的需求

### 8:00-11:00 行业特定影响
- Ian Gott分析不同行业的影响程度
- 金融服务业受影响较大，日本可能有15%的劳动力受影响
- 零售业在印度和英国将看到不成比例的影响
- 专业服务和技术行业更多体现在AI增强工作角色方面
- 新加坡的知识工作者在AI应用方面较为先进

### 11:00-15:00 组织和个人应对建议
- Ray Murray提出"学习是终身力量倍增器"的概念
- 从"学习以赚钱"转向"学习以学习"的模式转变
- 学习必须嵌入到工作流程中，而非边缘化
- Ian Gott强调拥抱AI、与AI合作，保持人类创造力和复杂思维
- 强调治理责任，每个人都是数据治理的守护者

### 15:00-18:30 实际应用与总结
- Pearson内部AI应用经验分享：18个月来AI已成为公司DNA
- 每位员工都可通过日常工具访问AI
- 设立AI培训和认证的KPI和个人目标
- 未来5年预计每个工作日将释放12-15小时的工作时间
- 推广劳动力技能预测报告，已发布10个国家的独立报告
- 报告可在ServiceNow网站免费获取
