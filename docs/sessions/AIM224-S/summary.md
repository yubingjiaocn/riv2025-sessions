# AWS re:Invent 2025 技术会议总结：使用 Cloudflare 构建 MCP 服务器

## 会议概述

本次技术会议由 Cloudflare 高级解决方案架构师 Patrick Gritzka 主讲，重点介绍了如何在 Cloudflare 开发者平台上构建模型上下文协议（MCP）服务器。MCP 是由 Anthropic 在去年定义的协议，旨在为基于 LLM 的智能体提供与孤立数据交互的接口。

演讲者展示了 Cloudflare 如何通过其无服务器边缘计算产品 Workers 和持久对象（Durable Objects）为企业提供快速构建生产就绪 MCP 服务器的解决方案。会议强调了 MCP 服务器作为新兴渠道的重要性，使企业能够通过智能体交互方式提供服务，这可能会改变传统搜索和信息获取方式。

## 详细时间线与关键要点

### 0:00-2:00 - 开场介绍与 MCP 概念
- Patrick Gritzka 自我介绍，Cloudflare 高级解决方案架构师
- 介绍 Cloudflare 开发者平台，专注于 Workers 无服务器边缘计算
- 定义 MCP（模型上下文协议）：由 Anthropic 定义的协议
- MCP 服务器作为 LLM 智能体与孤立数据交互的接口

### 2:00-4:00 - MCP 服务器功能与合作案例
- MCP 服务器可以更新 Slack 消息、Jira 工单等跨服务内容
- 工具可以简单地暴露现有 API，也可以包含复杂的自定义逻辑
- Cloudflare 运行约 13 个远程 MCP 服务，涵盖可观测性、文档、雷达服务等
- 合作伙伴包括 PayPal、Stripe、Webflow、Block、Atlassian
- PayPal 在 3 天内从概念到生产就绪的发票场景 MCP 服务器

### 4:00-6:00 - 技术架构与优势
- MCP 服务器基于 Cloudflare 原语构建，使用持久对象
- 持久对象提供无服务器计算与有状态存储的结合
- 支持零扩展计算，同时维护短期和长期会话记忆
- 分布式架构提供单一全球数据平面
- 无需管理跨区域的独立控制集群或更新

### 6:00-8:00 - 实时演示准备
- MCP 作为新渠道的重要性，类似于搜索的演进
- 企业需要通过 MCP 服务器暴露产品和服务
- 演示聊天界面连接到 MCP 服务器
- 服务器可以访问浏览器渲染服务，与网页进行交互

### 8:00-10:00 - 浏览器渲染功能演示
- 智能体可以与没有明确定义 API 的页面交互
- 浏览器渲染提供无头浏览器功能
- 支持 HTML、Markdown、JSON 格式的内容抓取
- 支持 Puppeteer 风格的交互：点击按钮、输入文本
- 演示预订系统的动态交互

### 10:00-12:00 - 扩展性与附加功能
- 持久对象提供卓越的水平扩展能力
- 每个会话绑定到单独的持久对象实例
- 自动分片计算和存储，无需担心流量扩展
- 支持单用户或多用户会话模式
- 介绍动态 Worker 加载器：支持 JavaScript/TypeScript 代码执行
- 适用于沙箱和不受信任的智能体生成代码执行场景

### 12:00-12:18 - 总结与问答
- 提供近零毫秒冷启动时间
- 按毫秒计费的计算模式
- 邀请观众就实现方法和技术细节提问
- 强调 Cloudflare 在智能体工作负载方面的丰富 SDK 和工具支持
