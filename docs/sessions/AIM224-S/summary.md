# AWS re:Invent 2025 技术会议总结：使用 Cloudflare 构建 MCP 服务器

## 会议概述

本次技术会议由 Cloudflare 高级解决方案架构师 Patrick Gritska 主讲，重点介绍了如何在 Cloudflare 开发者平台上构建模型上下文协议（MCP）服务器。MCP 是由 Anthropic 在去年定义的协议，旨在为大语言模型代理提供与孤立数据交互的标准化接口。

Cloudflare 已与多家知名公司合作，包括 PayPal、Stripe、WebFlow、Block 和 Atlassian，帮助他们在今年成功部署了首批生产级 MCP 服务器。演讲者展示了 Cloudflare 平台如何通过其无服务器边缘计算产品 Workers 和持久对象（Durable Objects）来提供快速、可扩展的 MCP 服务器解决方案，同时最小化运营开销。
