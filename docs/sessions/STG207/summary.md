# AWS re:Invent 2025 - STG207: 使用 AWS Backup 保护您的数据

## 会议概述

本次会议全面介绍了 AWS Backup 服务的核心功能、最新特性以及实际应用案例。会议由 AWS Backup 产品管理负责人 Palak Desai 和团队成员 Bismani 主持,并邀请了埃克森美孚和摩根大通两家企业客户分享他们的数据保护实践经验。

AWS Backup 是 AWS 原生的集中式数据保护解决方案,目前服务超过 15 万客户,保护着 3.8 EB 的数据。该服务支持 20 多种 AWS 资源的备份,包括计算、存储、数据库和容器等类型。2025 年,AWS Backup 在勒索软件防护、资源支持和平台性能方面都有重大突破,特别是推出了对 Amazon EKS 容器的支持、增强的逻辑隔离保险库功能、以及与 Amazon GuardDuty 集成的恶意软件扫描能力。

客户案例展示了如何利用 AWS Backup 构建主动弹性架构。埃克森美孚强调了不可变备份、自动化编排和隔离恢复环境的重要性,而摩根大通则分享了如何满足严格的监管要求,确保备份的可恢复性和合规性。两家企业都将 AWS Backup 作为应对勒索软件攻击的"保险策略",实现快速、安全的灾难恢复。
