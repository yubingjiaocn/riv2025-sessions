# AWS re:Invent 2025 会议总结：修复昂贵且不准确的AI代理

## 会议概述

本次breakout session由Databricks产品管理负责人Craig Wiley主讲，联合Nick和来自Kynast的Vickerman Marcos共同呈现。会议聚焦于如何构建高质量、可部署到生产环境的AI代理系统，而不仅仅是个人生产力工具。

Craig强调了当前AI代理开发面临的两大核心挑战：成本过高和准确性不足。他指出，超过85%的AI代理仅通过"感觉测试"就被部署，缺乏严格的评估体系。Databricks的解决方案是通过Agent Bricks平台，将评估驱动的开发方法论置于核心位置，使企业能够将AI代理部署到风险路径中（无论是内部工作流还是面向客户的应用）。

会议展示了Databricks如何通过Unity Catalog实现全面的治理能力，不仅管控数据访问，还能监控AI代理的每一个操作步骤，包括API调用、工具使用和模型访问。同时，通过AIBI的自然语言能力和Agent Bricks的模板化构建方式，大幅降低了高质量AI代理的开发门槛。演示环节展示了从费用报销单据提取到完整工作流自动化的端到端实现过程。
