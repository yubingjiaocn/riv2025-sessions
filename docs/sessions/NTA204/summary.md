# AWS re:Invent 2025 - TDK Sensei 工业运营转型会议总结

## 会议概述

本次会议介绍了 TDK Sensei 如何利用 AWS 云服务转型工业运营。TDK Sensei 是一个下一代边缘智能平台，融合了先进传感器、AI 和机器学习技术，为制造业和工业环境提供实时可操作的洞察。该平台已从基础的基于条件的监控发展到预测性维护，现在更进一步实现了代理式 AI，不仅能预测机器何时会故障，还能推荐解决方案并协调响应流程。

TDK Sensei 构建在 AWS 之上，利用 IoT Core 和 Greengrass 进行边缘计算，SageMaker 用于机器学习，Bedrock 用于代理式 AI 功能。该架构采用安全优先的设计理念，从第一天起就内置了全面的设备配置和租户隔离。平台能够减少计划外停机时间，优化维护成本，并创建运营数据的数字孪生，支持制造业、智能建筑、物流和能源等多个行业的决策制定。
