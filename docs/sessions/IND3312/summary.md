# AWS re:Invent 2025 - Coinbase 使用生成式AI扩展支持、合规和开发者生产力工作流程

## 会议概述

本次会议（Industry 3312）由AWS金融服务高级解决方案架构师Joshua Smith和Coinbase机器学习与人工智能总监Varsha Mahadevan共同主讲。会议重点分享了Coinbase如何在AWS上使用生成式AI来扩展其客户支持、合规性和开发者生产力工作流程。

Joshua首先介绍了金融服务领域生成式AI的发展历程，从2023年的基础信任和安全问题，到2024年客户开始在客户支持等关键领域应用AI，再到2025年构建完全多智能体自主应用程序。他详细介绍了AWS的代理AI服务栈，特别是Amazon Bedrock和新推出的Agent Core服务，这些服务为构建可扩展的生产级AI代理提供了专门的运行时、身份管理、内存管理和可观测性功能。

Varsha随后深入分享了Coinbase的AI愿景和具体实施案例。作为一个加密货币交易平台，Coinbase的使命是为超过10亿人扩展经济自由。公司在三个关键支柱上运营：建立信任（通过顶级安全保护用户资产）、使加密货币易于访问（通过AI和机器学习设计直观个性化体验）、以及全球扩展（高效运营，支持100多个国家的数百万用户）。

## 详细时间线与关键要点

### 0:00-10:00 - 开场介绍与AWS服务概述
- Joshua Smith介绍会议议程和演讲者
- 回顾金融服务生成式AI发展历程：2023年基础信任问题 → 2024年客户支持应用 → 2025年多智能体自主应用
- 介绍AWS代理AI服务栈，重点讲解Amazon Bedrock作为完全托管的生成式AI应用构建服务
- 强调生产环境部署AI代理的挑战：专门运行时、内存管理、细粒度身份控制、工具发现与交互、多步骤工作流程执行、专门监控

### 10:00-20:00 - Amazon Bedrock Agent Core详解
- 介绍Agent Core作为模块化、快速迭代的可扩展生产代理构建方法
- Agent Core Runtime：专为部署和扩展动态代理而构建的安全无服务器运行时
- Agent Core Gateway：轻松集成MCP服务器和API
- Agent Core Browser和Code Interpreter：允许代理在浏览器中自主操作或执行代码
- 支持工具：Agent Core Identity（企业级安全身份管理）、Agent Core Memory（短期和长期内存存储）、Observability（集中化可观测性栈）

### 20:00-30:00 - Coinbase公司背景与AI基础
- Varsha介绍Coinbase作为加密货币交易所的使命：为超过10亿人扩展经济自由
- 三大运营支柱：建立信任、使加密货币易于访问、全球扩展
- 传统机器学习应用：登录时防止账户接管、资金转移风险评估、区块链资金释放前风险评估
- 个性化用户体验：搜索结果、新闻推送、推荐系统、价格提醒
- 创新应用：基于图神经网络的区块链地址风险评分系统、ERC-20诈骗代币检测、预测性机器学习模型

### 30:00-40:00 - 客户支持中的生成式AI应用
- 客户支持面临的独特挑战：加密货币高度波动性（用户活动月内波动50%）、信任至关重要、全球运营需求
- 三层迭代设计的聊天机器人架构：
  - 第一层：简单FAQ式RAG响应层，处理登录、双因素认证等基础问题
  - 第二层：业务流程执行层，能够收集信息并代表用户采取直接行动
  - 第三层：主动问题解决层，通过用户信号和活跃事件预测并解决问题
- 代理架构包含：RAG检索器、响应生成、输入输出防护栏、业务流程分类器

### 40:00-50:00 - Agent Assist工具与合规应用
- Agent Assist工具为人工客户支持代理提供实时AI辅助
- 客户支持成果：65%的客户联系由AI系统自动处理，每年节省近500万员工工时，自动化案例在10分钟内解决（人工处理需40分钟）
- 合规领域应用：反洗钱(AML)、反恐怖主义融资(CFT)、反贿赂腐败(ABC)
- 合规流程：了解客户(KYC)、了解业务(KYB)、交易监控系统(TMS)
- 合规自动解决引擎(CAR)：协调人机协作流程，生成叙述性摘要报告，最终决策仍由人工合规代理做出

### 50:00-53:10 - 开发者生产力与未来展望
- 软件开发生命周期(SDLC)的AI应用：代码编写、代码审查、质量保证
- 代码编写工具：Claude Code、Cursor等智能IDE，通过Bedrock提供的Anthropic模型支持
- 自动化PR审查：开源工具增强版，作为AI驱动的GitHub Action实现，总结代码变更、生成审查评论、执行编码规范
- 自动化UI测试：将自然语言测试描述转换为自主浏览器操作
- 开发者生产力成果：40%的日常代码为AI生成或影响，PR审查每年节省75,000小时，QA测试准确性与人工相当但检测bug数量是人工的3倍，成本降低86%
- 未来计划：使用Bedrock Agent Core现代化升级现有AI工具，民主化AI能力让每位员工都能利用AI技术
