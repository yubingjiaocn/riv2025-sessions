# AWS re:Invent 2025 - 企业级AI软件工程代理实践

## 会议概述

本次技术分享由Cognition公司联合创始人兼总裁Russell Kaplan主讲，深入探讨了企业在软件工程中采用AI代理的实际应用情况。Cognition是一家专注于长期推理和规划的AI软件工程公司，开发了Devon（AI软件工程师）和Windsurf（智能IDE）等产品。

演讲重点分析了AI编程工具的三个发展阶段：从GitHub Copilot式的代码补全，到完整的AI IDE环境，再到能够独立完成端到端工作的AI软件工程师。通过与全球1000多家企业的合作经验，分享了成功部署AI代理的关键模式和最佳实践，包括同步与异步工作模式的选择、代码库理解的重要性，以及覆盖整个软件开发生命周期的自动化策略。
