# AWS re:Invent 2025 技术会议总结

## 会议概述

本次会议由 Impetus Technologies 的解决方案架构团队负责人 Sachneet 主讲，主题为"通过自动化现代化帮助企业释放 GenAI 潜力"。会议重点介绍了如何将企业数十年积累的遗留数据系统迁移到 AWS 平台，以及 Impetus 在数据现代化和 GenAI 领域的解决方案。

Impetus Technologies 是一家专注于数据、云和 GenAI 的全球性公司，拥有 3600+ 员工，服务超过 30 家财富百强客户。作为 AWS 顶级服务合作伙伴，该公司在 ETL 迁移领域排名第一，并在 AWS Marketplace 上提供 10+ 项服务和解决方案。会议详细介绍了其核心产品 LeapLogic 和新推出的 Prism 解决方案。

## 详细时间线与关键要点

### 0:00-3:00 公司介绍与背景
- Impetus Technologies 公司概况：全球 3600+ 员工，服务 30+ 财富百强客户
- AWS 顶级服务合作伙伴地位，ETL 迁移领域排名第一
- 建立了 AWS 卓越中心，获得 D2E 认证
- 在 AWS Marketplace 提供 10+ 项服务和解决方案

### 3:00-6:00 智能企业解决方案组合
- 数据和云工程服务：数据策略、云策略、GenAI 策略设计
- 数据平台加速器和 Prism 统一可观测性解决方案
- 专门的 DevOps 实践：CI/CD、AI 和 ML 运维
- GenAI 实验室：加速创意和解决方案设计，支持 RAG 框架和代理 AI

### 6:00-9:00 现代化需求与挑战
- 调研显示：90% 的领导者优先考虑遗留系统现代化
- 企业面临的关键问题：如何重用数十年的业务逻辑投资
- 现代化工作量评估：6个月到2年的项目周期
- 业务连续性保障的重要性

### 9:00-12:00 LeapLogic 现代化方法论
- 平衡方法：80% 基于模式的转换，20% 重写或重新设计
- 四步方法论：评估、转换、验证、运营化
- 评估阶段关键洞察：整体评分、业务逻辑复杂性、技术债务、端到端血缘分析
- 支持的遗留系统：EDW、ETL、分析、Hadoop 等

### 12:00-15:00 自动化转换能力
- 90-95% 的遗留代码自动转换到现代 AWS 平台
- 支持 Informatica、DataStage、Teradata、Oracle 到 Glue 和 Redshift 的迁移
- 最小化手工工作：5-10% 由 Impetus 专家处理
- 包含标准化和优化的隐式处理

### 15:00-17:00 LeapLogic 竞争优势
- 端到端自动化：支持 EDW、ETL、Hadoop、BI 全栈
- 基于模式的转换 vs 逐行转换
- 智能依赖关系和血缘分析，支持分阶段迁移
- 90-95% 自动化水平，单元级验证
- 企业级定制化和性能优化

### 17:00-19:00 客户案例与新产品 Prism
- United Airlines：Teradata 到 Redshift 迁移，节省 70% 工作量和 22% 时间
- American Express：Teradata Ab Initio 迁移，超过 70% 自动化
- Bank United：报告时间从 12+ 小时减少 50%
- Prism 产品介绍：AI 驱动的统一可观测性解决方案，支持多云和数据平台统一视图
