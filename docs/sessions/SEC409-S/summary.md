# AWS re:Invent 2025 会议总结：内部威胁检测与员工监控

## 会议概述

本次技术分享由Arrivia公司收入完整性运营合规副总裁Justin Kagan主讲，他同时也是认证舞弊审查师(CFE)。会议重点介绍了如何利用TerraMind员工监控软件来检测和预防内部威胁，特别是员工欺诈行为。

Justin分享了从传统的被动欺诈检测转向主动实时监控的实践经验。Arrivia作为一家拥有2100多名全球员工的白标旅游代理公司，面临着员工可能滥用客户信用卡信息和忠诚度积分的风险。通过部署TerraMind监控系统，公司成功识别并阻止了多起内部欺诈案件，将调查时间从3-4周缩短至2天，并实现了从被动响应到主动预防的转变。

## 详细时间线与关键要点

### 00:00-05:00 演讲者背景介绍
- Justin Kagan自我介绍，分享童年受电影《Hudson Hawk》启发想成为艺术盗贼的经历
- 介绍从法医科学转向私营部门，最终进入收入完整性部门的职业路径
- 强调现在可以"像罪犯一样思考，像罪犯一样行动，获得报酬且不会惹麻烦"

### 05:00-10:00 公司背景与欺诈三角理论
- 介绍Arrivia公司：20多年历史的白标旅游代理，从分时度假交换发展为全方位旅游服务
- 与Spirit等合作伙伴的忠诚度积分项目
- 详细解释欺诈三角理论：机会(Opportunity)、动机(Motive)、合理化(Rationalization)
- 分享"Tipper X"内幕交易案例，强调欺诈者如何将犯罪合理化为"无受害者犯罪"

### 10:00-15:00 调查案例：发现可疑活动
- 描述发现欺诈性酒店预订的调查过程
- 员工承认共享密码，开始互相指控
- 发现一名员工账户查看次数异常高（比其他代理高10-20倍）
- 怀疑该员工在构建潜在客户名单用于销售

### 15:00-20:00 墨西哥卡特尔诈骗模式
- 详细描述卡特尔如何接触呼叫中心员工购买潜在客户名单
- 卡特尔分时度假诈骗的完整流程：承诺高价购买分时度假→收取"手续费"→循环收费
- 二次诈骗：冒充执法部门帮助受害者追回资金，实际继续收费
- 强调这是卡特尔当前主要收入来源之一

### 20:00-25:00 TerraMind解决方案的引入
- 面临墨西哥劳动法限制，无法在没有确凿证据的情况下解雇员工
- 传统调查方法需要3-4周时间收集Teams消息、Outlook邮箱、OneDrive文件
- 劳动力管理部门介绍TerraMind试点项目
- 幸运地发现目标员工正好在20人试点名单中

### 25:00-30:00 TerraMind功能展示与调查突破
- 描述TerraMind如"上帝模式"的全面监控能力
- 观察到员工下班后访问CRM系统的可疑行为
- 实时观看员工复制PII信息到Excel文件的过程
- 发现员工将字体改为白色并隐藏标签页的高级隐藏技巧
- 调查时间从3-4周缩短至2天，成功阻止数据泄露

### 30:00-35:00 系统扩展与组织挑战
- 6个月内将监控范围扩展至1500名员工（受不同地区法律限制）
- 克服"老大哥监控"的员工抵触情绪
- 建立治理委员会和权限管理系统
- 实施HIPAA和PCI合规措施，屏蔽敏感信息

### 35:00-38:00 积极用例与关键收获
- 生产力优化：发现按键作弊行为，重新分配工作负载
- 学习顶级销售员的方法论，制定标准操作程序
- 证明员工清白：通过时间戳和IP地址信息为被误指控的员工辩护
- 发现虚假病假、系统问题造假、时间卡操纵等各种违规行为
- 强调行为分析、员工信任和风险缓解的重要性
- 实现从被动响应到主动预防的转变
