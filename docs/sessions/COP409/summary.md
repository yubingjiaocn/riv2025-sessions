# AWS re:Invent 2025 会议总结：构建主权云环境

## 一、会议概述

本次技术会议由AWS首席工程师Bo Leoner和高级软件交付经理Randy Domingo主讲，重点介绍了如何使用Landing Zone Accelerator (LZA)解决方案构建符合数字主权要求的云环境。会议深入探讨了数字主权的概念、客户面临的合规挑战，以及AWS如何通过开源的LZA解决方案帮助客户在多账户、多区域环境中快速部署安全控制措施。

LZA是一个基于AWS CDK构建的开源解决方案，旨在帮助高度监管行业的客户快速建立符合各种合规框架（如NIST 853、GDPR、C5等）的安全基线。该解决方案通过YAML配置文件驱动，自动化编排和部署安全资源，大幅缩短了传统Landing Zone的开发周期。会议还通过Security Hub的配置实例，详细展示了LZA如何利用CDK的动态配置能力，在整个组织的所有账户和区域中自动部署和管理安全服务。
