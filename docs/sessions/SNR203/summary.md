# AWS re:Invent 2025 技术侦察与双重使命：CIO的新时代挑战

## 会议概述

本次会议由前CIO Arvind和前NASA CTO Tom Soderstrom主讲，重点探讨了技术领导者面临的"双重使命"挑战：既要维持核心技术运营的卓越表现，又要成为组织的创新先锋和技术侦察员。演讲者分享了他们在各自职业生涯中如何应对这一挑战的经验，并介绍了一个由AWS韩国解决方案架构师Jiyun Park开发的智能体系统TechRecon，该系统能够自动化技术趋势侦察工作。

会议强调了当今技术加速发展的现实：AI、机器人技术、生物技术等多种技术不仅独立加速发展，更重要的是它们相互融合，创造出指数级的商业影响。这种趋势使得技术领导者必须在保持运营卓越的同时，积极探索和评估新兴技术对业务的潜在影响。演讲者提出了从"卓越中心"(COE)向"参与中心"转变的理念，强调通过业务实验而非纯技术实验来验证新技术的价值。
