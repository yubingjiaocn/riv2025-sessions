# AWS re:Invent 2025 公共部门创新演讲总结

## 会议概述

本次AWS re:Invent 2025公共部门创新演讲由AWS全球公共部门副总裁Dave Levy主持,重点介绍了AWS如何通过创新技术支持政府机构、医疗保健、科研机构和非营利组织完成关键任务。演讲以胡佛大坝的建设历史作为类比,强调了AWS在公共部门基础设施建设中的开创性工作,特别是AWS GovCloud的推出如何改变了政府云计算的格局。

会议深入探讨了AWS在AI和超级计算领域的重大投资承诺,包括在美国投资高达500亿美元用于专为政府用例设计的AI和超级计算基础设施,以及在欧洲投资78亿欧元建设AWS欧洲主权云。演讲还展示了多个实际应用案例,包括美国海军使用多模态AI代理系统改进潜艇维护流程,劳伦斯利弗莫尔国家实验室利用云和AI技术实现核聚变点火突破,以及医疗保健和政府服务领域的AI代理应用。

整场演讲传达的核心信息是:AWS不仅提供技术基础设施,更重要的是从客户使命出发进行逆向工作,创新解决看似不可能完成的挑战,帮助公共部门组织实现数字化转型和任务加速。

## 详细时间线

### 开场与历史回顾

[00:00 - 05:30] 开场致辞与胡佛大坝类比
- Dave Levy欢迎与会者参加公共部门创新演讲
- 介绍胡佛大坝建设历史(1931-1936年),作为"不可能任务"的象征
- 强调大型项目虽然风险高,但能够带来突破性创新
- 将AWS的公共部门基础设施建设与胡佛大坝工程相提并论

[05:30 - 10:45] AWS GovCloud的创新历程
- 回顾14年前(2011年)推出AWS GovCloud,成为首个专为政府设计的云基础设施
- 介绍GovCloud面临的技术挑战:物理隔离、合规性要求、定制硬件安全模块
- 2014年推出AWS Top Secret区域,2017年推出Secret区域
- 2024年10月推出AWS Secret West区域,实现跨大陆距离的机密工作负载同步

### 重大基础设施投资宣布

[10:45 - 15:20] 美国AI和超级计算基础设施投资
- 宣布高达500亿美元的投资计划,专为政府用例和安全要求设计
- 这是亚马逊历史上最大的联邦技术投资
- 提供近1.3吉瓦的计算容量(胡佛大坝产生约2吉瓦)
- 将支持自主系统开发、能源创新和基因组数据处理等应用

[15:20 - 17:30] 欧洲主权云计划
- 宣布AWS欧洲主权云,投资78亿欧元
- 首个完全功能的独立运营主权云
- 仅由欧盟居民在欧盟境内控制日常运营
- 首个区域将在德国勃兰登堡州启动

[17:30 - 20:15] AI工厂和沙特阿拉伯合作
- 推出AI工厂概念,将专用AI基础设施直接部署到客户数据中心
- 客户提供数据中心和电力,AWS处理其他一切
- 宣布与沙特阿拉伯Humane的合作,投资50亿美元
- 支持沙特阿拉伯成为全球AI领导者的使命

### 实际应用案例展示

[20:15 - 24:00] 美国海军潜艇维护案例
- 海军潜艇部队司令部面临的维护挑战:手动导航多个系统和数百份文档
- 开发多模态AI代理解决方案,在AWS GovCloud上运行
- 达到Impact Level 5分类标准(海军最高级别的非机密关键数据)
- 维护处理时间从数周缩短到数小时

[24:00 - 26:30] Maverick移动创新实验室
- 介绍Maverick移动创新实验室,与Anduril和Gambit合作开发
- 将标准车辆改造成移动实验室,用于自主系统快速原型开发
- 演示对话式无人机控制:操作员输入简单命令,Amazon Bedrock代理翻译成自主任务
- 运行在AWS Outpost服务器上,集成合作伙伴硬件和软件

### 劳伦斯利弗莫尔国家实验室访谈

[26:30 - 28:45] Greg Herwig介绍与开场
- 介绍劳伦斯利弗莫尔国家实验室(LLNL)背景
- 2022年实现核聚变点火,AI在其中发挥关键作用
- 实验室拥有约10,000名员工,包括2,000多名博士科学家

[28:45 - 33:00] 云采用的挑战与策略
- 面对科学家群体的怀疑态度,通过小规模试点证明可行性
- 提出"绿色闪烁灯"(green blinky lights)概念来展示进展
- 从非机密环境开始,逐步扩展到机密环境
- 强调合作伙伴关系和持续推进的重要性

[33:00 - 37:15] 云和AI的驱动因素
- 创造力:为非计算机科学家提供更广泛的工具选择
- 敏捷性:加快科学发现循环
- 效率:减少基础设施管理时间,将资源投入科学研究
- 混合环境:结合本地资源和云资源的挑战

[37:15 - 42:00] AI在实验室的全面部署
- 将多个大语言模型(LLM)部署到所有科学家手中
- 对全部10,000名员工进行AI技能提升培训
- 强调"每个人都需要成为AI人员"的理念
- 流程再造的重要性:不能简单插入LLM,需要重新设计工作流程

[42:00 - 45:30] Smart Lab计划和NIF应用
- Smart Lab计划:对整个实验室(一平方英里,600栋建筑)进行仪器化
- 部署工业控制和物联网设备,收集运营和科学数据
- 建立大型数据存储库,打破数据孤岛
- 机密环境中的传感器部署挑战

[45:30 - 47:00] 给公共部门的建议
- 采用"爬行、行走、奔跑"的渐进式方法
- 找到愿意学习和尝试新事物的早期采用者
- 通过试点项目展示"绿色闪烁灯"
- 与AWS等合作伙伴建立强有力的合作关系

### 医疗保健AI应用

[47:00 - 51:30] Amazon Connect在医疗保健中的应用
- 介绍Amazon Connect的AI驱动患者参与能力
- 美国医疗系统面临的挑战:大量患者沟通、预约安排困难
- 新的AI代理功能:与电子健康记录实时集成,自动化常规任务
- 成功案例:
  - 加州大学圣地亚哥分校医疗中心:患者放弃呼叫平均减少29%,某些部门高达59%
  - 佛罗里达州Jupiter医疗中心:放射科积压减少60%,预约等待时间从2周降至24小时

[51:30 - 54:00] Cohere Health的保险授权案例
- 每年处理超过1200万份授权请求
- 使用Amazon Bedrock Agent Core
- 授权审查时间减少30-40%,同时提高临床决策准确性
- 分析结构化和非结构化数据(临床记录、患者笔记、传真)

### Capita的政府服务转型

[54:00 - 57:30] Capita公司介绍和BPO行业变革
- Capita CEO Adolfo Hernandez登台
- 公司背景:近40年历史,总部位于英国,服务近500个政府机构
- 每年服务超过2200万公民,超过1亿次互动
- BPO行业正在经历根本性转变:从传统流程管理到AI代理驱动

[57:30 - 1:02:00] AI Catalyst技术栈
- 基于近40年业务流程经验
- 与AWS合作,利用云架构和可扩展服务
- 流程到代理(Process-to-Agent, P2A)框架
- 模块化架构:从公民体验层到应用层再到遗留系统集成

[1:02:00 - 1:07:30] 英国政府机构评估案例研究
- 与AWS和BCG合作开展12周冲刺项目
- 处理数百万评估,影响数十万人生活
- 挑战:复杂工作流程、大量证据(多种格式,包括手写文档)、服务交付瓶颈
- 开发五个AI代理:
  1. 证据管理代理
  2. 案例管理和智能路由代理
  3. 评估辅助和转录代理
  4. 决策辅助代理
  5. 质量保证代理

[1:07:30 - 1:10:00] 项目成果
- 处理时间减少40%
- 吞吐量提高25%
- 质量改善,减少问题升级
- 公民和专业人员满意度提高
- Catalyst Lab作为内部能力工厂,将此蓝图应用于其他政府机构

- 涵盖医疗保健、教育、人道主义服务和科学研究领域

[1:13:30 - 1:16:00] Jane Goodall研究所案例

- 研究所使用AWS解锁60年灵长类动物研究
