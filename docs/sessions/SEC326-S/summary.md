# AWS re:Invent 2025 Fastly 网络安全欺骗技术会议总结

## 会议概述

本次技术会议由 Fastly 团队的 Rick 和 Memo 主讲，重点介绍了他们在网络安全领域的创新欺骗技术。会议以一个生动的高尔夫故事开场，讲述了1958年盲人高尔夫冠军 Charley Boswell 如何通过选择夜间比赛来平衡与世界顶级高尔夫选手 Ben Hogan 之间的竞技差距，以此类比网络安全中如何通过欺骗技术来平衡攻防双方的力量对比。

演讲者深入探讨了行为科学在网络安全中的应用，特别是"nudge"（正向引导）和"sludge"（增加摩擦）概念如何被用于影响攻击者的决策过程。Fastly 团队展示了他们如何将欺骗技术直接集成到 Next-Gen WAF 产品中，通过在正常工作流程中添加欺骗层来迷惑攻击者，而不是传统的单纯阻断方式。这种方法旨在增加攻击者的心理成本，包括挫败感、困惑和声誉损失。

## 详细时间线与关键要点

### 0:00-3:00 开场故事与类比
- Rick 分享1958年盲人高尔夫冠军 Charley Boswell 与 Ben Hogan 的故事
- 通过选择夜间比赛时间，盲人选手成功平衡了与世界顶级选手的竞技差距
- 引出网络安全中如何平衡攻防双方力量的核心主题
- 使用橄榄球攻防对比来说明攻击者通常具有的天然优势

### 3:00-7:00 行为科学理论基础
- Memo 介绍行为科学中的"nudge"和"sludge"概念
- Nudge：正向引导，简化流程（如密码强度提示器、运动提醒应用）
- Sludge：增加摩擦和复杂性（如复杂的 CAPTCHA、订阅服务的取消流程）
- 强调攻击者也是人类，具有相同的认知模式和决策过程
- 解释如何通过增加摩擦来影响攻击者的行为和决策

### 7:00-10:00 攻击者成本分析与欺骗技术历史
- 详细分析对攻击者施加摩擦的各种成本类型：
  - 量化成本：金钱、时间、机会成本
  - 质化成本：声誉损失、招募困难、信息获取成本
  - 心理成本：挫败感、羞耻感、困惑感
- 回顾欺骗技术在历史中的应用（特洛伊木马、自然界伪装）
- 介绍网络安全中传统蜜罐技术的概念和局限性

### 10:00-14:00 Fastly 欺骗技术实现
- Rick 展示 Fastly Next-Gen WAF 中的创新欺骗功能
- 重点针对账户接管攻击场景的防护策略
- 技术特点：
  - 可配置的触发条件（恶意机器人、特定路径、IP、国家等）
  - 密码和用户名混淆技术，而非直接阻断
  - 内联集成，无需单独基础设施维护
  - GUI 驱动界面，易于配置和测试
- 强调这是深度防御策略的一部分，结合速率限制、DDoS 防护等多层安全措施
- 所有功能支持 API 驱动和 Terraform 管理，便于部署和迭代
