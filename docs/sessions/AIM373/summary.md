# AWS re:Invent 2025 - 营销图像生成管道会议总结

## 会议概述

本次会议由AWS生成式AI创新中心的高级应用科学家Henry Wang和高级数据科学家Parma主讲,重点介绍了如何构建一个自动化的营销图像生成管道。会议探讨了生成式AI在营销领域的应用挑战,特别是品牌安全性问题。尽管AI图像生成技术能够快速创建视觉效果惊艳的图像,但在实际应用中仍面临产品细节变化、色彩纹理不一致等问题,这对品牌营销来说是不可接受的。

演讲者提出了一个完整的解决方案架构,通过多个模块协同工作来解决这些挑战。该管道包括提示词分解、资产检索、标题生成、图像合成规划、以及多步骤质量控制等环节。整个系统充分利用了Amazon Nova系列模型的能力,特别是Nova Pro的多模态能力和Nova Canvas的图像生成能力。通过这套自动化流程,可以大幅减少人工干预,将专家审核推迟到最后阶段,并将审核数量控制在最小范围内,从而实现规模化的营销图像生成。

测试结果显示,该管道在物体相似度指标上表现优异,人类偏好测试中获得了70%的支持率。演讲者还强调了持续改进的重要性,建议引入代理反馈机制、人类反馈循环以及针对特定品牌需求的微调等优化方向。

## 详细时间线

00:00 - 开场与介绍
- 演讲者Henry Wang和Parma介绍自己,来自AWS生成式AI创新中心
- 互动环节:询问观众re:Invent期间的步数记录(有人达到30,000步)
- 介绍会议主题:营销图像生成管道设计

02:30 - 会议议程概览
- 讨论生成式AI在营销中的应用观察
- 架构概述
- 各组件深入讲解
- 行动建议

03:15 - 现状调研
- 询问观众使用生成式AI生成图像的经验
- 询问是否有人在营销领域工作
- 介绍传统营销图像制作流程的手动性质

04:30 - 问题陈述
- 展示生成式AI图像生成的潜力(钱包、运动鞋、沙发等示例)
- 指出品牌安全性问题:产品纹理和颜色会发生变化
- 展示实际案例:沙发图像中产品细节改变的问题
- 强调非确定性特性导致的一致性挑战

07:00 - 核心挑战
- 如何从大量生成的图像中自动选择最佳图像
- 展示街头展位场景中不符合物理规律的图像示例
- 提出解决方案目标:可扩展和自动化的图像选择

08:45 - 解决方案管道概览
- 用户输入简单查询(例如:"日落时分城市街道上的鞋子")
- 提示词分解:提取产品、背景、主题等实体
- 向量搜索:从媒体库检索相关资产
- 自动生成优化的提示词
- 图像合成:不同位置、方向、旋转的产品布局
- 多步骤质量控制:使用视觉语言模型检查
- 美学评分:选出排名最高的图像

12:00 - 提示词分解与资产检索
- 示例:"在晴天的城市街道地面上放置一只鞋子"
- 提取关键实体:产品、背景、主题
- 使用轻量级大语言模型进行实体提取
- 通过向量嵌入空间进行相似度匹配
- 从产品库和背景库中检索候选资产

15:30 - 标题生成模块(Parma讲解)
- 强调标题生成在图像生成中占90%的工作量
- 介绍标题拓扑结构:主体、参考点、环境
- 可选元素:位置姿态、光照描述、相机位置、风格
- 使用Nova Pro进行自动标题生成

17:45 - 标题生成的三大原则
1. 生成类似标题的提示词,而非指令式提示词
2. 保持标题长度在1,024字符以内(图像生成模型的训练特点)
3. 让Nova Pro理解标题拓扑结构(相机位置、主体、描述、光照等)

19:30 - 生成任务类型回顾
- 文本到图像(Text-to-Image):基础功能,但无法保证产品细节一致
- 内绘制(Inpainting):在现有图像中插入元素,保持环境不变
- 外绘制(Outpainting):保持产品不变,改变周围环境(选择此方案)
- Nova Canvas的优势:支持提示词遮罩,无需图像遮罩

22:00 - 合成规划模块
- 三个关键要素:相对尺寸、位置、旋转
- 使用Nova Pro的多模态能力确定产品缩放比例
- 输入:图像标题、背景参考(或空画布)、产品对象
- Nova Pro根据画布尺寸提供缩放建议

24:15 - 网格搜索策略
- 将画布分为不同区域(示例中使用3个区域)
- 每个位置提供3种旋转角度:原始、右转15度、左转15度
- 网格大小取决于性能与成本的权衡
- 该方案选择9个网格配置
- 使用Nova Canvas进行外绘制生成候选图像

26:30 - 智能过滤组件
- 目标:减少需要人工审核的图像数量
- 考虑规模化和自动化需求
- 使用Nova Pro的多模态能力进行质量评估

27:45 - 质量检查指南
- 基准指南:针对图像生成器的常见失败模式
- 品牌特定指南:颜色调色板、品牌规范等
- Nova Pro提供二进制评分(通过/不通过)
- 灵活性策略:可以对某些指南(如标题对齐)放宽要求
- 该项目采用严格标准:所有指南必须通过

30:00 - 过滤示例
- 去除有重复元素的图像
- 去除生成了不需要元素(如腿部)的图像
- 去除未生成阴影的图像
- 从9张候选图像筛选至5张

31:30 - 最终排名模块
- 标注预算限制:最多3张图像供人工选择
- 使用美学评分模型进行自动排名
- 三种可选模型:
  - Aesthetic Model:轻量级,基于监督学习,可在CPU上运行
  - Image Reward:基于人类反馈训练
  - HPSV2:基于人类反馈训练,更准确
- 该项目选择Aesthetic Model以节省成本

33:45 - 性能评估
- 定量指标:
  - MSSIM:基于边缘的传统比较方法
  - Dinov2:基于嵌入的相似度比较
- 两者都测量物体相似度
- 对比测试:Nova Canvas原生输出 vs 管道集成输出
- 定性评估:人类偏好测试显示70%支持率

36:00 - 改进方向
- 迭代机制:质量检查失败时的反馈循环
- 将质量检查LLM转变为代理,提供具体改进建议
- 整合人类专家反馈到管道中
- 针对特定品牌指南进行微调
- 鼓励观众根据自身需求定制管道
