# AWS re:Invent 2025 会议总结：Sophos 使用 Cribble 实现 AWS 数据安全现代化

## 会议概述

本次会议由 Cribble 公司的 Camir 和现场 CISO Ed Bailey 主讲,介绍了 Sophos 如何利用 Cribble 解决方案实现其 AWS 数据及其他数据源的安全现代化。Sophos 是一家全球领先的网络安全公司,拥有 60 万客户,提供 MDR(托管检测和响应)服务。

会议重点讨论了企业在数据管理方面面临的核心挑战:数据量呈指数级增长(Sophos 的增长率高达 60%),而预算增长远远跟不上数据增长速度。传统工具集无法应对这种规模,导致成本失控和工程资源浪费。Cribble 作为一家专注于数据管道和遥测数据解决方案的后期创业公司,通过其产品组合帮助 Sophos 实现了显著的成本节约和运营效率提升。

Sophos 通过部署 Cribble 的解决方案,在前 30 天内就实现了 48% 的成本节约,同时获得了数据独立性、更好的治理能力和安全设计。这种现代化方法使 Sophos 能够在控制成本的同时,提供更高质量的数据用于安全检测和合规需求。
