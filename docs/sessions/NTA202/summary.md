# AWS re:Invent 2025 - 代理AI：从概念验证到生产规模

## 会议概述

本次技术分享会聚焦于代理AI（Agentic AI）从概念到生产环境的实际应用。主讲人包括AWS高级客户解决方案经理Sujal Shah、Siteimprove AI和数据科学总监Hamed Shahir，以及AWS高级解决方案架构师Pradeep Sridharan。

会议核心围绕三个关键要素展开：首先是如何超越技术炒作，识别真正符合业务目标的AI计划；其次通过Siteimprove的实际案例展示从想法到生产就绪解决方案的完整旅程；最后介绍AWS GenAI服务如何为企业级代理AI解决方案提供全面支持。代理AI被定义为能够独立规划、推理和主动采取行动的系统，它将企业从被动的自动化工作流转变为主动的结果驱动系统。

## 详细时间线与关键要点

### 0:00-10:00 代理AI基础概念与市场趋势
- **代理AI定义**：超越传统自动化和生成式AI，能够独立规划、推理并主动采取行动
- **市场预测**：Gartner预测一年内40%的企业应用将嵌入任务特定的代理AI
- **未来展望**：到2035年，代理AI驱动的软件预计将占据约35%的企业应用收入（约4500亿美元）
- **两大应用方向**：运营改进（提升效率）和产品创新（创造新收入流）

### 10:00-20:00 AI投资优先级框架
- **工作倒推法**：从客户结果出发，确定真正值得构建和资助的项目
- **投资框架**：快速胜利（高影响低努力）+ 战略押注（最高潜力项目）
- **三大评估指标**：信任度、速度、采用率
- **代理AI作为力量倍增器**：通过正确的基础设施和人才投资获得杠杆效应

### 20:00-35:00 Siteimprove公司介绍与平台概述
- **公司定位**：SaaS平台，帮助组织确保数字化存在的可访问性、合规性和高性能
- **统一平台**：集成数字可访问性、分析、SEO和内容策略的AI代理解决方案
- **四大代理类型**：可访问性代理、分析代理、SEO代理、内容代理
- **Forrester认可**：被评为数字可访问性平台领导者

### 35:00-45:00 代理AI采用的三阶段信任曲线
- **第一阶段 - 人在环中**：AI建议和验证，人类做最终决策
- **第二阶段 - 人在环上**：AI执行大部分工作，人类作为监管者
- **第三阶段 - 人出环外**：完全自动化流程
- **关键原则**：每个阶段都必须回答用户是否信任系统以及系统是否证明了其价值

### 45:00-55:00 技术架构与实际用例
- **三个核心用例模式**：
  - 异步批处理（每月处理1亿页面的可访问性检查）
  - 同步对话式修复（实时问题解决）
  - 高优先级异步处理（上下文图像分析）
- **AWS Bedrock选择理由**：企业级安全、多区域支持、多模型API、灵活定价
- **成本优化**：Nova Micro模型相比领先模型成本降低75%

### 55:00-58:24 AWS AgentCore与生产部署
- **生产挑战**：从原型到生产的鸿沟，包括内存管理、安全性、治理和扩展性
- **AgentCore核心组件**：代理运行时、代理网关、代理内存、代理身份、代理可观测性
- **AWS完整生态系统**：从预构建代理到基础设施，再到专家支持的一站式解决方案
- **实施框架**：从业务结果出发 → 选择模型和框架 → 考虑安全合规 → 创建价值证明 → 开发部署扩展
