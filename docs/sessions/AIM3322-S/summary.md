# AWS re:Invent 2025 技术会议总结：AI智能体时代的企业转型

## 会议概述

本次技术会议由Core.AI首席战略官Cahal McCarthy主讲，主题为"在AI智能体时代中蓬勃发展"。会议探讨了企业如何从传统的AI工具使用转向真正的业务模式变革，强调了AI智能体与人类协作的变革性潜力。

McCarthy指出，当前大多数企业虽然声称是"AI优先"公司，但实际上只是在各部门零散地部署AI解决方案，缺乏统一的治理和合规框架。他认为企业正处于关键的转折点，需要从单纯追求生产力提升转向利用AI智能体实现业务模式的根本性变革。

会议重点介绍了Core.AI的企业级AI智能体平台，该平台提供端到端的解决方案，支持员工体验和客户体验的全面优化，并强调了从垂直系统架构向水平对话式AI编排层的转变。

## 详细时间线与关键要点

### 0:00-3:00 开场与核心观点
- 提出企业面临的关键选择：在AI智能体时代如何重新定义工作方式
- 强调应该问"什么更好"而不是"什么不同"
- 介绍Core.AI作为企业AI智能体构建平台的定位

### 3:00-6:00 AI发展现状分析
- 回顾过去24个月AI发展历程：从初期乐观到买家疲劳和试点困境
- 指出企业普遍存在的问题：以"洒水式"方式部署AI，缺乏协调性
- 强调当前正处于关键转折点，需要从自动化思维转向变革思维

### 6:00-9:00 变革思维与商业模式创新
- 批评企业过度关注生产力边界而忽视商业模式挑战
- 通过历史案例说明变革阻力：Digital Equipment、手机发展、加密货币等
- 强调需要跳出历史局限，拥抱真正的业务变革

### 9:00-12:00 AI智能体的定义与特征
- 定义AI智能体：具备独立推理能力，可执行多步骤任务并做出决策
- 区别于传统确定性聊天机器人，强调自主性和跨功能协作能力
- 提出选择：让人力成为商品还是通过AI智能体放大人类能力

### 12:00-15:00 平台化解决方案
- 介绍从垂直系统向水平对话式AI编排层的转变
- 强调统一平台的重要性，避免各部门独立部署造成的治理问题
- 提出新的工作模式：无需直接操作底层系统，通过对话界面完成工作

### 15:00-18:00 Core.AI平台功能展示
- **AI for Work**: 跨部门搜索、工作流自动化、智能编排
- **AI for Service**: 客户体验重塑，支持46个交互渠道
- **AI for Process**: 复杂业务流程自动化，零接触IT运维

### 18:00-19:00 成功衡量标准与总结
- 强调成功不应以部署的AI智能体数量或LLM复杂度衡量
- 关键指标：创新收入流、新的运营模式、新产品和服务
