# AWS re:Invent 2025 - Crowning the Curo Code Champions 会议总结

## 会议概述

本次会议由AWS开发者体验团队技术负责人Brian Beach主持，主要庆祝Curo迷宫编程挑战赛的获胜者。会议首先介绍了Curo这一从原型到生产的AI开发工具，重点讲解了规范驱动开发（spec-driven development）的概念和优势。

Brian回顾了AWS开发工具的演进历程：从2023年的代码自动补全（Code Whisperer），到2024年的聊天助手模式，再到2025年的智能代理时代。Curo作为新一代IDE，能够充分利用代理AI的能力，不仅能提供响应，还能使用工具、读取文档、连接数据库、运行单元测试等，从而做出更明智的决策。会议详细演示了如何使用Curo进行规范驱动开发，通过构建需求、技术设计和任务列表来避免传统"感觉编程"带来的技术债务问题。
