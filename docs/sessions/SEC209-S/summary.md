# AWS re:Invent 2025 数据安全会议总结

## 会议概述

本次会议由Varonis公司现场CTO Brian Vecci主讲，主题为"超越态势管理的数据安全"。Brian拥有16年网络安全经验，在这场17分钟的技术分享中，他深入探讨了传统DSPM（数据安全态势管理）和CSPM（配置安全态势管理）的局限性，以及如何通过全面的数据安全平台来应对现代威胁。

会议重点强调了一个核心观点：攻击者不是"闯入"系统，而是"登录"系统。88%的数据泄露都涉及被盗凭证，因此身份保护和数据访问监控是网络安全的关键。Brian通过实际案例和统计数据，展示了Varonis数据安全平台如何帮助企业在AWS环境中实现全面的数据保护。

## 详细时间线与关键要点

### 0:00-3:00 威胁态势分析
• 核心观点：攻击者通过身份登录而非技术入侵
• 88%的数据泄露涉及被盗凭证
• 攻击链：身份盗用 → 横向移动 → 权限提升 → 数据窃取/加密
• 实际威胁案例：Codefinger勒索软件针对AWS环境
• 攻击向量：信息窃取恶意软件、游戏破解文件中的恶意代码

### 3:00-6:00 传统安全工具的局限性
• DSPM和CSPM工具的不足之处：
  - 无法检测授权变化
  - 无法检测跨平台横向移动
  - 缺乏异常行为检测能力
  - 无法识别内部威胁和活跃威胁
• 数据风险评估发现：高达40%的敏感数据对所有认证账户开放

### 6:00-9:00 全面安全框架需求
• 四层防护体系：
  - 身份保护（人员和机器身份）
  - 数据和配置安全态势监控
  - 以数据为中心的用户实体行为分析
  - 完整的取证能力
• 信用卡欺诈检测类比：通过了解正常行为模式来识别异常

### 9:00-12:00 事件响应的关键问题
• 安全事件发生时的首要问题：
  - 哪些数据被触及？
  - 是否有数据被盗？
  - 需要通知哪些相关方？
• 取证层的重要性：快速完整回答"发生了什么"

### 12:00-15:00 Varonis平台解决方案
• 20年数据安全平台发展历程
• 跨云环境数据可观测性：
  - 准确的数据分类（结构化/非结构化）
  - 全面的访问控制分析
  - 身份权限映射
• 自动化修复能力：安全地移除权限和修复暴露

### 15:00-17:00 AWS集成与总结
• AWS安全工具集成：
  - Security Hub信号增强
  - CloudTrail遥测数据丰富
  - IAM Identity Center上下文增强
  - GuardDuty威胁检测增强
• 风险评估流程：15-20分钟快速接入，一周内识别真实风险
